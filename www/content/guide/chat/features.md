Claude와의 대화는 단순히 질문하고 답을 듣는 것에서 끝나지 않습니다. 마치 옆자리에 앉은 동료가 문서를 함께 펼쳐놓고 고치고, 자료를 표로 정리하고, 복잡한 흐름을 그림으로 그려 보여 주는 것처럼, Claude는 결과물을 직접 만들어 내고 그 자리에서 다듬어 줍니다.

이 페이지에서는 별도 창에서 다루는 **아티팩트**, **이미지·시각 콘텐츠**, **파일 업로드와 분석**, **문서 생성**, **수식 계산** 등 Claude의 핵심 기능을 하나씩 살펴봅니다. 어떤 기능이 어느 요금제에서 되는지, 어떻게 켜는지까지 함께 정리했습니다.

## 한눈에 보는 핵심 기능

| 기능 | 하는 일 | 실제 예시 |
|---|---|---|
| **아티팩트** | 문서·코드·웹사이트 등 분량 있는 결과물을 별도 창에서 편집·반복 작업 | "간단한 할일 목록 HTML을 만들어 줄 수 있나요?" → 별도 창에서 미리보기·수정 |
| **파일 업로드** | PDF, Word, Excel, 이미지 등을 올려 분석 | "이 마케팅 리포트(PDF)의 핵심 3가지를 알려 줄 수 있나요?" |
| **파일 생성** | Excel, PowerPoint, Word, PDF 등을 만들어 바로 다운로드 | "이 데이터를 엑셀 표로 정리해 줄 수 있나요?" |
| **시각 콘텐츠** | 다이어그램, 차트, 날씨·레시피 카드 등을 보여 줌 | "이 과정을 흐름도로 그려 줄 수 있나요?" |
| **이미지 분석** | 사진·스크린샷·다이어그램을 인식하고 설명 | 오류 화면을 올려서 "이 메시지가 뭔가요?" |
| **코드 설명** | 코드를 읽고 작동 방식을 설명하거나 버그 찾기 | "이 Python 코드의 버그를 찾아 줄 수 있나요?" |
| **수식·계산** | 수식을 이해하고 계산 수행 | "이 방정식을 풀이 과정과 함께 설명해 줄 수 있나요?" |

## 아티팩트 — 별도 창에서 만들고 다듬는 결과물

아티팩트(Artifacts)는 분량 있는 결과물(보통 15줄 이상)을 대화창 옆 **별도 창**에 띄워, 한눈에 보면서 편집하고 반복 작업할 수 있게 해 주는 기능입니다. 메모장에 흩어진 메모가 아니라, 전용 작업판에 결과물을 올려놓고 함께 고쳐 나가는 셈입니다.

### 아티팩트로 만들 수 있는 것

- 마크다운 문서, 일반 텍스트
- 코드 조각
- HTML 웹사이트
- SVG 이미지, 다이어그램
- React 컴포넌트 등 인터랙티브 요소

### 주요 기능

| 기능 | 설명 |
|---|---|
| **Edit with Claude** | 마크다운에서 원하는 부분을 짚어 그 자리에서 수정 요청 |
| **버전 선택** | 여러 번 고친 결과물 사이를 오가며 비교 |
| **코드 보기** | 결과물의 내부 코드를 확인 |
| **복사·다운로드** | 클립보드 복사 또는 파일로 내려받기 |
| **여러 개 동시 열기** | 한 대화에서 여러 아티팩트를 함께 작업 |
| **영구 저장** | 텍스트 기준 20MB까지, 개인용 또는 공유용으로 보관 |

### 켜는 방법

1. 왼쪽 아래 **이름(이니셜)** 클릭 → **설정(Settings)** → **기능(Capabilities)**
2. **아티팩트(Artifacts)** 토글을 켜기
3. 고급 아티팩트가 필요하면 **코드 실행 및 파일 생성(Code execution and file creation)** 도 켜기

### 마크다운 아티팩트 직접 편집하기

1. 아티팩트에서 고치고 싶은 텍스트를 드래그해 선택
2. **Edit with Claude** 클릭
3. 원하는 변경 내용을 입력
4. Claude가 해당 부분을 그 자리에서 수정

### 요금제별 제공 범위

| 기능 | 제공 요금제 |
|---|---|
| 기본 아티팩트 | Free / Pro / Max / Team / Enterprise |
| 영구 저장 · MCP 외부 서비스 연동 | Pro / Max / Team / Enterprise |
| Claude Code 아티팩트(베타) | Team / Enterprise 전용 |

{{< hint type="note" >}}
아티팩트는 Asana, Google Calendar, Slack 같은 외부 서비스와 MCP로 연동할 수 있습니다. 다만 각 사용자가 직접 MCP 서버에 인증해야 하며 조직 관리자가 아티팩트가 쓰는 특정 MCP 서버를 일괄 관리하지는 못합니다.
{{< /hint >}}

{{< hint type="warning" >}}
영구 저장은 개발·테스트 중에는 쓸 수 없고 게시(publish)가 필요합니다. 게시를 해제(unpublish)하면 저장된 데이터가 영구히 삭제되며 같은 아티팩트를 다시 게시할 수 없습니다. Claude는 아티팩트의 오류를 고치려 시도하지만 항상 성공하는 것은 아닙니다.
{{< /hint >}}

### 아티팩트 게시하고 공유하기

만든 아티팩트는 공개 링크로 게시하거나(Free / Pro / Max), 조직 내부에서만 공유할 수 있습니다(Team / Enterprise).

| 동작 | 제공 요금제 | 특징 |
|---|---|---|
| **게시(Publish)** | Free / Pro / Max | 회원가입 없이도 누구나 보고 다룰 수 있는 공개 링크 생성 |
| **공유(Share & copy link)** | Team / Enterprise | 조직 구성원에게만 내부 공유 |
| **커스터마이즈(Customize)** | 전체 요금제 | 남의 아티팩트를 내 버전으로 복제해 새 대화 시작 |

웹사이트 임베드도 가능합니다.

1. 게시하려는 아티팩트로 이동
2. **Publish** 버튼 클릭
3. 생성된 공개 링크 복사
4. 임베드하려면 자동 생성된 코드를 복사하고 허용할 도메인을 쉼표로 구분해 지정

{{< hint type="note" >}}
게시된 아티팩트는 가입하지 않은 사람도 보고 다룰 수 있지만 고급 기능을 쓰려면 계정이 필요합니다. 공유된 아티팩트를 열면 원래 대화의 첨부 파일까지 함께 열람되며 프로젝트에서 시작된 아티팩트는 보는 사람이 해당 프로젝트에 접근 권한이 있어야 합니다. 아티팩트 사용량은 만든 사람이 아니라 보는 사람 본인의 구독 한도에서 차감됩니다.
{{< /hint >}}

## 이미지와 시각 콘텐츠

### Claude가 이미지를 만들 수 있나요?

Claude는 사진이나 일러스트 같은 이미지를 직접 생성하지는 못합니다. 대신 **HTML과 SVG로 다이어그램과 차트** 같은 인터랙티브 시각 자료를 만들 수 있고 사용자가 올린 이미지를 분석하고 설명할 수 있습니다. 이 기능은 현재 Claude 웹과 데스크톱에서 베타로 제공됩니다.

| 할 수 있는 것 | 할 수 없는 것 |
|---|---|
| HTML·SVG 기반 다이어그램·차트 생성 | 사진·일러스트 같은 사실적 이미지 생성 |
| 만든 시각 자료를 대화로 실시간 조정 | — |
| 올린 이미지 인식·분석 | — |

### 채팅과 Cowork의 커스텀 시각 자료

Claude는 설명을 더 잘 전달할 수 있을 때 대화 안에서 인터랙티브 다이어그램·차트를 자동으로 만들어 줍니다. 직접 "이걸 다이어그램으로 그려 줘", "이 데이터를 차트로 만들어 줘"처럼 요청할 수도 있습니다. 이 기능은 채팅과 Cowork 양쪽에서 베타로 동작합니다.

**자주 쓰이는 경우**

- 작업 흐름 정리(플로차트)
- 데이터 분석(차트)
- 의사결정 비교
- 개념 설명(다이어그램)

**사용 방법**

1. 시각 설명이 어울리는 요청을 하거나 "다이어그램으로 그려 줘"처럼 명시적으로 요청
2. Claude가 답변의 일부로 인터랙티브 시각 자료를 생성
3. 버튼 클릭, 슬라이더 조정, 전체 화면 확대, 후속 질문 등으로 상호작용
4. 보존하려면 아래 세 가지 중 선택

**시각 자료 보존하기**

| 방법 | 형식 |
|---|---|
| 이미지로 복사 | 클립보드 이미지 |
| 다운로드 | .svg 또는 .html 파일 |
| 아티팩트로 저장 | 계속 다듬을 수 있는 형태 |

{{< hint type="warning" >}}
커스텀 시각 자료는 베타라 품질과 복잡도가 들쭉날쭉할 수 있고 기본적으로 저장되지 않으므로 직접 보존해야 합니다. iOS·Android 앱에서는 쓸 수 없습니다. Cowork에서는 공유 링크로 시각 자료가 보이지 않고 시각 자료 안을 클릭해 후속 질문을 할 수 없어 직접 질문을 입력해야 합니다. 복잡한 시각화에는 Opus 같은 상위 모델이 권장됩니다.
{{< /hint >}}

### 날씨·레시피·스포츠 같은 시각 위젯

Claude는 날씨, 레시피 카드, 스포츠 점수 같은 데이터 위젯도 보여 줍니다.

| 콘텐츠 | 특징 | 플랫폼 |
|---|---|---|
| **날씨** | 현재 날씨·예보, 미국은 화씨·그 외 지역은 섭씨 | 웹·데스크톱만 |
| **레시피** | 인분 조정·단위 변환·단계별 안내가 있는 카드 | 웹·데스크톱만 |
| **스포츠** | 미국 프로 리그(NFL·NBA·NHL·MLB·WNBA·MLS), NCAA, 국제 축구, 테니스·골프 등 | 모든 플랫폼에서 텍스트만 |
| **인터랙티브 입력** | 객관식·다중 선택·순위 매기기 등 클릭형 선택지 | 질문 유형에 따라 다름 |

{{< hint type="note" >}}
데이터 위젯이 동작하려면 웹 검색이 켜져 있어야 하며 인터넷 연결이 필요합니다. 모바일(iOS·Android)에서는 날씨·레시피가 텍스트로만 표시되고 스포츠 데이터는 모든 플랫폼에서 텍스트로만 표시됩니다.
{{< /hint >}}

## 파일 업로드와 분석

문서나 이미지를 올리면 Claude가 읽고 분석합니다. 올리는 곳이 일반 채팅인지 프로젝트인지에 따라 한도와 처리 방식이 다릅니다.

| 구분 | 채팅 업로드 | 프로젝트 업로드 |
|---|---|---|
| 파일당 최대 크기 | 500MB | 30MB |
| 개수 제한 | 대화당 최대 20개 | 제한 없음(컨텍스트 창에 따라) |
| 처리 방식 | 시각 요소 포함 분석 | 텍스트 추출만 |
| 이미지 | 최대 8000×8000 픽셀 | — |

**지원 형식**

- 문서: PDF, DOCX, CSV, TXT, HTML, ODT, RTF, EPUB, JSON, XLSX
- 이미지: JPEG, PNG, GIF, WebP (1000×1000 픽셀 이상 권장)

**올리는 방법**

1. 채팅 입력창 왼쪽 아래 **'+'** 버튼 클릭
2. **파일 또는 사진 추가(Add files or photos)** 선택
3. 기기에서 파일을 고르거나 채팅 창으로 드래그 앤 드롭
4. **열기(Open)** 클릭, 또는 클립보드에서 이미지를 바로 붙여넣기

{{< hint type="note" >}}
PDF는 시각 요소·차트를 100페이지까지 분석하고 1,000페이지가 넘으면 텍스트 추출로 처리합니다. PDF를 언급할 때는 인쇄된 쪽번호 대신 실제 PDF 뷰어의 페이지 번호를 기준으로 하세요. XLSX 파일은 코드 실행이 켜져 있어야 합니다. 프로젝트 업로드는 파일당 30MB, 텍스트 추출만 지원하므로 멀티모달 PDF는 제외됩니다.
{{< /hint >}}

## 파일 생성 — Excel, PowerPoint, Word, PDF 만들기

Claude는 코드를 실행해 바로 내려받을 수 있는 전문 문서를 만들어 줍니다. 데이터를 던져 주면 표가 들어간 엑셀로, 요약을 주면 발표용 슬라이드로 정리하는 식입니다.

| 만들 수 있는 형식 | 설명 |
|---|---|
| **.xlsx** | 수식이 들어간 Excel |
| **.pptx** | PowerPoint 슬라이드 |
| **.docx** | Word 문서 |
| **PDF** | PDF 문서 |
| **Python 스크립트** | 실행 가능한 코드 |
| **PNG** | 시각화 이미지 |

- 최대 파일 크기: 업로드·다운로드 각각 30MB
- 사용 가능: 웹, Claude Desktop, Claude Mobile(iOS·Android)
- 요금제: Free / Pro / Max(기본 켜짐), Team(기본 켜짐, 소유자가 끌 수 있음), Enterprise(신규 조직 기본 켜짐, 소유자 조정 가능)

**켜는 방법**

1. **설정(Settings)** → **기능(Capabilities)** 으로 이동
2. **코드 실행 및 파일 생성(Code execution and file creation)** 토글 켜기
3. Team·Enterprise는 필요 시 네트워크 접근 단계를 설정(차단 / 패키지 매니저만 / 사용자 지정 도메인 화이트리스트)
4. 대화에서 원하는 형식의 파일 생성을 요청하면 만들어진 즉시 다운로드

{{< hint type="warning" >}}
파일 생성은 격리된 샌드박스에서 실행되며 일반 사용 한도를 함께 소비합니다. 파일 아티팩트의 공개 공유는 비활성화되어 있습니다. 외부 파일이나 웹사이트에 악의적 지시가 숨겨져 Claude를 속일 위험이 있어, 프롬프트 인젝션 분류기 탐지와 사용자 간 샌드박스 격리로 대응합니다. Team·Enterprise는 네트워크 접근이 기본 차단이며 조직 소유자가 설정해야 합니다.
{{< /hint >}}

## 수식과 계산

Claude는 수식을 이해하고 계산을 수행할 수 있습니다. 풀이 과정을 함께 보여 달라고 하면 단계별로 설명해 줍니다.

{{< hint type="warning" >}}
복잡하거나 중요한 계산은 Claude의 결과만 믿지 말고 전용 수학 소프트웨어나 직접 검산으로 따로 확인하는 것이 안전합니다.
{{< /hint >}}

## 코드 설명

낯선 코드를 이해하거나 버그를 찾을 때 코드를 붙여넣고 물어보면 됩니다.

**사용 시기**

- 낯선 코드를 이해하고 싶을 때
- 코드의 버그를 찾을 때
- 코드를 다른 언어로 옮기고 싶을 때

**예시**

```
사용자가 Python 코드 붙여넣기
"이 코드가 하는 일을 설명해 줄 수 있나요?"
→ Claude가 코드를 라인별로 설명
```

## Claude Security — 코드 취약점 점검(Enterprise 베타)

Claude Security는 Claude.ai에 내장된 기능으로 코드의 보안 취약점을 찾아내고 수정 패치를 제안합니다. 현재 **Enterprise 요금제** 사용자에게 공개 베타로 제공되며 여러 파일에 걸친 데이터 흐름을 추적해 복잡한 취약점 패턴까지 식별합니다.

- 탐지 범위: SQL·명령어·코드·XSS 인젝션, XXE·ReDoS, 경로 탐색·SSRF·오픈 리다이렉트, 인증·접근 제어 결함, 메모리 안전성, 암호화 취약점, 역직렬화 취약점, 프로토콜·인코딩 문제
- 심각도: High(인증 없이 원격 악용 가능) / Medium(인증 또는 1~2개 전제 필요) / Low(3개 이상 전제 또는 로컬 접근 필요)
- 발견된 취약점은 Claude Code 세션으로 바로 이어 받아 수정 가능

{{< hint type="note" >}}
Claude Security는 GitHub 호스팅 저장소만 지원하고 Enterprise 요금제가 필요합니다. 공개 베타라 기능이 바뀔 수 있고 심각도는 사용자가 바꿀 수 없으며 스캔 결과는 비결정적(실행마다 달라질 수 있음)입니다.
{{< /hint >}}
