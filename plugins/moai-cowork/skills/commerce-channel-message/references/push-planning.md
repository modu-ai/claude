# 앱 푸시 기획 가이드 (Push Mode)

채널 메시지 엔진의 **앱 푸시 모드** 상세 참고자료. "앱 푸시 문구 만들어줘", "리텐션 푸시 카피" 같은 발화 시 이 가이드로 4원칙 + 3요소 + 변형 3안 + 클릭률 예측을 적용한다. 짧은 잠금화면 카피는 push-copy.md(카테고리별 카피 패턴·발송 시점·빈도), 기획 프레임은 본 문서를 함께 본다.

## 푸시 4원칙 (발송 전 4질문)

| 원칙 | 질문 | 점검 기준 |
|------|------|----------|
| **WHY** (왜) | 발송 목적은 4분류 중 어디? | 서비스 노출 / 할인·이벤트 / 인지도 / 리텐션 |
| **WHEN** (언제) | 행동 또는 시간 트리거? | 행동 후 3분/3시간/3일 / 데일리 골든타임 / 시즌·이슈 |
| **WHO** (누구에게) | 세그먼트가 정의됐는가? | 전체 회원 / 코호트 / 1:1 개인화 |
| **HOW** (어떻게) | 톤앤매너가 명확한가? | 숫자 · 게이미피케이션 · 브랜딩 · 이모지 정책 |

> 4원칙 미충족 시 발송 금지 — 하나라도 답하지 못하면 차단. 무차별 푸시는 옵트아웃(앱 알림 OFF) 비율을 높여 장기 ROI를 깎는다.

## 3요소 자가 점검 (Timely · Personal · Actionable)

- **Timely**: 받았을 때 즉각 행동 가능한 상황인가? (출근 7-9시 / 점심 12-13시 / 퇴근 18-20시 / 자기 시간 22-24시)
- **Personal**: 세그먼트·이름·구매 이력 반영? (전체 < 세그먼트 < 1:1)
- **Actionable**: 다음 액션이 명확한가? (탭 위치 + 시간 + 혜택)
  - ❌ "신상품 입고" → 무엇을? 어떻게?
  - ✅ "지금 탭 → 신상 30% 한정 진입 (10:00까지)"

> 3요소 모두 충족 시 클릭률 1.5% → 4-8% (2-5배 상승).

## 발송 목적 4분류 매트릭스

| 목적 | 핵심 KPI | 발송 빈도 | 한국 베스트 사례 |
|------|---------|---------|-----------------|
| **① 서비스 노출** | DAU·앱 진입율 | 주 2-3회 | 배민 "오늘 점심 뭐 드세요?" / 카카오헤어샵 "이번 주 인기 헤어샵 TOP 5" |
| **② 할인·이벤트** | 즉각 매출·CTR | 캠페인 단위 | 쿠팡 "로켓와우 멤버 추가 10%" / 에이블리 "오늘만 20%" / 지그재그 "퇴근 후 30분 한정" |
| **③ 인지도** | 브랜드 호감도·검색량 | 월 1-2회 | 토스 "송금하면 1억 추첨" / 오늘의집 "이번 주 인기 인테리어" / 클래스101 "신규 강의 50% 오픈" |
| **④ 리텐션** | 재방문·재구매·휴면 복귀 | 사용자 행동 기반 | 인프런 "마지막 학습 14일 전 — 이어 듣기" / 야놀자 "찜한 숙소 5만원 떨어졌어요" / 퍼블리 "지난주 읽다 만 글 이어보기" |

## 비즈니스 타입 3분류 톤앤매너

| 타입 | 톤앤매너 특성 | 한국 사례 |
|------|--------------|----------|
| **D2C 브랜드** (스킨케어·패션·식품) | 정체성·라이프스타일·감성 | 29CM·아리따움·콜린스·사운드짐 |
| **마켓플레이스** (가격·다양성·즉시 구매) | 숫자·게이미피케이션·CTA 강함 | 쿠팡·올웨이즈·당근마켓·요기요·쿠팡이츠 |
| **결제/구독** (안정·신뢰·습관 형성) | 절제·명확·기능 강조 | 토스·리디·넷플릭스·피파온라인·듀오링고 |

## 카피 변형 3안 — 자동 생성 축

같은 발송 목적에서도 표현을 3안으로 변형해 A/B 테스트한다.

### 변형 축 1: 오늘만 vs 매일
| 변형 | 패턴 | 클릭률 효과 |
|------|------|-----------|
| 오늘만 (긴급성) | "오늘만 20% — 자정 마감" | 즉각 클릭 ↑ but 반복 시 신뢰 ↓ |
| 매일 (안심성) | "매일 갱신되는 추천 상품" | 클릭률 균일 + 옵트아웃률 ↓ |

### 변형 축 2: 누구나 vs 너에게만
| 변형 | 패턴 | 클릭률 효과 |
|------|------|-----------|
| 누구나 (일반) | "전 회원 20% 할인" | 베이스라인 1.5% |
| 너에게만 (개인화) | "지연 회원님 한정 — 어제 본 그 세럼 20%" | 4-8% (개인화 효과) |

### 변형 축 3: 숫자 vs 게이미피케이션 vs 브랜딩
| 변형 | 패턴 | 한국 사례 |
|------|------|----------|
| **숫자** | "오늘만 30% / 3시간 남음" | 쿠팡·올웨이즈 (마켓 스타일) |
| **게이미피케이션** | "🎮 출석 5일째 — 다음 보상 ★★★" | 토스·리멤버·듀오링고 |
| **브랜딩** | "오늘 저녁, 우리 잔에는 어떤 향이 어울릴까요?" | 29CM·아리따움·사운드짐 (감성 톤) |

## 클릭률 예측 공식

```
기본: 1.5%
+ 3요소 충족: +2.5~4.5%p
+ 개인화 추가: +1~3%p
+ 게이미피케이션 추가: +0.5~1.5%p
- 야간 발송 (정통망법 동의 필수): -0.5~1%p (옵트아웃 증가)
```

## 한국 30+ 브랜드 푸시 레퍼런스

- **D2C (정체성·감성)**: 29CM, 아리따움, 콜린스, 사운드짐, 클래스101, 인프런, 퍼블리, 브런치, 오프린트미, 리디
- **마켓플레이스 (숫자·CTA)**: 쿠팡, 쿠팡이츠, 요기요, 배민, 올웨이즈, 당근마켓, 지그재그, 에이블리, 브랜디, 야놀자
- **결제/구독 (절제·기능)**: 토스, 카카오페이, 넷플릭스, 피파온라인, 듀오링고, 리멤버
- **콘텐츠·기타 (게이미피케이션)**: 라운즈, 네이버웹툰, 카카오웹툰, 인스타그램, 카카오톡

    "who": "최근 60일 미접속 구매 이력 회원 (휴면 직전)",
    "how": "브랜딩 톤 — 정체성 환기"
  },
  "three_factors": {
    "timely": "PASS — 20시는 자기 콘텐츠 소비 골든타임",
    "personal": "PASS — 이름 + 마지막 구매 상품 + 재구매 시점 반영",

  "copy_variants": [
