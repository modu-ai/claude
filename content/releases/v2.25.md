**릴리스 날짜**: 2026-06-19
**버전**: v2.25.0 (MINOR, 최신)
**업데이트 명령**: `/plugin marketplace update cowork-plugins`

## 하이라이트

**"PDF로 생성"이 `moai-office:pdf-writer`로 안정적으로 라우팅되도록 개선하고, 28개 플러그인의 스킬·에이전트·커넥터 지침을 전수 감사해 정정했습니다.**

- **pdf-writer 미발동 근본 해소** — 사용자가 "PDF로 생성해줘"라고 해도 스킬이 발동하지 않고 Claude가 직접 weasyprint를 설치해 처리하던 문제를, (1) 트리거를 일반 표현으로 대폭 보강, (2) "weasyprint 직접 설치 말고 이 스킬 사용" 포지셔닝, (3) **weasyprint 단일 엔진 재작성**(스타일 HTML 리포트 디자인 보존 + 번들 Noto Sans CJK `@font-face` 임베딩)으로 해소했습니다.
- **산출물 스킬 발동 강화** — office(docx·pptx·xlsx·hwpx) "Claude 기본 생성 대신 이 스킬 사용" 포지셔닝, media(higgsfield-image/video) 생성 포지셔닝, html-report·html-slide → pdf-writer **PDF 핸드오프** 신설.
- **전수 콘텐츠 감사 정정** — 끊긴 교차참조, deprecated 참조(`social-media`→`sns-content`), 미번들 MCP 호출(sequential-thinking 조건부 fallback), 노후 사실(Core Web Vitals FID→INP, G마켓/옥션 운영사 지마켓, 나라장터 웹표준, 자살예방 위기상담 109), 오탈자를 정정. 세법·개인정보보호법 과징금·4대보험 요율 등 미검증 도메인 수치는 **공식 출처 참조로 완화**(새 수치 단정 회피).
- **리다이렉트 stub 2개 제거** — `moai-core:ai-diagnostic`, `moai-education:course-curriculum-design`. user-invocable 리다이렉트 전용 스킬로 `/ai-diagnostic` 이름 충돌·카운트 부풀림을 유발하던 것을 제거. 178 → 176 스킬.
- **31 코디네이터 에이전트 정합** — 전 플러그인 번들 코디네이터의 모델을 `inherit`(메인 세션 사용자 선택 모델 승계)로 통일하고 역할별 `effort`를 부여. `/project` 생성 에이전트 기본 도구에서 Bash 제거(Cowork-safe).

## 사용 예시

```
이 HTML 리포트를 디자인 그대로 PDF로 만들어줘.
주간 보고서 마크다운을 한글 깨짐 없는 PDF로 생성해줘.
```

위 요청은 이제 weasyprint를 별도로 설치하지 않고 `pdf-writer` 스킬이 일관되게 처리합니다.
