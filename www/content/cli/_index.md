터미널에서 Claude Code와 MoAI-ADK를 다루는 개발자를 위한 축입니다. 명령어 한 줄로 코드를 짜고, 테스트를 돌리고, 개발 사이클 전체를 자동화하는 세계 — 그 심화 학습은 이제 전용 사이트에서 이어집니다.

## 최신 심화 문서는 여기서 보세요

CLI 개발 문서는 한국어 클로드 코드 공식 안내 사이트로 통합되었습니다. 이 페이지의 예전 하위 문서(시작하기·핵심 개념·일상 사용·MoAI-ADK·레퍼런스)를 북마크해 두셨다면, 아래 대응 섹션에서 같은 주제의 최신 문서를 찾을 수 있습니다.

**[→ 클로드 코드 한국어 안내 문서 (adk.mo.ai.kr/ko)](https://adk.mo.ai.kr/ko)**

| 예전 이 사이트 문서 | 이어서 볼 곳 |
|---|---|
| 시작하기 (설치·첫 SPEC) | [adk.mo.ai.kr/ko](https://adk.mo.ai.kr/ko) 시작 가이드 |
| 핵심 개념 (SPEC·DDD/TDD·TRUST 5·하네스) | [adk.mo.ai.kr/ko](https://adk.mo.ai.kr/ko) 개념 문서 |
| 일상 사용 (작업 흐름·프롬프트·토큰·디버깅) | [adk.mo.ai.kr/ko](https://adk.mo.ai.kr/ko) 사용 가이드 |
| MoAI-ADK (브리지·워크플로우·품질 명령어) | [adk.mo.ai.kr/ko](https://adk.mo.ai.kr/ko) MoAI-ADK 문서 |
| 레퍼런스 (명령어 색인·다중 LLM·고급) | [adk.mo.ai.kr/ko](https://adk.mo.ai.kr/ko) 레퍼런스 |

데스크탑 앱에서 코드를 다루는 방법이 궁금하다면 [사용법 가이드의 코드 트랙](/guide/code/)부터 시작하세요. 개발이 처음이라면 코드 트랙 → CLI 순서를 권합니다.
