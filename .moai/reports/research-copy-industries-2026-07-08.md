# 리서치 보고서: 한국어 AI-tell 카피 패턴(A-20~A-25) 업종 확장 소재

> 수집: 2026-07-08, research-copy 에이전트 (WebSearch/WebFetch 교차검증)
> 용도: ai-tell-taxonomy.md A-20~A-25 다업종 예시 확장 + scholarship.md 출처 계보

## 0. 방법론 노트

- "실사용 소재"는 WebSearch/WebFetch로 확인된 **실제 게시·출간된 슬로건/카피**. AI-tell 패턴(A-20~A-25)은 정의상 "부자연스러운 실패 사례"이므로, 출간된 프로 카피에는 거의 나타나지 않음 — 실제 소재는 대부분 **"자연스러운 after 예시"** 앵커로 쓰이고, 대응되는 **"AI틱 before"는 패턴 정의에 따라 직접 구성(constructed)**. 모든 constructed 항목에 `[구성]` 표시, 실사용 확인 소재에 `[출처확인]`+출처 번호.
- A-20(기계적 동사 축자역), A-24("더는" 문두), A-25(3인칭 설명체) 관련해 이 정확한 프레이밍으로 다룬 한국어 자료는 미발견 — 스킬 자체 관찰 세분류로 보이며 §4(Gaps)에 명시.
- 교차검증 기준: 최소 2개 독립 출처가 뒷받침하는 주장만 "확인됨", 단일 출처는 "single-source" 명시.

## 1. 업종별 발견 사항

| 업종 | 자연스러운 카피 관습 | AI틱 실패 양상 | 예시 소재 |
|---|---|---|---|
| **뷰티/코스메틱** | 여러 장점 나열보다 **단일 핵심 키워드**에 집중(확인됨, 출처①④). 유통 채널(로드샵/올리브영/백화점)별로 톤이 갈림(single-source, 출처①). "당신/Your"류 2인칭 소구가 흔함(확인됨, 출처②④) | 특장점을 한 문장에 다 욱여넣는 나열형, 3인칭 설명문("이 제품은 ~한 효과가 있습니다") | Florence by Mills "Love the Skin You're In"[출처확인②④], Bobbi Brown "Be Who You Are"[출처확인①] |
| **식품/F&B** | **언어유희·중의적 표현**(확인됨, 출처③⑤), 소비자 관점 구어체·스토리텔링(확인됨, 출처③), 짧고 정서적 공감형 문장 | 판매자 시점 설명체, 감성 없는 스펙 나열 | 옥수수수염차 "다짜고짜 어떡해… 다,짜고,짠 한국인의 음식엔"[single-source, 출처③], 다시다 "잊을 수 없는 고향, 고향의 맛"[출처확인⑤], 카누 "오늘 나를 웃게 한 건 무엇이었나요?"[출처확인⑥] |
| **금융/핀테크** | 브랜드별 **한 단어 압축 전략** — 토스="쉽다/연결", 카카오페이="마음을 놓다(안심)", 카카오뱅크="다름"(확인됨, 출처⑦⑧ 2개 독립 일치) | 정책·기능 나열형 설명체, 신뢰감 없는 사무적 어투 | 토스 "금융이 쉬워진다"[출처확인⑥], 카카오페이 "마음 놓고 금융하다"[출처확인⑦], 캐롯손해보험 "당신의 일상이 보험의 기준이 됩니다"[출처확인⑥](2인칭+"~이 됩니다" 종결 실사용례) |
| **SaaS/B2B** | Before→After 구조 + **구체적 수치**(확인됨, 출처⑨ single-source), 페인포인트→해결 스토리 구조 | "자동화가 굴러갑니다"류 3인칭 기계적 서술, 뭉뚱그린 효율 주장 | "매주 5시간씩 수동으로 만들던 리포트, 이제 버튼 클릭 한 번으로 10분 만에 완성됩니다"[single-source, 출처⑨] — "이제"(더는 아님) 자연 대비의 실사용 앵커 |
| **교육/에듀테크** | 브랜드 정체성 압축형 슬로건(확인됨, 출처⑩⑫ 2개 독립), 최근 유머·라임 활용 캐주얼 톤 증가 | 딱딱한 "OO를 제공합니다"류 기관 설명체 | 스픽 "틀려라, 트일 것이다"[출처확인⑥], 야나두 "야 너두 영어할 수 있어"[출처확인⑥], 눈높이 컴퓨터 "1:1로 방문관리 해주는 컴퓨터 학습"[출처확인⑤⑩] |
| **이커머스/앱** | 한국어는 **명사형 "~하기" CTA**가 표준(영어식 1-3단어 초단축 아님)(출처⑪ single-source, 문법적 근거 탄탄), 대화체·개인화("당신을 위한") 소구(확인됨, 출처⑥⑪) | 영어 CTA 직역 명사 축약("구매" 단독 등), 3인칭 기능 설명 | 당근 "당신 근처의 당근"[출처확인⑥], 번개장터 "취향을 잇는 거래"[출처확인⑥], 에어비앤비 "여행은 살아보는 거야"[출처확인⑥] |
| **공공/기관** | 정책 취지 압축형 슬로건 존재(single-source, 출처⑬), 상업 카피 대비 **개조식/음슴체** 잔존(출처⑭ single-source, 정황 일치 다수) | 개조식·음슴체 딱딱함, 다정책 압축 장황함, 신뢰 없는 2인칭 호명의 냉소적 밈화 위험("국민 여러분 안심하십시오")[single-source, 출처⑬] | 국가보훈부 "국민통합에 기여하는 책임보훈"[출처확인⑬] |

## 2. 패턴별(A-20~A-25) 업종 태그 Before→After 시드

> 표기: `[구성]` = 패턴 정의에 따라 새로 구성한 시드(실제 출처 없음) · `[출처확인-after]`/`[출처확인-anchor]` = after 문구가 실사용 확인 소재에 앵커링됨

### A-20 — 기계적 동사 축자역 ("굴러가다/굴리다" ← run/roll)
(이 정확한 프레임을 직접 다룬 한국어 자료 미발견 — 자체 관찰 기반)

| 업종 | Before (AI틱) | After | 비고 |
|---|---|---|---|
| SaaS/B2B | "이 자동화 시스템은 24시간 굴러갑니다" [구성] | "이 자동화는 24시간 알아서 작동해요" [구성] | run→굴러가다 축자 대응 |
| 제조/물류 | "전체 공급망이 하나의 톱니바퀴처럼 굴러갑니다" [구성] | "전체 공급망이 하나처럼 맞물려 돌아갑니다" [구성] | roll/run 기계 은유 중첩 |
| 금융/핀테크 | "당신의 자산 관리가 스스로 굴러갑니다" [구성] | "당신의 자산 관리를 대신 챙겨드려요" [구성] | 토스류 "쉽다" 톤(출처⑦) 대비 |
| 교육 | "학습 루틴이 자동으로 굴러가도록 설계했습니다" [구성] | "학습 루틴이 저절로 잡히도록 설계했습니다" [구성] | 시스템 주어+자동사 남용이 어색함의 근원 |
| 공공 | "정책 집행이 원활히 굴러가고 있습니다" [구성] | "정책이 차질 없이 시행되고 있습니다" [구성] | 개조식 톤(출처⑭) 결합 시 이중 어색 |

### A-21 — 추상명사 헤드라인 종결 ("~이 됩니다" ← be/become + noun)
실사용 after 앵커: 캐롯손해보험 "당신의 일상이 보험의 기준이 **됩니다**"[출처확인⑥]

| 업종 | Before (AI틱) | After | 비고 |
|---|---|---|---|
| 뷰티 | "당신의 루틴이 하나의 흐름이 됩니다" [구성] | "아침 루틴이 5분으로 끝나요" [구성] | 추상명사("흐름") 종결 vs 구체 결과 |
| 식품 | "이 한 끼가 당신의 하루의 시작이 됩니다" [구성] | "이 한 끼로 하루를 시작하세요" [구성] | 다시다류 구체 정서(출처⑤) 대비 |
| SaaS/B2B | "업무가 하나의 경험이 됩니다" [구성] | "업무 하나가 이렇게 쉬워질 줄 몰랐어요" [구성] | 원 편중 도메인 패턴 |
| 금융 | "자산 관리가 습관이 됩니다" [구성] | "자산 관리가 매일 하는 루틴이 돼요" [출처확인-anchor⑥] | 캐롯 실사용례 구조 차용 |
| 공공 | "국민의 삶이 더 나은 미래가 됩니다" [구성] | "국민의 삶이 더 나아집니다" [구성] | 개조식 추상 종결(출처⑭) 전형 |

### A-22 — 수혜자 구문 축자역 ("나 대신/나와 함께 일합니다" ← works for/with me)
(전부 constructed — 이 프레임을 다룬 실사용 자료 없음)

| 업종 | Before (AI틱) | After | 비고 |
|---|---|---|---|
| SaaS/B2B | "이 도구가 저 대신 일합니다" [구성] | "이 도구 하나면 제 손이 덜 가요" [구성] | works for me 축자역 |
| 금융 | "저와 함께 일하는 자산관리 앱" [구성] | "제 자산을 같이 챙겨주는 앱" [구성] | 뱅크샐러드류 대화체(출처⑥⑦) 대비 |
| 뷰티 | "이 세럼이 제 피부를 위해 일합니다" [구성] | "이 세럼 하나로 피부가 편해졌어요" [구성] | works for my skin 축자역 |
| 이커머스 | "이 앱이 당신 대신 장을 봅니다" [구성] | "장보기, 이제 이 앱이 대신해요" [구성] | 3인칭 주어 축소, 결과 중심 |
| 교육 | "이 커리큘럼이 여러분과 함께 일합니다" [구성] | "이 커리큘럼, 여러분 페이스에 맞춰 갑니다" [구성] | works with you 축자역 |

### A-23 — 은유 축자역 ("X의 엔진/심장/날개", "손발을 맞추다", "같은 페이지")
("같은 페이지에 있다"(on the same page) 번역투 지적은 통용되나 single-source. "엔진/심장" 클리셰 직접 비평 자료 미발견)

| 업종 | Before (AI틱) | After | 비고 |
|---|---|---|---|
| SaaS/B2B | "이 API가 우리 시스템의 심장입니다" [구성] | "이 API 없인 아무것도 안 돌아가요" [구성] | heart/engine 축자역 |
| 제조 | "이 부품이 생산라인의 엔진 역할을 합니다" [구성] | "이 부품이 멈추면 라인 전체가 서요" [구성] | engine → 인과관계 서술 치환 |
| 팀 협업툴 | "이제 모두가 같은 페이지에 있습니다" [구성] | "이제 다들 같은 걸 보고 얘기해요" [구성] | on the same page 축자역 |
| 조직문화 | "팀이 손발을 맞추듯 협력합니다"(한국어 고유 관용구 — after 후보) [구성] | (대비) "팀이 하나의 유기체처럼 움직입니다" [구성] | 고유 관용구 vs 수입 은유 대비 |
| 패션/뷰티 | "이 컬렉션은 브랜드의 날개입니다" [구성] | "이 컬렉션으로 브랜드가 한 단계 커졌어요" [구성] | wings 축자역 |

### A-24 — 문두 "더는" (← "no longer", 자연스러운 한국어는 "이제는")
SaaS 실사용례(출처⑨) "**이제** 버튼 클릭 한 번으로"가 자연 대비 표현의 실사용 앵커.

| 업종 | Before (AI틱) | After | 비고 |
|---|---|---|---|
| SaaS/B2B | "더는 반복 업무에 시간을 낭비하지 마세요" [구성] | "이제 반복 업무는 버튼 하나로 끝나요" [출처확인-anchor⑨] | no longer 축자역 vs 실사용 "이제" |
| 뷰티 | "더는 여러 단계를 거칠 필요가 없습니다" [구성] | "이제 한 단계면 충분해요" [구성] | |
| 금융 | "더는 복잡한 서류가 필요 없습니다" [구성] | "이제 서류 없이 3분이면 끝나요" [구성] | 토스류 "쉽다" 톤(출처⑦) 결합 |
| 교육 | "더는 어려운 영어에 좌절하지 마세요" [구성] | "이제 영어, 야 너두 할 수 있어" [출처확인-anchor⑥, 야나두 패러디] | |
| 이커머스 | "더는 줄 서서 기다릴 필요가 없습니다" [구성] | "이제 줄 설 필요 없이 3초 결제" [구성] | |

### A-25 — 3인칭 설명체·2인칭 소구 결여 ("자동화가 ~합니다" vs "여러분이/여러분의")
2인칭 소구는 뷰티·보험·생활플랫폼에서 강하게 검증(출처①②④⑥). 3인칭 설명체는 공공 개조식(출처⑭) + ChatGPT 생성 카피의 뉴스기사체(2개 독립 스니펫 일치, 출처⑯)로 확인 — **패턴 자체 confirmed**.

| 업종 | Before (3인칭 설명체, AI틱) | After (2인칭 소구) | 비고 |
|---|---|---|---|
| SaaS/B2B | "자동화 시스템이 반복 업무를 처리합니다" [구성] | "여러분의 반복 업무, 이 자동화가 대신 처리해드려요" [구성] | 원 편중 도메인 원본 패턴 |
| 뷰티 | "이 세럼은 피부 장벽을 강화합니다" [구성] | "여러분의 피부 장벽, 이 세럼이 지켜드려요" [출처확인-톤anchor②④] | Florence by Mills류 2인칭 톤 |
| 금융 | "이 앱은 자산 현황을 통합 관리합니다" [구성] | "여러분의 자산, 한 곳에서 다 보여드려요" [출처확인-톤anchor⑥⑦] | 뱅크샐러드 대화체 유사 구조 |
| 교육 | "이 커리큘럼은 학습 격차를 해소합니다" [구성] | "여러분의 학습 격차, 이 커리큘럼이 채워드려요" [구성] | |
| 공공 | "본 정책은 청년 고용률을 개선합니다" [구성] | "여러분의 첫 취업, 이 정책이 함께합니다" [구성] | 국가보훈부류 3인칭 정책 서술(출처⑬) 대비 |
