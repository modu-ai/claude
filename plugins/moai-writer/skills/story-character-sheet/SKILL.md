# story-character-sheet

> 캐릭터의 설정과 비주얼을 한 시트로 정리하고 Soul ID로 학습시키는 스킬. 학습된 ID는 webtoon-art·conti·cover-art에서 인물 일관성에 재사용된다.

## 1. 개요

이 스킬은 **무엇을** 만드는가 — 캐릭터 설정서 + Soul ID 학습 세트. **언제** 필요한가 — 인물이 등장하는 모든 생성형 스킬의 선행 단계.

## 크레딧 고지 (사전 고지 의무)

이 작업은 **Higgsfield 크레딧을 소모**합니다. 생성 전 예상 소모량을 사용자에게 고지하고 확인을 받는다.

| 항목 | 예상 크레딧 |
|------|-----------|
| 캐릭터 학습 20장 세트 (Soul ID) | 약 10 크레딧 |

진행하기 전 사용자에게 "위 크레딧이 소모됩니다. 계속 진행할까요?"로 확인한다. 사용자가 거부하면 생성을 중단하고 프롬프트 온리 모드로 전환한다.

## 모델 라우팅 표

| 용도 | 1순위 Higgsfield 모델 |
|------|---------------------|
| 캐릭터 일관성 학습 | Soul ID (다각도 이미지 세트 → 인물 ID) |
| 학습용 시드 이미지 | AI Anime Generator · Soul 2.0 |

## MCP 미연결 폴백 (프롬프트 온리 모드)

Higgsfield MCP에 연결할 수 없을 때는 생성을 생략하고 완성 프롬프트를 텍스트로 출력한다. 출력 끝에 "Higgsfield 웹(https://higgsfield.ai)에 위 프롬프트를 붙여넣으세요" 안내를 추가한다. 서버 과부하 시 잠시 대기 후 재시도 안내를 함께 표시한다.

## 3. 워크플로우

### Step 1: 설정 정리
성격·외형·관계·갈등 동기·말투를 텍스트로 정리.

### Step 2: 시드 이미지 확보
기존 일러스트 또는 AI Anime Generator/Soul 2.0으로 시드 1~3장을 생성.

### Step 3: 크레딧 고지 + 확인
학습 장수(권장 20장) × 크레딧을 고지하고 승인받는다.

### Step 4: Soul ID 학습
시드로 다각도(정면·측면·반쯔렴·표정 변화) 세트를 생성해 Soul ID에 학습.

### Step 5: ID 발급·공유
발급된 캐릭터 ID를 webtoon-art·conti·cover-art 프롬프트에 사용.

## 4. 주의사항

- 학습 세트가 부족하면 일관성이 깨진다. 표정·각도 다양성을 확보.
- 복수 인물은 인물마다 별도 Soul ID 학습.

## 5. 관련 스킬

- After: `story-webtoon-art` · `story-conti` · `story-cover-art` (ID 재사용)
