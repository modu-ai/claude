**릴리스 날짜**: 2026-06-19
**버전**: v2.26.0 (MINOR)
**업데이트 명령**: `/plugin marketplace update cowork-plugins`

## 하이라이트

**Claude Cowork 공식 가이드라인에 맞춰 28개 플러그인 매니페스트를 현대화하고 고아 스킬 발견성을 개선했습니다.**

- **displayName 현대화** — 28개 `plugin.json`에 한글 `displayName` 추가. Cowork 플러그인 목록에서 친숙한 한글 이름으로 표시됩니다.
- **고아 스킬 매핑** — 코디네이터 에이전트가 라우팅하지 않던 orphan 스킬 46개를 발견·범주화. `moai-commerce` 12개를 포함해 코디네이터 매핑 보강으로 자동 발동 확률 향상.
- **`moai-media` 기본값 조정** — `defaultEnabled: false` 설정으로 Higgsfield·ElevenLabs API 키 없이 설치해도 에러 없이 동작.
- **설계 문서 신설** — `.moai/docs/` 에 플러그인 아키텍처 설계 문서(instructions 전략, connector 전략) 추가. 스킬 전용 의도 문서화.

## 사용 예시

업데이트 후 Cowork 플러그인 목록에서 한글 이름으로 플러그인을 더 쉽게 찾을 수 있습니다.

## 업그레이드 방법

```
/plugin marketplace update cowork-plugins
```

플러그인 상세 재진입 시 반영됩니다.

## 호환성

**기능적 비파괴** — 기존 워크플로우·스킬 호출 무변경. 매니페스트 메타데이터·발견성 개선이 전부입니다.

- 28 플러그인 · 176 스킬 유지
- Breaking change 없음
