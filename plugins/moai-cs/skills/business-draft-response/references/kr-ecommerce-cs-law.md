# 한국 이커머스 CS 법정 기준 요약 (kr-ecommerce-cs-law)

CS 응답 초안의 "해결책" 단계에서 환불·교환·반품 관련 안내를 작성할 때 참조하는 법정 기준 요약입니다. 고객에게 법정 기준보다 불리한 안내를 하면 그 안내 자체가 무효이며 분쟁 리스크가 됩니다.

> ⚠️ 본 문서는 법률 자문이 아닌 응대 참고 자료입니다. 분쟁이 임박한 건은 법무 검토(`moai-lawyer`)로 에스컬레이션하십시오. 법령·고시는 개정될 수 있으므로 응답에 기준일을 명시하고, 최신 조문은 국가법령정보센터에서 확인합니다.

## 1. 청약철회 (단순 변심 반품) — 전자상거래법 제17조

| 항목 | 기준 | 근거 |
|------|------|------|
| 기본 기간 | **재화를 공급받은 날(수령일)로부터 7일 이내** | 전상법 제17조 제1항 |
| 계약 내용 서면(공급서 등) 미교부·지연 시 | 그 사실을 안 날(주소를 안 날)로부터 30일 | 전상법 제17조 제1항 |
| "환불 불가" 일괄 고지 | 무효 — 법정 철회권은 약관·고지로 배제 불가 | 전상법 제17조·제35조(소비자에게 불리한 약정 무효) |

**청약철회 제한 사유** (전상법 제17조 제2항 — 사전 고지 + 시험사용 상품 제공 등 조치가 있어야 유효):
- 소비자 책임으로 재화가 멸실·훼손된 경우 (확인용 포장 훼손 제외)
- 사용·일부 소비로 가치가 현저히 감소한 경우
- 시간 경과로 재판매가 곤란할 정도로 가치가 감소한 경우 (신선식품 등)
- 복제 가능한 재화의 포장을 훼손한 경우 (음반·소프트웨어 등)
- 주문 제작 상품 (사전 고지 + 소비자 서면 동의 시)

## 2. 하자·표시광고 상이 시 청약철회 — 전자상거래법 제17조 제3항

- 재화의 내용이 **표시·광고 내용과 다르거나 계약 내용과 다르게 이행된 경우**: 공급받은 날로부터 **3개월 이내**, 그 사실을 안 날(알 수 있었던 날)로부터 **30일 이내** 청약철회 가능.
- 이 경우 위 1의 제한 사유가 적용되지 않습니다 (하자 상품은 포장 개봉·사용 후에도 철회 가능).

## 3. 반품 배송비 부담 — 전자상거래법 제18조

| 사유 | 반품비 부담 | 비고 |
|------|------------|------|
| 단순 변심 | **소비자 부담** | 단, 금액·부담 주체를 사전 고지한 범위 내 (전상법 제18조 제9항) |
| 하자·오배송·표시광고 상이 | **판매자 부담** | 전상법 제18조 제10항 |
| 환급 기한 | 재화 반환받은 날로부터 **3영업일 이내** 대금 환급 | 전상법 제18조 제2항 (지연 시 지연배상금) |

## 4. 소비자분쟁해결기준 — 품목별 대표 조항

공정거래위원회 고시 「소비자분쟁해결기준」은 분쟁 시 **합의·권고의 기준**입니다(법적 강제는 아니나 실무 표준). 대표 조항:

| 품목 | 대표 기준 | 비고 |
|------|----------|------|
| 의류·신발 | 봉제 불량·염색 불량 등 하자 시 무상수리 → 교환 → 환급 순 | 구입가 환급 기준 |
| 가전제품 | 구입 후 **10일 이내** 중요 수리를 요하는 하자 → 교환 또는 환급 / **1개월 이내** → 교환 또는 무상수리 | 품질보증기간 통상 1년 (품목별 상이 — 확인 필요) |
| 식료품 | 부패·변질 시 교환 또는 환급 | 신선식품은 수령 즉시 촬영 증빙 안내 권장 |
| 화장품 | 이상 반응(부작용) 시 치료비·교환·환급 협의 기준 존재 | 개별 사안 판단 필요 — 확인 필요 |

> 품목별 세부 기준(보증기간·감가 기준 등)은 고시 개정으로 변동됩니다. 응답에 구체 수치를 단정하지 말고 "소비자분쟁해결기준(공정위 고시)에 따라"로 인용하되, 확정 안내가 필요하면 최신 고시 원문 확인 후 답변하십시오 (확인 필요).

## 5. CS 응답 작성 시 적용 원칙

1. **법정 기준 미달 안내 금지**: "교환·환불 불가", "수령 후 24시간 내에만 반품" 등은 법정 기준(7일)보다 불리 → 사용 금지.
2. **사유 먼저 분류**: 단순 변심 vs 하자·오배송을 먼저 확인한 뒤 반품비 부담 주체·기한을 안내.
3. **기한은 조문 기준으로**: "수령일로부터 7일 이내"(변심), "3개월/안 날부터 30일"(하자·광고 상이), "반환 후 3영업일 내 환급".
4. **근거 병기**: 고객 안내문에는 조문 번호까지 쓰지 않아도 되지만, 내부 초안 메모에는 근거 조문을 남깁니다.
