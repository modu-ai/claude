# 계정과 설정

Claude의 설정 화면에서 기능·프로필·보안을 관리합니다.

## 기능(역량) 토글

각 기능을 켜고 끕니다:

- **코드 실행** — 파이썬 스크립트·데이터 분석
- **파일 생성** — 문서·파일 만들기
- **스킬** — 확장 능력 불러오기
- **웹 검색** — 실시간 정보 검색
- **메모리** — 대화 간 기억

## 모델 선택

Pro/Max 플랜에서 모델을 고릅니다:

| 모델 | 특징 | 언제 |
|------|------|------|
| Opus | 가장 똑똑, 비용 높음 | 복잡한 추론·설계 |
| Sonnet | 균형 | 일상 작업 |
| Haiku | 빠르고 저렴 | 단순 질문 |

## 데이터 통제

- 데이터 내보내기
- 계정 삭제
- 모든 기기에서 로그아웃

## 보안

2단계 인증(2FA)·패스키로 계정을 보호하세요.
