**릴리스 날짜**: 2026-05-20
**버전**: v2.13.0 (MINOR, 최신)
**업데이트 명령**: `/plugin marketplace update cowork-plugins`

## Highlights

`moai-media` 플러그인에 **Higgsfield MCP를 직접 호출**하는 신규 2 스킬을 도입했습니다. [higgsfield.ai 공식 페이지](https://higgsfield.ai) 명시 기준 11 이미지·11 영상 모델 + 6 비디오 프리셋을 자연어 한 줄로 자동 선택·생성합니다.

- **higgsfield-image** — 11 이미지 모델: Soul · Soul 2.0 · Soul Cinema · Nano Banana · Nano Banana Pro · GPT Image · GPT Image 2 · Seedream 4.0 · Flux Kontext · Wan 2.2 Image · Wan 2.5
- **higgsfield-video** — 11 영상 모델: Sora 2 · Google Veo 3 · Kling 2.1 Master · Kling 2.5 Turbo · Kling 3.0 · Kling Avatars 2.0 · Seedance 2.0 · Seedance Pro · Cinema Studio 3.5 · MiniMax Hailuo 02 · Wan 2.5
- **6 비디오 프리셋**: UGC · Unboxing · Product review · Hyper motion · TV spot · Wild Card

## What's New

### higgsfield-image (11 공식 이미지 모델)

자연어로 이미지를 요청하면 다음을 자동 처리:

| 단계 | 처리 |
|---|---|
| 1. 의도 추출 | 주제·용도·청중·톤 |
| 2. 모델 선택 | 11 모델 자동 (**글자·카드뉴스→GPT Image 2 (1순위)**, 시네마틱→Soul Cinema·Soul 2.0, 사진→Flux Kontext 등) |
| 3. 파라미터 | width_and_height (5 비율) · quality (720p/1080p/2K/4K) · seed · style |
| 4. MCP 호출 | `mcp__higgsfield__generate_image` |
| 5. 비동기 폴링 | queued → in_progress → completed (5-15초) |
| 6. 결과·변형 | seed 변경·배치 4종·Soul Characters reference |

**Soul Characters로 캐릭터 일관성**:
1. Soul 2.0으로 인물 1장 생성
2. Soul Characters에 reference 등록 → UUID 수령
3. 이후 호출: `custom_reference_id`로 동일 인물 다양한 포즈

### higgsfield-video (11 공식 영상 모델 + 6 프리셋)

| 단계 | 처리 |
|---|---|
| 1. 의도 추출 | 길이·용도·입력 유형 |
| 2. 모델·프리셋 선택 | 11 모델 + 6 프리셋 자동 (시네마틱→Cinema Studio 3.5+TV spot, 빠른→Kling 2.5 Turbo+UGC, 액션→Seedance Pro+Hyper motion 등) |
| 3. 파라미터 | aspect_ratio (16:9·9:16·1:1·4:5) · quality · duration |
| 4. MCP 호출 | `mcp__higgsfield__generate_video` |
| 5. 비동기 폴링 | 10-90초 (모델별) |
| 6. 결과 검수 | 모션·길이·자연스러움 |

**캐릭터 일관성**: Kling Avatars 2.0으로 동일 캐릭터 시리즈 영상.

**6 비디오 프리셋**:

| 프리셋 | 용도 |
|---|---|
| UGC | 자연스러운 일반 사용자 톤 |
| Unboxing | 언박싱·제품 개봉 |
| Product review | 제품 리뷰·특징 강조 |
| Hyper motion | 빠른 모션·역동성 |
| TV spot | 광고 시네마틱 톤 |
| Wild Card | 실험적·창의적 자유 |

## 워크플로우 예시

### 광고 8초 영상 (시네마틱)

```
1. higgsfield-image (Soul Cinema) → 시작 이미지 1장
2. higgsfield-video → image_url + Cinema Studio 3.5 + TV spot
3. duration: 8s, aspect_ratio: 16:9, quality: high
4. 60-90초 대기 → 8초 영상
```

### 카드뉴스 이미지 4종 (글자 정확)

```
higgsfield-image:
- 모델: GPT Image 2 (글자 렌더링 SOTA) — 1순위, Nano Banana Pro fallback
- 비율: 1024x1024 (인스타 피드)
- batch_size: 4
- 결과: 4종 후보
```

### 캐릭터 시리즈 영상

```
higgsfield-video:
- 모델: Kling Avatars 2.0
- 입력: 마스코트 이미지
- 결과: 동일 캐릭터 다른 동작 영상
```

### 360도 제품 회전 영상

```
higgsfield-video:
- 모델: MiniMax Hailuo 02 (카메라 무브먼트 정교)
- 프리셋: Product review
- 입력: 제품 이미지
- 결과: 회전 노출 영상
```
