# mo.ai.kr 18개 서비스 법적 노출 매트릭스

> mo.ai.kr AI 직원 플러그인 18종별로 출시 체크리스트 14종 중 어느 규제에 노출되는지 매핑.
> 위험도는 해당 플러그인 산출물을 사용자가 외부 서비스로 제공할 때의 법적 노출 기준.

## 위험도 범례

- 🔴 높음: AI 생성물을 외부에 서비스 형태로 제공할 가능성이 높아 복합 규제 적용
- 🟡 중간: 도메인 특화 규제 + 개인정보 노출
- 🟢 낮음: 도구적 사용 중심, 기본 의무만 해당

## 핵심 인사이트

1. **AI 생성물을 외부에 제공하는 4개 플러그인(moai-media·story·seller·marketer)이 최고위험군** — 인공지능기본법 1번 의무가 거의 항상 발생.
2. moai-seller는 전자상거래법 전 영역(5·6·8·9·10·13)에 노출 — 가장 광범위한 컴플라이언스 부담.
3. moai (code)·moai-officer는 도구적 사용이면 1번 면제 대상이나, 산출물을 외부 서비스로 제공하면 의무 발생 (판별은 `ai-act-user-vs-provider.md`).
4. 개인정보(민감)를 다루는 moai-accountant·recruiter·career는 4·7·14번 + 자격법 규제 추가.

## 사용 방법

체크리스트 점검 시 사용자가 주로 사용하는 플러그인을 식별하고, 이 매트릭스에서 해당 행의 "노출 규제 #"를 우선 점검한다. 전체 14종은 `launch-checklist-2026.md` SSOT 참조.
