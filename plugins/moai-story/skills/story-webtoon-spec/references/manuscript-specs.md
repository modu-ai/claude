# 원고 규격

> **원칙:** 세로 스크롤이라는 **형식**은 사실. 픽셀·길이·용량 **수치는 전부 `[미확인 — 공식 안내 확인 필요]`.** 확인 방법 열의 공식 안내에서 최신값을 확인한다.

## 1. 형식 (사실)

- 세로 스크롤 웹툰은 **가로폭 고정 + 세로로 긴 하나의 흐름**이다. 페이지형 컷 그리드가 아니라 **위→아래 연속 스크롤**로 읽는다.
- 긴 회차는 하나의 초장문 이미지 또는 **여러 장으로 분할 업로드**한다.
- 컷과 컷 사이에 **세로 여백 띠**를 둬 호흡·리듬·장면 전환을 만든다. (식자 배치는 `story-webtoon-lettering` 참조.)
- 색공간은 화면 표시 기준 **sRGB**를 쓴다.

## 2. 수치 표 ([미확인] 표기)

| 항목 | 값 | 확인 방법 |
|------|-----|-----------|
| 캔버스 가로폭 | `[미확인]` | 각 플랫폼 공식 원고 제작 가이드 |
| 회차 세로 총길이 상한 | `[미확인]` | 공식 원고 제작 가이드 |
| 분할 업로드 1장 세로 길이 | `[미확인]` | 공식 원고 제작 가이드 |
| 컷 간 여백 띠 권장 높이 | `[미확인]` | 공식 가이드/실무 관행 |
| 파일 포맷 | 화면용 래스터(JPG/PNG 계열)로 통용 — 세부 허용 포맷 `[미확인]` | 공식 업로드 안내 |
| 파일 용량 상한 | `[미확인]` | 공식 업로드 안내 |
| 해상도(DPI/PPI) | 화면 기준 — 구체값 `[미확인]` | 공식 원고 제작 가이드 |

## 3. 주의

- 인쇄 만화 기준(CMYK·300DPI·판형 mm)을 웹툰에 그대로 적용하지 않는다. 웹툰은 화면 매체다.
- 위 수치를 **지어내지 말 것.** 사용자에게 `[미확인]` 항목은 그대로 전하고 공식 가이드 확인을 안내한다.
