# IDE에서 쓰기

## VS Code 확장

1. VS Code 확장 마켓플레이스에서 "Claude Code" 검색
2. 설치
3. 에디터 안 패널에서 Claude Code 사용

그래픽 UI라 CLI보다 시각적이어서 비개발자에게 친숙합니다.

## JetBrains 플러그인

- **지원 IDE**: IntelliJ / PyCharm / WebStorm / GoLand
- **상태**: 마켓플레이스 베타
- 마켓플레이스에서 "Claude Code" 검색 후 설치

## CLI vs IDE

| 구분 | CLI | IDE |
|------|-----|-----|
| 인터페이스 | 터미널 텍스트 | 그래픽 UI |
| 추천 사용자 | 개발자 | 에디터 사용자·비개발자 |
| 기능 범위 | 전체 | 핵심 (일부 고급 제한) |
