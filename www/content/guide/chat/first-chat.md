## 단계 1: Claude Desktop 다운로드 및 설치

1. [claude.ai](https://claude.ai) 방문
2. 우측 상단 **Download Desktop** 클릭
3. 운영 체제(macOS 또는 Windows) 선택
4. 설치 파일 실행 후 로그인

## 단계 2: 대화 시작

1. Claude Desktop 열기
2. 입력창에 당신이 하고 싶은 일을 자연스럽게 입력
3. **엔터** 키 또는 **전송** 버튼 클릭
4. Claude의 답변 대기

## 단계 3: 대화 이어가기

같은 채팅 안에서 계속 질문하거나 피드백할 수 있습니다. Claude는 대화 문맥을 기억합니다.

```
당신: "이메일 초안을 써줄 수 있나요?"
Claude: [이메일 초안 제공]
당신: "좀 더 친근하게 수정해 줄 수 있나요?"
Claude: [수정된 버전 제공]
```

## 5가지 실제 사용 사례

### 1. 이메일 초안 작성

```
당신의 입력:
"고객에게 프로젝트 연기를 알리는 이메일을 써 줄 수 있나요? 
톤은 정중하지만 친근하게."

결과:
Claude가 즉시 당신 상황에 맞는 이메일을 제공
```

### 2. 이 계약서를 쉽게 설명해 줄 수 있나요?

```
당신의 입력:
"이 계약서(파일 첨부)의 핵심 조항을 설명해 줄 수 있나요?
특히 내가 주의해야 할 부분은?"

결과:
Claude가 복잡한 법률 용어를 쉬운 말로 정리
```

### 3. 브레인스토밍

```
당신의 입력:
"새로운 제품을 출시하려고 해. 
마케팅 캠프레인 아이디어 10개를 제안해 줄 수 있나요?"

결과:
Claude가 다양한 관점의 캠페인 아이디어 제공
```

### 4. 문제 해결

```
당신의 입력:
"Python에서 CSV 파일을 읽어서 특정 열만 추출하는 코드를 보여 줄 수 있나요?"

결과:
Claude가 실행 가능한 코드 제공 + 설명
```

### 5. 회의 준비

```
당신의 입력:
"내일 회의에서 분기별 매출 현황을 설명해야 해.
이 엑셀 파일(첨부)을 바탕으로 발표 자료 개요를 만들어 줄 수 있나요?"

결과:
Claude가 발표 순서와 핵심 포인트 제시
```
