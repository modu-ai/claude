# 한국 편성 관행 · 회차 구성

> 한국 드라마 제출 문서의 실체는 **회차별 사건 표**다. 서구식 3막 구조는 참고 뼈대이고, 제작사가 보는 것은 "몇 부작으로, 각 회에 무엇이 일어나는가"다. 편성 회수·러닝타임 관행은 방송사·플랫폼·시기별로 다르므로 구체 수치는 `[미확인 — 편성 관행 확인]`로 둔다.

## 1. 편성 회수 관행

| 형태 | 통상 회수 | 특징 |
|------|-----------|------|
| 미니시리즈 | 16부작 | 지상파·주요 드라마 관행적 기준 |
| 단막·숏폼 | 12부작 이하 / 8부작 | OTT·케이블·짧은 편성 증가 |
| 웹드라마 | 회당 짧은 러닝타임 다회차 | 즉각 후크·모바일 소비 |

> 회수·러닝타임은 편성 계약 사안이다 — 확정 전엔 목표 회수만 잡고 `[미확인]` 유지.

## 2. 회차별 사건 표 (드라마 제출 핵심)

각 회에 무엇이 일어나고 감정이 어디로 가는지 표로 배치한다.

```
| 회차 | 핵심 사건 | 관계 변화 | 회말 훅 |
|------|-----------|-----------|---------|
| 1 | 세계·인물 소개 + 사건 발단 | | 강한 궁금증 |
| 2 | 갈등 진입 | | |
| ... | | | |
| 16 | 절정 해소 + 결말 | | |
```

- **1회**는 세계관·주인공·중심 갈등을 세우고 강한 회말 훅으로 다음 회를 부른다.
- **중반부(전개)**는 관계 심화·서브플롯·반전 씨앗.
- **후반부(절정~결말)**는 중심 갈등 폭발과 해소.
- 각 회말에 훅(다음 회 유인)을 둔다 — 편성 드라마의 시청 지속 장치.

## 3. 막 구조 (참고 뼈대로 강등)

3막(설정–대립–해결)은 회차 사건 표를 짤 때의 참고 뼈대다.

- 1막(설정): 도입부 회차 — 세계·인물·발단.
- 2막(대립): 중반 회차 — 갈등 상승·중간 반전.
- 3막(해결): 종반 회차 — 절정·해소.

막 구조는 감정 곡선을 점검하는 렌즈로만 쓰고, **제출 문서 본체는 회차별 사건 표**로 채운다.

## 4. 영화

- 러닝타임 안에서 시퀀스 단위로 배치(발단–전개–절정–결말).
- 한국 상업영화 관행 러닝타임·시퀀스 수는 작품·장르별 상이 → `[미확인]`, 레퍼런스작으로 감을 잡는다.
