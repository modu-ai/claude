# 대사 메모 — 클린 원고 편집 인계 스크립트

> `story-webtoon-lettering` 글자 3분기 중 **클린 원고**(그림에 글자를 넣지 않는 경로)에서 로드한다. 그림엔 말풍선이 없지만, 편집 도구(포토샵·클립스튜디오·미리캔버스 등)에서 식자를 얹을 때 쓸 **대사 스크립트**를 컷별로 함께 넘기면 편집이 깔끔하고 일관된다.

## 1. 형식 — 컷별 [화자 / 타입 / 대사]

```
컷 3
- [주인공 / 일반 대사] "여기서 멈출 순 없어."
- [내레이션 / 독백·내레이션] 그날의 선택이 모든 걸 바꿨다.
- [SFX / 효과음] 쿠구궁
```

## 2. 타입 4종 — 편집 시 형태 매핑

| 타입 | 형태(편집 도구에서) | 용도 |
|------|--------------------|------|
| 일반 대사 | 둥근 말풍선(타원) + 꼬리 | 입 밖으로 하는 말 |
| 독백·내레이션 | 사각 박스(회색조) | 마음속 말·해설 |
| 전화 너머·기계음 | 구름형 또는 점선 테두리 | 통화·스피커·방송 목소리 |
| 효과음(SFX) | 그래픽 텍스트 | 동작·충격음 — 클라이맥스는 강조색. 그림에 직접 넣으려면 `sfx-dictionary.md` 2패스 경로 |

## 3. 배치 규칙

- 한 컷당 말풍선 **3개 이하** — 핵심 표정·동작을 가리지 않게.
- 읽기 순서를 따른다 — 세로 스크롤은 **위→아래**가 기본, 같은 높이에서는 왼→오.
- 뷰포트 1화면에 동시에 보이는 말풍선 수는 `balloon-dictionary.md` §위치 거동의 상한을 따른다.

## 4. 경계

- 이 4타입 분류는 **클린 원고 전용**이다. 그림에 글자를 넣는 분기(대사 포함·큰 글씨)에서는 이 표 대신 `balloon-dictionary.md`의 검증된 말풍선 종류(대사/생각/외침/특수)를 영어 스니펫으로 프롬프트에 박는다.
