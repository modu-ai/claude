# 문제 해결

Claude 사용 중 자주 만나는 문제와 해결입니다.

## 자주 묻는 문제

- **로그인 문제**: 비밀번호 재설정, 브라우저 캐시 확인
- **결제·환불**: 결제 실패 시 카드 정보 확인, 환불은 지원 센터 요청
- **데스크톱 앱 오류**: 재설치, OS 버전 확인 (macOS 13.0+ / Windows 10 1809+)
- **사용 한도 도달**: 5시간 창 리셋 대기, 또는 상위 플랜

## 느린 응답·스트리밍 오류

- 브라우저 새로고침(강력 새로고침)
- 네트워크 연결 확인
- 브라우저 확장 비활성화 후 재시도

## 응답 없음·제한

- 5시간 창 리셋 대기
- Anthropic 상태 페이지(status page) 확인

## 모바일

- 알림 설정 확인
- 기기 간 동기화 문제 → 앱 업데이트·재로그인

## 지원 티켓

인앱 Help 메뉴로 문의. 스크린샷·재현 단계를 포함하면 빠릅니다.
