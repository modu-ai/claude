**릴리스 날짜**: 2026-06-16
**버전**: v2.23.0 (MINOR)
**업데이트 명령**: `/plugin marketplace update cowork-plugins`

## 하이라이트

**`drawio-diagram` 스킬을 제거하고 모든 다이어그램을 mermaid 인라인으로 단일화했습니다.**

- **렌더링 안정성 확보** — `viewer-static.min.js` CDN 기반 draw.io 렌더링이 테스트 21건 중 2건만 성공해 배포 품질 기준에 미달. CDN 의존 없이 Hugo·GitHub·Cowork 모든 환경에서 안정적으로 렌더링되는 Mermaid 인라인으로 단일화.
- **스킬 제거** — `moai-content:drawio-diagram` 스킬 제거. 178 → 177 스킬.
- **docs-site 산문·SVG 보강** — 관련 문서 페이지의 다이어그램을 Mermaid로 전환하고 설명 보강.

## 업그레이드 방법

```
/plugin marketplace update cowork-plugins
```

플러그인 상세 재진입 시 반영됩니다.

## 호환성

**기능적 비파괴** — `drawio-diagram` 스킬을 직접 호출하는 워크플로우에서는 해당 스킬이 더 이상 동작하지 않습니다.

**대체 방법**: `moai-content` 내의 다이어그램이 필요한 스킬에서 Mermaid 구문을 직접 사용하거나, Claude에게 "Mermaid 다이어그램으로 그려줘"라고 요청하세요.
