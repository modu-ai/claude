# 경제지표 읽기 (Economic Literacy)

## 개요

경제 뉴스의 금리·환율·물가·GDP를 남의 이야기가 아니라 내 대출·예금·투자·소비에 어떻게 영향을 주는지로 번역한다. 지표 자체의 정의보다 "그래서 내 돈은 어떻게 되나"를 우선한다.

**면책 고지: 일반적인 경제 교육 정보이며 투자 자문이나 시장 예측이 아닙니다. 지표는 참고일 뿐 투자 결정과 결과는 본인 책임입니다.**

## 트리거 키워드

경제지표, 금리, 기준금리, 환율, 물가, 인플레이션, CPI, GDP, 경기, 경기 사이클, 고용지표, 수출입, 경제 읽는 법

## 워크플로우

### 1단계: 핵심 지표를 '내 돈'으로 번역

| 지표 | 오르면 내 돈에 미치는 영향 |
|------|---------------------------|
| **기준금리** | 대출 이자 부담↑, 예적금 이자↑, 일반적으로 주식·부동산엔 부담 |
| **환율(원/달러)** | 수입물가·해외여행·직구 비용↑, 해외주식 원화 환산액 변동 |
| **물가(CPI)** | 생활비↑, 현금의 실질가치↓ → 인플레 헤지 자산에 관심 |
| **GDP/성장률** | 경기·기업 실적·고용 전반의 체온계 |
| **고용지표** | 소득·소비 여력, 경기 방향성 신호 |
| **수출입** | 한국 경제·환율·수출기업 실적과 직결 |

### 2단계: 금리를 중심에 두기

금리는 거의 모든 자산의 기준점이다.

- **금리↑**: 예금 매력↑, 대출·변동금리 부담↑, 위험자산엔 역풍, 채권 가격↓
- **금리↓**: 대출 부담↓, 위험자산·부동산엔 순풍, 예금 매력↓
- → 내 상황 점검: 변동금리 대출이 있나? 예적금 갈아탈 때인가?

### 3단계: 경기 사이클 이해

경기는 **회복 → 활황 → 후퇴 → 침체**를 순환한다.

- 지표(성장·물가·고용·금리)의 조합으로 대략의 국면을 가늠한다.
- 국면마다 유리한 자산이 다르다(절대 공식 아님, 분산·장기 원칙 우선).
- **타이밍 맞히기는 어렵다** → 사이클은 "이해"하되 "예측 베팅"하지 않는다.

### 4단계: 지표를 어디서 보나

- 한국은행(기준금리·경제통계), 통계청(물가·고용), 기획재정부, 한국은행 경제통계시스템(ECOS)
- → 1차 공식 통계를 보고, 해석은 본인의 자산 상황과 연결한다.

### 5단계: 행동으로 연결

- **대출**: 금리 방향에 따라 고정/변동 전략, 상환 우선순위 점검 → `finance-wealth-roadmap`
- **예적금·투자**: 금리·물가 환경에 맞춰 현금 비중·자산 배분 점검 → `finance-invest-primer`
- **소비**: 물가·환율 흐름을 큰 지출(여행·내구재) 타이밍에 참고

## 사용 예시

**예시 1**: "기준금리 오르면 내 변동금리 대출은 어떻게 돼?"
→ 이자 부담 증가 메커니즘 + 고정 전환·상환 우선순위 점검

**예시 2**: "환율이 오르는데 해외 주식·직구에 어떤 영향이야?"
→ 원화 환산·수입물가 영향 설명 + 환율 변동을 자산에 반영하는 관점

**예시 3**: "경제 뉴스가 너무 어려운데 뭐부터 봐야 해?"
→ 금리·물가·환율 3종부터 + 각 지표를 내 돈으로 번역하는 법

## 산출물

- 핵심 지표 → 내 돈 영향 번역표
- 금리 중심 자산 영향 정리
- 경기 사이클 개념
- 공식 통계 출처
- 지표→행동 연결 가이드

## 주의사항

- **면책**: 시장 예측·투자 권유가 아니다. 지표는 참고 정보.
- 지표 하나로 단정하지 않는다 — 조합과 추세로 본다.
- "타이밍 베팅"보다 분산·장기 원칙이 우선한다.
