# 2차 저작 확장 경로

> 원작을 다른 매체로 확장하는 경로별 적합도·난이도·선행 산출물을 정리한다. 적합도는 원작 특성에 따라 달라지는 상대 판단이지 고정값이 아니다.

## 1. 경로 표

| 원작 → 확장 | 적합 조건 | 난이도 | 선행 산출물 |
|-------------|-----------|--------|-------------|
| 웹소설 → 웹툰 | 시각적 매력·명확한 인물 | 중 | 인물 시트·콘티 (`story-character-sheet`, `story-webtoon-episode`) |
| 웹툰 → 드라마/영화 | 서사 응집·캐릭터 매력 | 상 | 시놉시스·대본 (`story-synopsis`, `story-screenplay`) |
| 웹소설 → 드라마/영화 | 강한 사건축·화제성 | 상 | 시놉시스·대본 |
| 웹툰/웹소설 → 게임 | 세계관·캐릭터 확장성 | 상 | 세계관·캐릭터 라이선스 자료 |
| 출판 단행본 → 영상 | 실적·인지도 | 상 | 단행본 실적·시놉시스 |
| IP → 굿즈/MD | 캐릭터 아이덴티티 | 하~중 | 캐릭터 라이선스·디자인 자료 |

## 2. 경로 선택 판단

- **적합도**: 원작의 핵심 매력이 목표 매체에서 살아나는가(예: 내면 독백 중심 웹소설은 영상화 난도가 높다).
- **난이도**: 필요 자본·제작 리스크·제작 주기.
- **선행 산출물**: 피칭 설득력을 높이는 자료(콘티·시놉·시각화 샘플)를 갖췄는가.

## 3. 협상력 근거 데이터

2차 저작 제안의 협상력은 원작 실적에서 나온다.

- 조회수·완독률·유료 전환·랭킹·팬덤 규모(커뮤니티·2차 창작).
- **실데이터만 사용** — 부풀리거나 추정치를 확정처럼 쓰지 않는다.
