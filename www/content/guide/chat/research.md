# 리서치 (Research)

**리서치**는 Claude가 여러 단계로 웹을 탐색하며 심층 조사를 수행하는 기능입니다.

## 시작하기

입력창의 "+" 버튼 → "Research"를 선택해 시작합니다. 진행 중에는 탐색 단계가 표시됩니다.

## 웹 검색·확장 사고와의 차이

| 기능 | 언제 |
|------|------|
| 웹 검색 | 빠르게 최신 정보/사실 확인 |
| 확장 사고 | 복잡한 추론·계산 (검색 불필요) |
| 리서치 | 다단계 심층 조사, 여러 출처 교차 |

## 인용과 출처

리서치 결과에는 번호가 매겨진 각주와 탭할 수 있는 출처 목록이 달립니다. 확인하며 읽을 수 있습니다.

## 가용성

지원 플랜과 지역에 따라 제공됩니다. Pro 이상 플랜에서 주로 사용 가능합니다.

## 활용 예시

- 제품 비교 조사 (여러 옵션 교차 분석)
- 배경 브리핑 (주제 심층 파악)
- 학술·사실 확인 다이빙
