# 자기돌봄 (Self Care)

## 개요

쉬어도 쉰 것 같지 않고, 가만히 있으면 불안하고, 의욕은 바닥난 상태 — 마음의 기초체력이 떨어진 상태다. 이 스킬은 지금 상태를 가볍게 점검하고, 생물학적·정서적 회복 루틴을 짜고, 일상의 부담을 조절해 다시 굴러가게 한다. 의료적 처치가 아니라 자기돌봄의 실천 가이드다.

## 트리거 키워드

번아웃, 자기돌봄, 회복, 멘탈 관리, 지친 마음, 쉼, 마음 기초체력, 소진, 무기력, 휴식 설계

## 워크플로우

### 1단계: 지금 상태 점검

자책하지 말고 현재 상태를 가볍게 신호로 확인한다.

- **신체 신호**: 수면 질, 피로, 식욕·소화, 잦은 잔병치레
- **정서 신호**: 의욕 저하, 짜증·예민함, 공허감, 사소한 일에 무너짐
- **행동 신호**: 미루기, 회피, 사람·연락 피하기, 일에 흥미 상실
- → 신호가 여러 개 겹치면 "게을러서"가 아니라 "충전이 필요한" 상태로 본다

### 2단계: 쉼의 종류 구분

"쉬어도 안 쉰 것 같은" 이유는 진짜 필요한 쉼이 아니어서다. 쉼을 종류로 나눠 맞춘다.

| 필요한 쉼 | 예시 |
|----------|------|
| 신체적 쉼 | 잠·낮잠·눕기·스트레칭 |
| 정신적 쉼 | 멍 때리기, 알림·정보 끊기 |
| 감각적 쉼 | 화면·소음에서 벗어나기, 자연 |
| 정서적 쉼 | 솔직한 감정 표현, 안전한 사람과 대화 |

→ 유튜브·SNS는 자극이지 쉼이 아닐 때가 많다. 부족한 종류의 쉼을 고른다.

### 3단계: 생물학적 회복 루틴

가장 먼저 몸의 기초를 복구한다. 마음은 몸 위에서 회복된다.

- **수면**: 취침·기상 시간 고정, 자기 전 화면 줄이기
- **햇빛·움직임**: 짧은 산책, 가벼운 운동(완벽 말고 자주)
- **식사**: 거르지 않기, 카페인·자극 조절
- → 거창한 결심이 아니라 작은 1개부터 → `office-habit-routine` 연계

### 4단계: 정서적 자기돌봄

자신을 다그치는 말투를 돌보는 말투로 바꾼다.

- **자기 대화**: "왜 이것밖에 못해" → "지금은 충전이 필요한 시기"
- **감정 적기**: 막연한 불안을 한 줄로 적어 밖으로 꺼내기
- **경계 세우기**: 거절·미루기를 죄책감 없이 — 할 일을 줄이는 것도 돌봄
- → 무리한 자기계발 루틴은 잠시 멈춘다. 회복이 우선이다

### 5단계: 부담 조절·재진입

- **업무량 재설계**: 당장 줄일 수 있는 일·회의·약속 1~2개 비우기 → `office-time-system` 연계
- **회복 신호 확인**: 의욕·수면이 돌아오는지 1~2주 단위로 점검
- **천천히 복귀**: 100%가 아니라 60~70%부터 재진입
- → 반복되면 패턴을 `office-retro-builder`로 점검하고 구조적 원인을 찾는다

## 사용 예시

**예시 1**: "요즘 번아웃인 것 같은데 어떻게 회복해야 할지 모르겠어"
→ 상태 신호 점검 → 부족한 쉼 종류 진단 → 수면·움직임 회복 1개부터 + 업무 줄이기

**예시 2**: "쉬는 날인데도 불안하고 쉰 것 같지가 않아"
→ 자극(SNS)과 쉼 구분 → 감각적·정신적 쉼 처방 → 죄책감 없는 쉬기 가이드

**예시 3**: "일은 많은데 의욕이 하나도 없어"
→ 무기력 신호 확인 → 당장 비울 일 1~2개 → 60~70% 재진입 플랜

## 산출물

- 상태 점검 신호 체크리스트
- 부족한 쉼 종류 진단
- 생물학적 회복 루틴(수면·움직임·식사)
- 정서적 자기돌봄 실천(자기 대화·경계)
- 부담 조절·재진입 플랜

## 주의사항

- **이 스킬은 의료·심리 치료가 아니다.** 일상 자기돌봄 가이드일 뿐이다.
- 우울·불안이 2주 이상 지속되거나 일상이 무너지면 정신건강의학과·상담을 권한다.
- 회복기에 새 목표·습관을 무리하게 더하지 않는다. 덜어내는 것이 먼저다.

## 관련 스킬

- **moai-cowork:office-time-system**: 과도한 업무량을 줄여 회복 여지 확보
- **moai-cowork:office-habit-routine**: 작은 회복 루틴을 부담 없이 정착
- **moai-cowork:office-retro-builder**: 번아웃 패턴을 가볍게 점검
- **moai-comms**: 직장 내 거절·요청 등 대인 커뮤니케이션이 필요할 때

## 이 스킬을 사용하지 말아야 할 때

- **임상 진단·치료가 필요한 상태**: 자해 충동·심각한 우울은 즉시 전문기관(자살예방·정신건강 위기상담 109 — 2024년 통합번호, 구 1577-0199)과 연결한다.
- **단순 일정·업무 효율 문제**: 마음이 아니라 시간 배분 문제면 `office-time-system`이 맞다.
- **타인의 정신건강 진단**: 본 스킬은 본인 자기돌봄용이며 타인을 진단하지 않는다.
