# 어디서 쓰나요 — 6가지 진입면

Claude Code는 한 엔진에 6가지 진입면(surface)을 제공합니다. **비개발자는 터미널 없이 웹·데스크톱·IDE로 시작**할 수 있습니다.

## 6가지 진입면

| 면 | 누구에게 | 필요 것 |
|----|----------|---------|
| CLI / 터미널 | 개발자 | Node.js 설치 |
| IDE 확장 | 에디터 사용자 | VS Code / JetBrains |
| 데스크톱 앱 | 일반 사용자 | 앱 설치 |
| 웹 (claude.ai/code) | 비개발자 | 브라우저 + GitHub |
| Slack | 팀 협업 | Slack 워크스페이스 |
| GitHub Actions | CI 자동화 | 저장소 권한 |

## Slack에서의 Claude Code

Slack 채널에서 `@claude` 멘션으로 작업을 요청하면, 채널 맥락을 읽고 코드 작업을 수행합니다.

## 처음 시작 추천 경로

비개발자 추천 순서: **웹 → 데스크톱 → (필요 시) CLI**

- 웹: 설치 없이 가장 만만
- 데스크톱: 자주 쓰면 앱이 편리
- CLI: 자유도가 가장 높음
