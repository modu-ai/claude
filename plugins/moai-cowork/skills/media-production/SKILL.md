# 미디어 프로덕션 (별칭 → 분리됨)

`media-production` 스킬은 산출물 단위로 두 개의 스킬로 분리되었습니다.
요청 성격에 따라 아래 스킬을 사용하세요.

- **콘텐츠 발행 캘린더·일정 기획** → `moai-cowork:content-calendar`
  - "콘텐츠 캘린더 짜줘", "이번 달 발행 일정 만들어줘", "채널별 게시 주기 정해줘"
- **유튜브·팟캐스트 에피소드 기획** → `moai-cowork:youtube-podcast-planner`
  - "유튜브 영상 기획해줘", "팟캐스트 에피소드 구성해줘", "쇼노트 만들어줘", "유튜브 대본 아웃라인"
