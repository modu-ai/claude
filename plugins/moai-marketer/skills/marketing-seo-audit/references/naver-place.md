# 네이버 플레이스 감사 체크리스트 (naver-place)

`marketing-seo-audit` 5단계(네이버 플레이스 감사)에서 로드하는 세부 점검표입니다. 대상: 네이버 스마트플레이스에 등록된(또는 등록해야 하는) 로컬 비즈니스.

> 플레이스 노출 로직·광고 단가 등 네이버 내부 기준은 공개 알고리즘이 아니며 수시로 변동됩니다. 본 문서는 스마트플레이스 공식 기능과 공개된 운영 관행 기준의 점검표이며, 순위 보장을 주장하지 않습니다. 구체 단가·가중치 수치는 '확인 필요'로 처리합니다.

## 1. 업체정보 완성도

| 점검 항목 | 기준 |
|----------|------|
| 기본 정보 | 업체명(실제 간판과 일치)·주소·전화번호·영업시간(브레이크타임·휴무일 포함) 전부 입력 |
| 가격/메뉴 | 대표 메뉴 + 가격 등록, 품절·시즌 메뉴 반영 |
| 편의시설·부가정보 | 주차·예약·포장·배달·반려동물 등 해당 항목 전부 체크 |
| 업체 소개 | 핵심 키워드가 자연스럽게 포함된 소개문 (키워드 나열식 금지) |
| 인증 상태 | 스마트플레이스 사업자 인증 완료 여부 |

**원칙**: 입력 가능한 항목은 전부 채웁니다. 미입력 항목은 검색 필터(예: "주차 가능") 매칭에서 제외되는 기회손실입니다.

## 2. 대표키워드·업종

- **업종 분류**: 실제 업태와 가장 정확한 세부 업종 선택 (넓은 업종보다 정확한 업종이 유리 — 검색 의도 매칭 기준).
- **대표키워드**: 스마트플레이스에서 설정 가능한 대표키워드를 "지역+업종/메뉴" 조합의 실검색어로 설정 (예: "성수동 브런치").
- **금지**: 업체명에 키워드 끼워넣기(예: "OO카페 성수맛집") — 업체명 정책 위반으로 수정·제재 대상이 될 수 있습니다.

## 3. 리뷰 전략 (영수증·방문자 리뷰)

| 항목 | 점검·전략 |
|------|----------|
| 리뷰 양·최신성 | 최근 리뷰가 꾸준히 쌓이는지 (오래된 리뷰만 있으면 활성도 신호 약화) |
| 영수증 리뷰 유도 | 매장 내 안내(POP·영수증 문구)로 방문 인증 리뷰 요청 — 대가 제공 시 표시광고법상 경제적 이해관계 표기 의무 |
| 사장님 답글 | 신규 리뷰에 답글 운영 (부정 리뷰일수록 신속·정중하게 — CS 응대는 moai-cs 스킬 연계) |
| 금지 | 리뷰 대행·조작 구매 — 네이버 어뷰징 제재 + 표시광고법 기만적 표시광고 리스크 |

## 4. 콘텐츠 갱신 주기 (사진·메뉴·소식)

- **사진**: 대표사진·메뉴사진·내부사진을 계절/메뉴 개편 시마다 갱신. 저화질·타 업체 이미지 사용 금지.
- **메뉴**: 가격 변경 즉시 반영 (플레이스 가격 ≠ 실제 가격은 리뷰 불만의 단골 원인).
- **소식(뉴스)**: 쿠폰·이벤트·휴무 공지를 주기적으로 발행 — 방문 전 확인 수요가 많고 업체 활성도 신호가 됩니다. 권장 주기: 최소 월 1회 이상 (내부 가중치는 비공개 — 확인 필요).

## 5. 예약·주문·톡톡 연동

| 기능 | 점검 |
|------|------|
| 네이버 예약 | 예약 기반 업종(미용·식당·클래스)은 활성화 + 슬롯 최신화 |
| 네이버 주문/스마트주문 | 해당 업종(카페·식당)의 포장·매장 주문 연동 여부 |
| 톡톡 | 문의 채널 활성화 + 응답률 관리 (미응답 방치 시 고객 이탈) |

연동 기능 활성화는 플레이스 상세 화면에 버튼으로 노출되어 전환 경로를 단축합니다.

## 6. 플레이스 광고

- **상품**: 네이버 검색광고의 "플레이스광고" — 지역+업종 검색 시 플레이스 영역 상단 노출, CPC 과금.
- **진입 판단**: (a) 오가닉 플레이스 순위가 첫 화면 밖이고 (b) 위 1-5 기본기가 정비된 후에 집행. 정보가 부실한 상태의 광고는 클릭만 소모합니다.
- **운영**: 지역·시간대 타겟과 일예산 하한 등 세부 수치는 광고시스템 화면 기준 확인 (확인 필요). 광고 문안은 `moai-seller:commerce-ad-claim-compliance-kr` 게이트 통과 권장.

## 감사 출력 형식

```markdown
### 네이버 플레이스 감사 결과

| 영역 | 점수 | 주요 결함 | 개선 액션 |
|------|------|----------|----------|
| 업체정보 완성도 | 8/10 | 브레이크타임 미입력 | 영업시간 상세 입력 |
| 대표키워드·업종 | 6/10 | 대표키워드 미설정 | "지역+메뉴" 실검색어 3개 설정 |
| 리뷰 | 5/10 | 최근 3개월 리뷰 2건 | 영수증 리뷰 안내 POP + 답글 운영 |
| 사진·메뉴·소식 | 4/10 | 소식 6개월 미발행 | 월 1회 소식 발행 캘린더 |
| 예약·톡톡 연동 | 7/10 | 톡톡 응답률 낮음 | 자동 응답 + 응대 담당 지정 |
| 플레이스 광고 | 보류 | 기본기 미정비 | 1-4 개선 후 재검토 |
```

## 출처

- 네이버 스마트플레이스 공식: `https://smartplace.naver.com` (기능·정책 기준)
- 네이버 검색광고(플레이스광고): `https://searchad.naver.com`
- 리뷰 대가 표기 의무: 「표시·광고의 공정화에 관한 법률」 + 공정위 추천·보증 심사지침
