# 웹툰 실무 용어 사전 + 금칙어

> **원칙:** 표준 실무어를 쓰면 협업·검색·심사에서 오해가 준다. 비표준어는 금칙어 표로 교정한다.

## 1. 용어 사전

| 표준어 | 뜻 |
|--------|-----|
| 말풍선 | 대사·생각을 담는 풍선. 종류·꼬리 규칙은 `story-webtoon-lettering` 참조 |
| 식자 | 원고에 글자(대사·효과음·타이틀)를 앉히는 작업 |
| 효과음(SFX) | 소리를 디자인된 글자로 표현한 것 |
| 콘티 | 컷 분할·구도·대사 배치를 잡는 초안 (스토리보드) |
| 회차 | 한 번에 업로드되는 연재 단위 (= 화) |
| 클리프행어 | 회차 끝에서 다음 회를 궁금하게 만드는 장치 |
| 세로 스크롤 | 위→아래로 읽는 웹툰 기본 형식 |
| 여백 띠 | 컷 사이 세로 빈 공간 — 스크롤 호흡을 만든다 |
| 도전만화 | 네이버웹툰 아마추어 공개 게시판 (1단계) |
| 베스트도전 | 도전만화 상위 승격 단계 (2단계) |
| 정식연재 | 편집부 계약 연재 (3단계) |
| 기다무 | "기다리면 무료" 수익화 구조 |
| 사이다 | 답답함이 해소되는 통쾌한 전개 |
| 고구마 | 답답하게 막히는 전개 |

## 2. 금칙어 표 (비표준어 → 표준어)

| 쓰지 말 것 | 대신 | 이유 |
|-----------|------|------|
| 말칸 | 말풍선 | '말칸'은 비표준 구어. 협업·심사에서 통용 안 됨 |
| 웹썰 | 웹툰 / 웹소설 | 매체를 흐리는 축약어 |
| 컷지 | 컷 | 불명확한 축약 |
| 폰트 흔들림 (문제 서술로만 OK, 용어로는) | 서체 일관성 | 서체 규칙은 lettering 참조 |

> 표준어를 기본으로 쓰되, 사용자가 비표준어로 물으면 뜻을 알아듣고 **표준어로 교정해** 답한다.
