## 첫 번째 프로젝트 만들기

Claude Code가 설치되었으면 이제 첫 번째 프로젝트를 시작해 보겠습니다.

### 1단계: 새 폴더 만들기

작업할 폴더를 만드세요:

```bash
mkdir my-first-project
cd my-first-project
```

### 2단계: Claude Code 시작하기

다음 명령어를 입력하세요:

```bash
claude
```

Claude Code 터미널이 열립니다. 이제 자연어로 요청할 수 있습니다.

### 3단계: 첫 번째 요청하기

아래 예시 중 하나를 복사해서 입력하세요:

**예시 1 — 간단한 웹페이지**

```
간단한 HTML 웹페이지를 만들어 줄 수 있나요? 
제목은 "환영합니다", 본문에는 "안녕하세요!" 라고 표시되는 페이지요.
```

**예시 2 — Python 스크립트**

```
Python으로 사용자 이름을 입력받아서 "안녕하세요, <이름>님!"이라고 
인사하는 스크립트를 만들어 줄 수 있나요?
```

**예시 3 — React 앱**

```
React로 간단한 할일 목록(To-Do) 앱을 만들어 줄 수 있나요?
항목을 추가하고 제거할 수 있어야 합니다.
```

Claude가 요청에 따라 필요한 파일들을 만들고 수정합니다.

## 결과 확인하기

Claude가 작업을 완료하면:

1. **생성된 파일 확인** — "다음 파일을 만들었습니다:" 라는 메시지 확인
2. **로컬에서 실행** — 제시된 명령어로 결과 실행 (예: `npm start`, `python main.py`)
3. **웹브라우저 열기** — 웹 프로젝트라면 브라우저에서 `http://localhost:3000` 접속

## 자연어로 수정하기

결과가 마음에 안 들면 자연어로 요청하면 됩니다:

**요청 예시:**

```
이 버튼의 색을 파란색으로 바꿔 줄 수 있나요?
```

```
이 함수가 더 빠르게 동작하도록 최적화해 줄 수 있나요?
```

```
이 페이지에 어두운 테마 지원을 추가해 줄 수 있나요?
```

Claude가 변경사항을 적용하고 로컬에서 즉시 테스트할 수 있습니다.

## 팁

| 팁 | 설명 |
|---|---|
| **구체적으로 설명** | "예쁜 페이지" 보다 "파란색 배경에 흰 글자" 가 더 효과적 |
| **단계별 요청** | 큰 프로젝트는 여러 번에 나눠서 요청하는 것이 효과적 |
| **스크린샷 공유** | 다른 웹사이트 스크린샷을 올려서 "이 스타일처럼" 요청 가능 |
| **기존 코드 활용** | 이미 작성한 코드가 있으면 업로드해서 기반으로 사용 가능 |
