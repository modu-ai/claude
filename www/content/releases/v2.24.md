## 하이라이트

**`html-slide` 신규 스킬 (moai-content)** — 발표용 슬라이드 덱을 브라우저에서 바로 열리는 단일 파일·자체 완결형 HTML로 생성합니다.

- **단일 파일 무의존 HTML** — 16:9 슬라이드, 자체 vanilla JS 런타임(네비게이션·풀스크린·`?print-pdf` 인쇄·speaker notes), `file://` 즉시 오픈
- **인포그래픽 = 인라인 SVG** — 차트·다이어그램·KPI를 한국어 숫자·라벨이 100% 정확한 SVG로 직접 렌더링 (AI 래스터 대신 코드→벡터)
- **이미지 백엔드 정책 변경** — Higgsfield MCP(기본) + codex(gpt-image-2, 공식 추가). codex exec가 ChatGPT 구독 한도로 gpt-image-2를 호출(API 키 불필요). antigravity는 비권장, 그 외 외부 이미지 백엔드는 사용하지 않습니다
- **design-system-library 56 브랜드 토큰** + getdesign.md 상세 페이지 링크로 미리보기
- **편집 가능 PPTX** — pptx-designer(moai-office) 체이닝. deck.json 원고 SSOT → pptxgenjs OOXML 직접 생성

## 사용 예시

```
AI 슬라이드 스킬 스타트업 사업계획서 10장 슬라이드로 만들어줘. claude 테마로.
3분기 매출 분석 슬라이드 HTML로 만들고, 편집 가능한 PPTX로도 저장해줘.
```
