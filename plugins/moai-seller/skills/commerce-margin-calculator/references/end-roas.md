# 엔드 ROAS (본전 ROAS) 공식 — SSOT

엔드 ROAS(본전 ROAS)는 **적자가 나지 않는 최소 광고수익률**이다. 이 문서가 엔드 ROAS 공식의 단일 진실 소스(SSOT)이며, `moai-seller:commerce-margin-calculator`와 `moai-seller:commerce-marketplace-coupang-ads`가 동일하게 참조한다.

## 공식

```
판매가 - 부가세 = 부가세 빠진 판매가
부가세 빠진 판매가 × (1 - 수수료율) = 실제 입금 매출
실제 입금 매출 - 원가 - 배송비 - 쿠폰 = 마진
마진 = 광고비 최대 한도

엔드 ROAS = (판매가 ÷ 마진) × 100   (부가세 포함 판매가 기준)
         = 광고비를 매출의 몇 배로 만들어야 본전인가
```

권장 목표 ROAS = 엔드 ROAS × 1.3~1.5.

## 예시 1 — 쿠팡 무선이어폰 (수수료·부가세 반영 정밀)

- 판매가 19,800원 / 원가 6,000원 / 배송 3,000원 / 쿠팡 수수료 10.7%
- 부가세 빠진 가격: 19,800 ÷ 1.10 = 18,000원
- 플랫폼 수수료: 18,000 × 0.107 = 1,926원
- 실제 입금: 18,000 - 1,926 = 16,074원
- 총 원가: 6,000 + 3,000 = 9,000원
- 마진: 16,074 - 9,000 = **7,074원**
- 엔드 ROAS: 19,800 ÷ 7,074 × 100 ≈ **280%**

> 즉 이 상품은 광고비를 19,800원 매출의 1/2.8(= 7,074원) 이하로 써야 적자가 안 난다. 이게 광고수익률 280% = 본전 ROAS.

## 예시 2 — 단순 개념 확인

- 1만원 상품, 원가+배송비 등 8천원 → 마진 2천원 → 광고비 최대 2천원
- 엔드 ROAS = 10,000 ÷ 2,000 × 100 = **500%** (광고비 2천원으로 1만원 매출 = 500%)

## 쿠팡 ROAS 보정 (필수 경고)

쿠팡 화면 ROAS는 **부가세 빠진 금액** 기준 → 실제 ROAS는 **10% 더 낮다**. 화면 ROAS 200% 표기 = 실제 180%, 화면 ROAS 표기 = 실제 ROAS × 0.9. 엔드 ROAS와 비교할 때 화면 수치를 그대로 쓰면 본전 이하인데도 흑자로 착각할 수 있다.
