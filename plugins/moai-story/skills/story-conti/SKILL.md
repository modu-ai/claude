# story-conti: 영상 콘티 보드

시나리오 씬을 **정지 프레임 보드**로 옮기는 스킬이다. 감독·촬영감독·광고주가 촬영 전에 화면을 합의할 수 있게 화각·앵글·전환을 설계한다. 용도 프리셋 2종(드라마·영화 / 광고)을 한 스킬에서 다룬다. **생성 실행·크레딧은 하지 않는다** — `moai-media`에 위임한다.

## 1. 개요

콘티는 **무엇을 보여줄지(프레임 구성)**와 **어떤 화각으로 잡을지(앵글·전환)**를 정한다. 산출물은 정지 프레임 보드다(카메라가 움직이는 영상 클립은 `story-previz` 소관 — 매체가 다르다). 드라마·영화와 광고는 산출물 형태는 같고 목적만 다르므로 프리셋으로 분기한다.

## 2. 이 스킬이 하지 않는 것

- **이미지·영상 생성·크레딧 고지·모델 선택 안 함** — `moai-media:media-higgsfield-core` 계약에 위임(§4-생성 실행).
- **카메라 무빙 영상 클립 안 만듦** — 달리·팬·트래킹이 들어간 시네마틱 클립은 `story-previz`.
- **모델 프롬프트 문법 안 소유** — 벤더별 문법은 `moai-media`.
- **대본 집필 안 함** — 씬·대사는 `story-screenplay`가 확정한 것을 받는다.

## 3. 사전 확인

- `references/shot-grammar.md` — 화각·앵글·전환 어휘 + 프레임 시퀀스 설계 원칙.
- `references/ad-conti-preset.md` — 광고 콘티일 때(15초/30초 구성·TVC vs 디지털·브랜드 가이드·광고주 제안 형식).

## 4. 워크플로우

### Step 1 — 용도 프리셋 판정

요청이 **드라마·영화 콘티**인지 **광고 콘티**인지 먼저 가른다. 광고면 `ad-conti-preset.md`를 함께 로드한다(시간 단위·후크 위치·CTA가 서사 콘티와 다르다).

### Step 2 — 씬 → 프레임 분해

씬을 프레임 단위로 나눈다. **컷 수는 할당량이 아니라 감정 비트가 정한다** — "8프레임 채워야 한다"가 아니라 "이 감정을 전달하는 데 몇 컷이 필요한가". 프레임마다 화각·인물 배치·동작을 정한다.

### Step 3 — 화각·앵글 설계

각 프레임의 화각(익스트림 롱~익스트림 클로즈업)·앵글(하이/로우/아이레벨)·전환을 `shot-grammar.md` 어휘로 지정한다. 감정 정점에 클로즈업, 상황 제시에 롱숏을 배치하는 식으로 화각이 감정을 실어 나른다.

### Step 4 — 보드 조합

프레임 스케치 프롬프트 + 대사/내레이션 + 연출 메모를 한 보드로 조합한다. 광고면 `ad-conti-preset.md`의 후크→문제→해결→CTA 골격으로 배열한다.

### Step 4-생성 실행 — moai-media 위임

조립된 프레임 프롬프트의 실제 생성·비용 프리플라이트·모델 선택은 `moai-media:media-higgsfield-core` 계약(`models_explore` 라이브 조회 + `get_cost` 사전 고지)에 위임한다.

> moai-media 미설치 시: 완성 프레임 프롬프트를 텍스트로 출력하고 Higgsfield 웹(https://higgsfield.ai)에서 직접 생성하도록 안내한다.

## 5. 출력 형식

```markdown
## 콘티 보드 — [작품/캠페인] [씬/컷 범위]
> 프리셋: [드라마·영화 / 광고 15s·30s]

| 프레임 | 화각·앵글 | 프레임 설명 | 대사/내레이션 | 전환 |
|--------|-----------|-------------|---------------|------|
| 1 | WS / 아이레벨 | ... | ... | CUT |
| 2 | CU / 로우 | ... | ... | DISSOLVE |

### 프레임별 생성 프롬프트
[프레임 1 복붙 프롬프트]
...
```
→ 생성 실행: `moai-media:media-higgsfield-core` 위임.

## 6. 주의사항

- **생성 전 비용 고지·승인.** `moai-media`의 `get_cost` 결과를 사용자에게 고지하고 승인받은 뒤 생성한다(책임이 위임 과정에서 증발하지 않게).
- **컷 수는 감정 비트가 정한다** — 프레임 할당량 사고 금지.
- **톤·색온도 통일** — 한 시퀀스 안에서 밝기·색온도가 흔들리지 않게 프롬프트에 반복 명시.
- **광고는 첫 3초 후크** — 없으면 스킵률 급증(`ad-conti-preset.md`).
