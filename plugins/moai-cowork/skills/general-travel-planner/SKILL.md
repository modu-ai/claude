# 여행 플래너 (Travel Planner)

여행 일정, 맛집, 숙소, 예산을 한 번에 설계합니다. 국내·해외 여행 모두 지원하며, 조건에 맞는 일정표와 예산표를 산출합니다.

## 트리거 키워드

여행, 여행 일정, 여행 코스, 맛집, 카페, 숙소, 호텔, 게스트하우스, 여행 예산, 국내 여행, 해외 여행, 가족 여행, 혼행, 배낭여행, 힐링 여행, 허니문, 패키지, 자유여행

## 실행 규칙

1. 사용자 요청 수신 → 여행 유형(국내/해외, 동행 구성, 여행 스타일) 판별
2. `references/general-travel-planner.md` 로드 → 일정 설계 워크플로우에 따라 실행
3. `--deepthink` 또는 복잡한 다중 도시·장기 일정 → sequential-thinking MCP가 설치돼 있으면 `mcp__sequential-thinking__sequentialthinking`를 활용, 없으면 `ultrathink` 키워드 기반 심층 추론 또는 일반 단계 추론으로 대체
4. 일정표·예산표 생성 후 사용자 검토 요청

## 독립 실행 워크플로우

전략 가이드 파일 없이도 다음 단계로 실행 가능합니다:

1. **요청 유형 파악**: 목적지, 여행 스타일(저예산/중간/프리미엄), 동행 구성 확인
2. **핵심 조건 수집**: 출발일, 기간, 예산, 관심사(맛집/액티비티/휴양) 확인
3. **일정 설계**: 일자별 동선·활동·이동 계획을 담은 일정표 초안 작성
4. **예산·리스크 분석**: 항공·숙소·식비·액티비티 예산 추정, 비수기 절약 팁, 주의 사항 제시
5. **최종 정리**: 준비물 체크리스트, 예약 우선순위, 다음 단계 액션 아이템 제공

## 사용 예시

- "도쿄 5박 6일 여행 일정과 예산 계획을 짜줘"
- "제주 겨울 여행 숙소·맛집·코스를 추천해줘"
- "강릉 1박 2일 커플 여행 동선 만들어줘"
- "유럽 3개국 2주 배낭여행 일정과 이동 경로를 설계해줘"

## 2026 한국 여행 트렌드 컨텍스트

부킹닷컴 2026 글로벌 여행 트렌드 조사 (34개국 32,800명) 및 레이디경향 분석 반영:

### 주요 트렌드

| 트렌드 | 설명 | 수치 |
|--------|------|------|
| 짧고 자주 | 연간 국내여행 평균 3.14회 | 글로벌 평균(2.53회) 초과 |
| 아시아 여행 | 아시아 지역 평균 1.95회 | 글로벌 평균(1.38회) 초과 |
| 해외여행 의향 | 해외여행 계획 중 | 39% (아시아 평균 +15%p) |
| 로맨타지 | 소설·영화 속 장소 여행 | 관심 응답 76% |
| 콰이어트케이션 | 디지털 피로 회복형 조용한 여행 | 2026 핵심 키워드 |
| 웰니스 트래블 | 스파·요리·걷기 포함 건강 여행 | 급성장 중 |
| 스마트 숙소 | 로봇 도입 숙소 예약 의향 | 85% (글로벌 77%) |

### 여행 계획 시 고려사항

- **여만추 트렌드**: 41%가 여행 중 인간 관계(친구/로맨스) 추구
- **비수기 절약**: 1-2월, 7-8월 항공·숙박 최대 30-40% 할인
- **AI 활용 여행**: 맞춤형 일정 추천, 현지 식당 예약, 실시간 번역 활용 증가
- **지속가능 여행**: 친환경 숙소, 탄소 발자국 고려 이동 수단 선호 증가

## 문제 해결

| 상황 | 해결 방법 |
|------|-----------|
| 예산 정보 미제공 | 여행 스타일(저예산/중간/프리미엄)을 알려주시면 맞춤 예산 기준으로 설계합니다 |
| 일정 조건 부족 | 출발일, 기간, 동행 인원만 알려주셔도 기본 일정 초안을 작성해 드립니다 |
| 실시간 항공·숙소 가격 조회 | 실시간 가격은 직접 플랫폼(네이버 항공, 아고다 등)에서 확인하세요 |

## 관련 스킬 (후처리 체인)

이 스킬은 여행 일정 설명 같은 **글(텍스트) 산출물**을 만듭니다. 글 산출물은 발행 전 다음 체인으로 후처리하세요.

```
general-travel-planner → moai-cowork:general-ai-slop-reviewer → moai-cowork:general-humanize-korean
```

- **moai-cowork:general-ai-slop-reviewer**: AI 티 나는 표현·과장 어조를 검수하고 다듬습니다
- **moai-cowork:general-humanize-korean**: 자연스러운 한국어 문장으로 마무리합니다

여행 예산표 같은 **숫자·표 산출물**은 위 글 체인 대신 **moai-cowork:office-xlsx-creator**로 정리하세요. (혼합 산출물은 글 부분만 체인에 태우고, 표는 그대로 xlsx로 보냅니다.)
