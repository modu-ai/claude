Claude Chat의 강력한 기능들을 소개합니다.

| 기능 | 하는 일 | 실제 예시 |
|---|---|---|
| **파일 첨부** | 이미지, PDF, Word, Excel 등을 Claude에 업로드해 분석하거나 수정 | "이 PDF 계약서를 분석해 줄 수 있나요?" → Claude가 내용을 읽고 분석 |
| **이미지 분석** | 사진, 스크린샷, 다이어그램을 인식하고 설명 | 화면 캡처를 올려서 "이 오류 메시지가 뭔가요?" 물어보기 |
| **코드 설명** | 코드를 읽고 작동 방식을 설명하거나 버그 찾기 | "이 Python 코드의 버그를 찾아 줄 수 있나요?" |
| **음성 모드** | 텍스트 대신 음성으로 입력·출력 | 차 운전 중 손 자유로운 상태에서 음성 명령 가능 |
| **아티팩트 뷰어** | 생성된 코드나 문서를 별도 창에서 미리보기·다운로드 | Claude가 작성한 HTML, Python 코드를 바로 실행 |

## 각 기능 자세히 보기

### 1. 파일 첨부

**사용 시기**:
- 긴 문서를 읽고 요약받을 때
- 이미지 콘텐츠를 분석받을 때
- 엑셀 데이터를 정리받을 때

**예시**:
```
"이 마케팅 리포트(PDF 첨부)의 핵심 인사이트 3가지와 
다음 단계 추천사항을 알려 줄 수 있나요?"
```

### 2. 이미지 분석

**사용 시기**:
- 화면 오류 메시지 해석
- 사진의 개체 인식
- 다이어그램 설명

**예시**:
```
사용자가 에러 스크린샷을 첨부
"이 오류가 뭔가요?"
→ Claude가 스크린샷을 읽고 오류를 설명하고 해결책 제시
```

### 3. 코드 설명

**사용 시기**:
- 낯선 코드를 이해하고 싶을 때
- 코드의 버그를 찾을 때
- 코드를 다른 언어로 번역받을 때

**예시**:
```
사용자가 Python 코드 붙여넣기
"이 코드가 하는 일을 설명해 줄 수 있나요?"
→ Claude가 코드를 라인별로 설명
```

### 4. 음성 모드

**사용 시기**:
- 손이 바쁠 때 (운전 중, 요리 중)
- 빠르게 생각을 말하고 싶을 때
- 듣는 것이 편할 때

**사용 방법**:
1. 마이크 아이콘 클릭
2. 질문 말하기
3. 완료 클릭
4. Claude가 음성 또는 텍스트로 답변

### 5. 아티팩트 뷰어

Claude가 코드, HTML, 또는 문서를 생성할 때, 별도 창에서 미리보기할 수 있습니다.

**예시**:
```
사용자: "간단한 할일 목록 HTML을 만들어 줄 수 있나요?"
Claude: (HTML 코드 생성) → 아티팩트 뷰어에서 실시간 미리보기
```
