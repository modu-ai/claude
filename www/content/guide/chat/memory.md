새 노트북으로 갈아탈 때, 예전 노트의 메모를 옮겨 적어 두면 처음부터 다시 설명할 필요가 없죠. Claude의 메모리도 비슷합니다. 다른 AI 서비스에서 쌓아 둔 "나는 누구이고, 어떤 일을 하며, 어떤 스타일을 좋아하는지" 같은 맥락을 Claude로 가져오면 Claude가 그만큼 빠르게 당신을 이해할 수 있습니다.

반대로 Claude가 기억하고 있는 내용을 파일로 내보내 백업하거나 다른 곳으로 옮길 수도 있습니다. 이 글에서는 메모리를 가져오고 내보내는 방법을 차근차근 살펴봅니다.

{{< hint type="note" >}}
메모리 가져오기 기능은 아직 실험 단계로, 계속 개선되고 있습니다. 가져온 내용이 항상 답변에 그대로 반영되지는 않을 수 있습니다.
{{< /hint >}}

## 메모리 가져오기·내보내기란

다른 AI 서비스에서 사용하던 기억을 Claude로 옮기거나 Claude의 기억을 백업·이전하기 위해 파일로 꺼내는 기능입니다. 가져온 텍스트는 시스템이 자동으로 정보를 추출해 여러 개의 개별 "메모리 항목"으로 저장합니다.

| 항목 | 내용 |
|---|---|
| **지원 요금제** | Free, Pro, Max, Team |
| **지원 환경** | 웹(claude.ai), Claude Desktop 앱 |
| **가져오기 진입 경로** | 설정 > Capabilities > Memory, 또는 홈 화면의 'Import memory to Claude' 카드 |
| **내보내기 진입 경로** | 설정 > Capabilities |
| **반영 시점** | 가져오기 완료 후 24시간 이내 결과 확인 |

## 메모리 가져오기

다른 AI 서비스의 기억을 Claude로 옮기는 과정은 크게 네 단계입니다.

1. 현재 사용 중인 AI 서비스에서 메모리를 내보냅니다. 제공된 프롬프트 템플릿을 사용해 지침, 개인 정보, 진행 중인 프로젝트, 사용하는 도구, 선호 사항 등을 추출하세요.
2. Claude에서 **설정 > Capabilities > Memory** 섹션으로 들어가거나 홈 화면의 **'Import memory to Claude'** 카드를 엽니다.
3. 내보낸 텍스트를 붙여넣고 **'Add to memory'**를 클릭합니다.
4. 메모리 관리 화면에서 추출된 항목들을 확인합니다.

가져오기를 신청하면 그 자리에서 바로 모든 게 반영되는 것은 아닙니다. 시스템이 텍스트를 분석해 기억으로 정리하는 데 시간이 걸리며 결과는 24시간 이내에 확인할 수 있습니다.

{{< hint type="tip" >}}
저장된 기억은 채팅 중에 직접 물어볼 수 있습니다. "나에 대해 어떤 걸 기억하고 있나요?"처럼 물어보면 Claude가 기억하는 메모리를 확인할 수 있습니다.
{{< /hint >}}

### 무엇이 잘 옮겨지고 무엇이 빠질 수 있나요

가져오기 기능은 모든 정보를 똑같이 다루지 않습니다. 업무와 관련된 맥락을 우선해서 저장하는 경향이 있어, 일과 무관한 개인적인 내용은 빠질 수 있습니다.

| 잘 반영되는 편 | 빠질 수 있는 편 |
|---|---|
| 업무·프로젝트 관련 맥락 | 업무와 무관한 개인적 세부 정보 |
| 사용하는 도구와 작업 방식 | 일과 관련 없는 취향·신변 정보 |
| 작업 지침과 선호 스타일 | — |

{{< hint type="warning" >}}
가져온 기억이 답변에 일관되게 반영되지 않을 수 있고 업무와 무관한 개인 정보는 저장 과정에서 걸러질 수 있습니다. 중요한 정보라면 가져오기 후 메모리 관리 화면에서 직접 확인하는 것이 좋습니다.
{{< /hint >}}

## 메모리 내보내기

Claude가 기억하고 있는 내용을 파일로 꺼내 백업하거나 다른 곳으로 옮길 수 있습니다.

1. **설정 > Capabilities** 메뉴로 들어갑니다.
2. 메모리 내보내기 옵션을 선택합니다.
3. 내보낸 메모리 파일을 내려받아 로컬에 보관합니다.

내보낸 파일은 내 컴퓨터에 안전하게 백업해 두거나 필요할 때 다른 환경으로 옮기는 데 사용할 수 있습니다.
