# 세무 도우미 (Tax Helper)

## 개요

세금 관련 질문에 답변하고 신고 절차를 안내한다. 프리랜서 3.3% 원천징수 계산, 종합소득세, 부가가치세, 홈택스 신고 절차, 2026년 4대보험 요율 변경사항을 한국 세법 기준으로 제공한다.

**면책 고지: AI 어시스턴트이며 공인 세무/회계 자문을 대체하지 않습니다. 중요한 세무 사안은 반드시 세무사와 상담하세요.**

## 트리거 키워드

세금, 세무, 부가세, 3.3%, 원천징수, 종합소득세, 종소세, 홈택스, 사업자 세금, 연말정산, 절세

## 워크플로우

### 0단계: 입력 개인정보 마스킹

입력에 주민등록번호·계좌번호·연락처 등 개인정보가 포함되면 `moai-coworker:general-ai-slop-reviewer`의 `references/kr-pii-masking.md` 규칙으로 마스킹한 뒤 처리한다 (산출물 재출력 금지).

### 1단계: 세무 유형 확인

**세무 유형 분류:**
- **프리랜서/사업자 소득**: 3.3% 원천징수, 종합소득세, 부가가치세
- **근로소득**: 연말정산, 원천징수영수증
- **법인세**: 법인 결산, 법인세 신고
- **상속·증여**: 재산 상속, 증여세

### 2단계: 3.3% 원천징수 계산

**사업소득 원천징수 (3.3%):**
- 소득세 3% + 주민세 0.3% = **3.3%**
- 지급명세서 제출 의무 (지급일이 속하는 연도의 다음 연도 2월 말까지)

**계산 예시:**
- 100만원 수입 → 33,000원 원천징수 → 실수입 967,000원
- 300만원 수입 → 99,000원 원천징수 → 실수입 2,901,000원

**계산 다음의 필수 액션 — 신고·납부:**
- 3.3% 계산은 끝이 아닙니다. 원천징수한 세액은 **지급일이 속하는 달의 다음 달 10일까지** 원천세 신고·납부해야 합니다 (소득세법 제128조 제1항). 아래 5단계 캘린더의 "매월 10일" 항목과 연결됩니다.
- 사업소득 **간이지급명세서**도 매월 제출 대상입니다 (소득세법 제164조의3) — 홈택스에서 원천세 신고와 함께 처리하세요.

**원천징수 면제:**
- 지급받는 자가 사업자등록증을 제출한 경우
- 부가가치세 과세 용역으로 세금계산서를 발행하는 거래 등 (정확한 면제 요건은 국세청 nts.go.kr 확인)

### 3단계: 종합소득세 신고

**신고 기간:**
- **5월 1일 - 5월 31일** (종합소득세 신고 기간)

**신고 대상:**
- 이자·배당소득, 사업소득, 근로소득, 연금소득, 기타소득이 있는 거주자

**신고 방법 (홈택스):**
1. 국세청 홈택스 접속 (hometax.go.kr)
2. 신고/납부 메뉴 → 종합소득세 신고
3. 소득 내역 입력
4. 소득공제·세액공제 입력
5. 세액 확인 및 납부

**종합소득세율 (2026년 기준):**
| 과세표준 | 세율 | 누진공제 |
|----------|------|----------|
| 1,400만원 이하 | 6% | - |
| 1,400만원 초과 5,000만원 이하 | 15% | 126만원 |
| 5,000만원 초과 8,800만원 이하 | 24% | 576만원 |
| 8,800만원 초과 1억 5,000만원 이하 | 35% | 1,544만원 |
| 1억 5,000만원 초과 3억원 이하 | 38% | 1,994만원 |
| 3억원 초과 5억원 이하 | 40% | 2,594만원 |
| 5억원 초과 10억원 이하 | 42% | 3,594만원 |
| 10억원 초과 | 45% | 6,594만원 |

### 4단계: 부가가치세 신고

**신고 기간:**
- **1기 예정신고**: 4월 1일 - 4월 25일
- **1기 확정신고**: 7월 1일 - 7월 25일
- **2기 예정신고**: 10월 1일 - 10월 25일
- **2기 확정신고**: 익년 1월 1일 - 1월 25일

> 예정신고(4·10월)는 원칙적으로 법인사업자 대상이며, 개인 일반과세자는 직전기 납부세액의 1/2을 예정고지로 납부하는 것이 원칙입니다(소규모 법인 예외).

**부가가치세율:**
- **과세표준 10%**
- 면세품목 (예: 의료, 교육, 도서 등) 제외

**신고 방법 (홈택스):**
1. 국세청 홈택스 접속 (hometax.go.kr)
2. 신고/납부 메뉴 → 부가가치세 신고
3. 매출·매입세금계산서 내역 입력
4. 납부세액 또는 환급세액 확인
5. 신고 및 납부

### 5단계: 홈택스 신고 마감 캘린더

**매월 반복 의무 (급여·프리랜서 대금을 지급하는 모든 사업자):**

| 주기 | 신고 항목 | 마감일 | 근거 |
|------|---------|--------|------|
| 매월 | 원천세(원천징수이행상황신고) 신고·납부 | 지급월의 다음 달 10일 | 소득세법 제128조 제1항 |
| 매월 | 간이지급명세서 제출 (사업소득·근로소득) | 지급월의 다음 달 말일(사업소득 등 유형별 상이 — 국세청 nts.go.kr 확인) | 소득세법 제164조의3 |

> **반기납부 특례**: 상시고용 인원 20명 이하 소규모 사업자는 관할 세무서 승인을 받아 원천세를 반기별(1~6월분 → 7.10, 7~12월분 → 익년 1.10)로 신고·납부할 수 있습니다 (소득세법 제128조 제2항). 간이지급명세서 제출 주기는 별도이니 혼동하지 마세요.

**연간 캘린더:**

| 월 | 신고 항목 | 마감일 |
|----|---------|--------|
| 1월 | 부가세 2기 확정, 연말정산 간소화, 반기납부 원천세(하반기분) | 1.25 / 1.10 |
| 2월 | 연말정산 원천징수영수증, **사업장현황신고(면세사업자)** | 2.28 / **2.10** |
| 3월 | 법인세 신고 | 3.31 |
| 4월 | 1기 부가세 예정 | 4.25 |
| 5월 | 종합소득세 | 5.31 |
| 7월 | 부가세 1기 확정, 반기납부 원천세(상반기분) | 7.25 / 7.10 |
| 8월 | 주민세 | 8.31 |
| 9월 | 재산세 | 9.30 |
| 10월 | 2기 부가세 예정 | 10.25 |
| 11월 | 종합부동산세 | 11.30 |
| 12월 | 연말정산 준비 | 12.31 |

> **사업장현황신고**: 부가가치세 **면세사업자**(병의원·학원·농축수산물 도소매 등)는 전년도 수입금액·사업장 현황을 **다음 해 2월 10일까지** 신고해야 합니다 (소득세법 제78조). 과세사업자의 부가세 신고에 대응하는 면세사업자의 연 1회 의무입니다.

## 사용 예시

**예시 1**: "이번 달 프리랜서 수입 300만원인데 3.3% 계산해줘"
→ 300만원 × 3.3% = 99,000원 원천징수 → 실수입 2,901,000원

**예시 2**: "종합소득세 신고 기간이랑 방법 알려줘"
→ 5월 1일-31일 → 홈택스 접속 → 소득 내역 입력 → 공제 입력 → 신고

**예시 3**: "개인사업자 부가가치세 신고 어떻게 해?"
→ 1기/2기 구분 → 매출·매입 입력 → 납부세액 확인 → 홈택스 신고

**예시 4**: "홈택스에서 원천징수 영수증 어떻게 발급해?"
→ 홈택스 접속 → 증명발급 → 원천징수영수증 → 조회·발급

### 2026년 4대보험 요율

> 2026년 4대보험 요율표·세율표는 이 스킬의 `references/2026-rates.md` 단일 SSOT를 참조한다. 변경 시 해당 파일만 갱신한다.

## 문제 해결

- **세율 변경 확인 불가**: 세율은 매년 변경될 수 있습니다. 국세청 홈택스(hometax.go.kr) 또는 세법 개정 고시를 직접 확인하세요.
- **과거 데이터 미제공**: 종합소득세 신고 지원 시 전년도 소득·경비 내역이 필요합니다. 홈택스에서 소득 자료 조회 후 제공해 주세요.
- **지방세 연동 계산**: 지방소득세(소득세의 10%)는 별도 신고가 필요하며 위택스(wetax.go.kr)에서 처리합니다.
- **원천징수의무자 불분명**: 사업소득(3.3%)과 근로소득(연말정산)이 혼재된 경우 세무사와 소득 분류를 먼저 정리하세요.

## 관련 스킬

- **moai-accountant:finance-close-management**: 월말 결산, 급여 정산
- **moai-accountant:finance-financial-statements**: K-IFRS 재무제표 작성
- **moai-accountant:finance-variance-analysis**: 예산 대비 실적 분석

### 후처리 체인 (세무 안내문·신고 절차 설명·절세 가이드 등 서술형 산출물)

세무 안내문, 신고 절차 설명, 절세 가이드, 고객 회신문처럼 문장 산출물을 작성한 뒤에는 아래 체인으로 마무리한다.

```
finance-tax-helper → moai-coworker:general-ai-slop-reviewer → moai-writer:general-humanize-korean
```

- **moai-coworker:general-ai-slop-reviewer**: AI 티 나는 표현·과장·상투구 검수 및 수정
- **moai-writer:general-humanize-korean**: 한국어 자연스러움 보정 (서술 산출물 한정)

> 원천징수 계산 내역·세율표·신고 일정 캘린더 같은 표·숫자 산출물은 체인 대상이 아니며, 엑셀화가 필요하면 `moai-officer:office-xlsx-creator`로 라우팅한다.

## 이 스킬을 사용하지 말아야 할 때

- **실제 세금 신고·납부**: 홈택스 신고 대행은 공인 세무사에게 의뢰하세요. AI가 작성한 내용은 참고용입니다.
- **법인세 신고**: 법인 결산 및 법인세 신고는 `moai-accountant:finance-close-management` 스킬과 세무사를 활용하세요.
- **재무제표 작성**: K-IFRS 기준 재무제표 작성은 `moai-accountant:finance-financial-statements` 스킬을 사용하세요.
- **세무 조사 대응**: 세무조사 대응은 반드시 세무사·변호사와 함께 진행하세요.
