# 산출물 파일명 규약

> 파일명의 **공백은 모두 언더바(`_`)로 바꿔 빈칸 없이** 쓴다.

## 1. 규칙

- **공백 → 언더바.** 예: "우리집에는 철학자가 산다" → `우리집에는_철학자가_산다`
- **`EP##`는 두 자리**(EP01, EP12)로 쓰고, 작품제목과 에피소드제목 사이의 **구분자** 역할을 한다.
- 파일명에 못 쓰는 **금지 문자(`/ : * ? " < > |`)는 빼거나 비슷한 글자로 바꾼다.**
- 저장 위치: 현재 작업 폴더.

## 2. 두 파일 형식

| 산출물 | 파일명 형식 |
|--------|-------------|
| 전체 기획서 (딱 1개) | `[작품제목]_기획서.md` |
| 에피소드 원고 (화마다 1개) | `[작품제목]_EP##_[에피소드제목].md` |

## 3. 예시

작품 "우리집에는 철학자가 산다"의 ep01~03:

```
우리집에는_철학자가_산다_기획서.md                    ← 전체 기획서 (딱 1개)
우리집에는_철학자가_산다_EP01_소크라테스가_이사왔다.md   ← ep01 원고
우리집에는_철학자가_산다_EP02_월요일_출근_대참사.md      ← ep02 원고
우리집에는_철학자가_산다_EP03_라면은_정의로운가.md       ← ep03 원고
```

## 4. 주의

- EP는 항상 2자리(EP01, 한 자리 금지) — 정렬·구분자 일관성 때문.
- 제목에 콜론·물음표가 들어가면 비슷한 뜻의 한글로 바꾸거나 뺀다(예: `?` 제거).
