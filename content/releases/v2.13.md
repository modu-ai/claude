**릴리스 날짜**: 2026-05-20
**버전**: v2.13.0 (MINOR, 최신)
**업데이트 명령**: `/plugin marketplace update cowork-plugins`

## Highlights

`moai-media` 플러그인에 **Higgsfield MCP를 직접 호출**하는 신규 2 스킬을 도입했습니다. 자연어 한 줄 요청으로 5 이미지 모델·31 영상 모델 중 적합한 것을 자동 선택해 생성합니다. 캐릭터 일관성·DOP 카메라 디렉팅·립싱크·비동기 잡 폴링까지 통합 처리됩니다.

- **higgsfield-image**: Soul · Nano Banana · Flux · Seedream · Cinema Studio (5 모델)
- **higgsfield-video**: Sora 2 · Veo 3 · Veo 3.1 · Kling 3 · Kling 2.6 · Seedance · Minimax Hailuo · Wan 2.2/2.5/2.6 · Speak (infinite-talk) · DOP · Image2Video (31 모델)

## What's New

### higgsfield-image

자연어로 이미지를 요청하면 다음을 자동 처리:

| 단계 | 처리 |
|---|---|
| 1. 의도 추출 | 주제·용도·청중·톤 |
| 2. 모델 선택 | 5 모델 중 자동 (글자 → Nano Banana, 시네마틱 → Soul, 사진 → Flux 등) |
| 3. 파라미터 | width_and_height (5 비율) · quality (720p/1080p) · seed · style |
| 4. MCP 호출 | `mcp__higgsfield__generate_image` |
| 5. 비동기 폴링 | queued → in_progress → completed (5-15초) |
| 6. 결과·변형 | seed 변경·배치 4종·캐릭터 reference |

**캐릭터 일관성** 핵심:
- 1차: Soul로 인물 1장 생성
- 2차: `create_character`로 reference UUID 저장
- 3차+: `custom_reference_id`로 동일 인물 다양한 포즈

### higgsfield-video

영상은 더 복잡한 31 모델 + DOP 카메라 무브먼트:

| 단계 | 처리 |
|---|---|
| 1. 의도 추출 | 길이·용도·입력 유형 (text-to-video / image-to-video / speak) |
| 2. 모델 선택 | 31 모델 자동 (사실적 → Sora 2 / 인물 → Kling 3 / 빠른 시안 → Seedance) |
| 3. 파라미터 | aspect_ratio (16:9·9:16·1:1·4:5) · quality · duration · motion |
| 4. MCP 호출 | `mcp__higgsfield__generate_video` 또는 `generate_video_dop` |
| 5. 비동기 폴링 | 5-90초 (모델별) |
| 6. 결과 검수 | 모션·길이·자연스러움 |

**DOP 카메라 디렉팅**:
- 줌·팬·트래킹·돌리·틸트·특수 효과 (bullet_time·handheld·rotation)
- 단일/2개/3개 조합 가능
- strength 0.5-0.7이 자연스러움

**Speak (말하는 머리)**:
- 인물 이미지 1장 + 음성 → 립싱크 영상
- 음성은 `audio-gen` (ElevenLabs)으로 먼저 생성

## 워크플로우 예시

### 광고 8초 영상 (시네마틱)

```
1. higgsfield-image (Cinema Studio) → 시작 이미지 1장
2. higgsfield-video → image_url + Veo 3.1
3. prompt: "Slow camera pan revealing product"
4. 60초 대기 → 8초 영상
```

### 카드뉴스 이미지 4종 (글자 포함)

```
higgsfield-image:
- 모델: Nano Banana (글자 정확도 SOTA)
- 비율: 1024x1024 (인스타 피드)
- batch_size: 4
- 결과: 4종 후보
```

### CEO 신년 인사 영상

```
1. higgsfield-image (Soul) → CEO 정면 이미지
2. audio-gen (ElevenLabs) → 한국어 TTS 15초
3. higgsfield-video → Speak (infinite-talk)
4. 90초 대기 → 립싱크 영상
```

### 360도 제품 회전 영상

```
higgsfield-video → generate_video_dop:
- input_image_url: 제품 이미지
- motions: [
    { id: "orbit_360", strength: 0.8 },
    { id: "slow_zoom_in", strength: 0.5 }
  ]
```
