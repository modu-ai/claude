# SFX(효과음) 사전 (세로 스크롤 웹툰용)

> **성격:** SFX는 **조합형** — *소리(글자 내용) × 글자스타일(L) × 효과처리(E) × 배치.* 소리는 자유 입력, **스타일만 사전에서 고른다.** SFX는 2패스로 얹는다(→ `two-pass-overlay.md`).

## 1. 음 종류 → 글자스타일 + 효과 매핑 (고를 때 이 표)

| 음 종류 | 글자 스타일 | 효과 처리 | 대표(한글) |
|---------|------------|----------|-----------|
| 충격·타격 | L1 굵은블록 / L2 깨진 | E1 버스트 / E6 균열 | 쾅·콰직·퍽 |
| 동작·이동 | L4 기울어진 | E2 속도선 | 슈욱·쌩·타다닥·와다닥 |

## 2. 글자 스타일 팔레트 (L1~L16)

> **★ 이 팔레트는 천장이 아니라 바닥선이다.** 검증된 "확실히 되는" 스타일 모음일 뿐 "이것만 써라"가 아니다. 맥락만 주고 모델에 맡기는 경로도 항상 열려 있고, 표에 없는 스타일은 자유 묘사로 확장한다.
> **★ SFX 스타일 생성 공식:** `이름 토큰(예: Slime SFX·Electric SFX·Tremble SFX) + 형태 묘사 + 소리별 컬러`. 이름만 주면 뭉개지니 형태를 함께 박는다.
> **★ 가독성 트레이드오프:** 파쇄·긁힘·번개·비명처럼 강렬할수록 글자가 읽기 어려워진다 → 강한 변형은 감정 정점 컷·짧은 효과음에, 가독이 필요하면 강도를 낮춘다.

| # | 스타일 | 프롬프트 묘사(영/한) |
|---|--------|----------------------|
| L1 | 굵은 블록 | bold heavy block lettering / 굵고 묵직 |
| L2 | 깨진·각진 | jagged, cracked, fractured |
| L3 | 둥근·말랑 | round, bouncy, soft |
| L4 | 기울고 늘어진 | italic, stretched, dynamic |
| L5 | 물결·흔들 | wavy, wobbly, vibrating |
| L6 | 가는 점점이 | thin, tiny, faint dotted |
| L7 | 긁힌·거친 | scratchy, rough, brittle |
| L8 | 각진·픽셀 | square, pixel, blocky |
| L9 | 반짝·빛나는 | bright, sparkly, radiant |
| L10 | 흘러내림·녹음 | dripping, melting |
| L11 | 해체·붕괴 | deconstructed & crumbling — 굵은 블록이 조각나 낙하(위 온전·아래 파쇄) + 파편·균열·먼지 |
| L12 | 액체·점액 | liquid / slime — 매끈 광택 액체, 늘어짐·흘러내림·방울(물=blue / 점액=slime-green). 가독성 좋음 |
| L13 | 전기·글리치 | electric / glitch — RGB 어긋남·스캔라인 / 지그재그 lightning-bolt / 회로·dot-matrix. ⚠️강할수록 가독성↓ |
| L14 | 얼음·냉기 | ice — 얼음 결정 facets·균열선·서리·shard(icy blue-white), 날카로운 각진 획 |
| L15 | 화염·작열 | fire / blaze — 불꽃 혀·heat-shimmer·불티(red-orange-yellow) |
| L16 | 호러 | horror — 유혈 dripping(crimson) / 긁힘 clawed-torn(거친 black) / 비명 tall-jagged spike / 떨림 shaky duplicated outline. ⚠️긁힘·비명 강할수록 가독성↓ |

### L11 파편 질감 레버 (한글 자모 분리 전제)

> L11 해체·붕괴형은 실존 폰트가 아니라 모델이 **"한글 자모 분리 + 붕괴"**를 즉석 조합해 그리는 것이다. 획 스타일을 함께 지시하면 파편 질감까지 바뀐다.

- **획 스타일 = 파편 질감** — 고딕 = 각지고 단단한 **돌덩이 파편**(콘크리트·바위) / 손글씨 = 둥근 **잉크 방울·얼룩**(진흙·젖은 것) / 명조 등 응용.
- **두 레버는 별개** — **해체(자모 분리) = 구조** / **무너짐(낙하·파쇄) = 연출.** 무너짐은 `tumble downward · collapsing · scattering`을 따로 박아야 나온다(자모 분리만 주면 가지런히 벌어지기만 함).
- **실전 팁** — 순수 해체보다 **굵은 SFX 블록 파쇄형**(위 온전·아래 붕괴)이 더 잘 읽힌다. 컬러는 만화 팝 + 흰 테두리: 뜨거운 그라데이션(레드→오렌지 = 붕괴·충격) / 차가운(블루→시안 = 유리·얼음) / 먼지(브라운→오커 = 흙·콘크리트).

## 4. 배치 — 세로 스크롤 기준 (신규 1절)

- **위치 지정 = 중립 표현** — 캐릭터 이름 금지(모델은 완성본만 봄). **컷(N번째 패널) + 그림 속 식별(동작·위치·사물)**로 지정. 정밀 좌표 불필요.
- **크기:** 소리 세기·거리로 정한다(가까운 충격·큰 소리=크게 / 배경·먼 소리=작게). 화면·컷 대비 과대 금지, 캐릭터 얼굴·핵심 동작을 가리지 않게 여백에 앉힌다.
- **★ 세로 여백 띠 침범 금지** — SFX가 **컷 사이 세로 빈 띠를 침범해** 스크롤 리듬을 깨거나 아래 컷 그림을 덮지 않게 한다. 큰 SFX는 컷 내부 여백에서 키운다.
- **★ 1화면 과다 금지** — 모바일 한 화면(스크롤 한 눈)에 SFX를 몰지 않는다. 감정 정점·핵심 소리에 **1~2개만 집중**(모든 소리에 SFX 달지 않는다). 다 강조하면 아무것도 강조되지 않는다.
- 배치·크기의 상세 통제(동재질 함정·매체 일치·2패스 이점)는 `two-pass-overlay.md`.

## 5. 컷 설계 연동

SFX는 2패스라 1패스 그림엔 안 들어가지만, **어디에 무슨 효과음이 올지는 컷 설계 때 대사와 함께 미리 정리**한다. 그러면 ① SFX 맥락이 손에 있어 2패스가 빨라지고 ② 1패스 구도 잡을 때 **SFX 들어갈 자리(인물·핵심 안 가릴 여백)**를 미리 비워둘 수 있다. (컷 설계는 `story-webtoon-episode`.)
