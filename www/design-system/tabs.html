카테고리 탭 — 선택 시 surface-card 배경 + ink 텍스트, 비선택은 muted. 바닐라 JS로 ARIA 탭을 구현합니다.

카테고리 필터
category-tab-active (surface-card) · category-tab (muted)
ARIA 탭 (vanilla JS)

개요

Claude 따뜻한 에디토리얼 디자인 시스템은 크림 캔버스 위에 세리프 헤드라인과 코랄 액센트를 얹습니다.

설치

npm i @claude/design-tokens 로 토큰을 가져옵니다.

사용법

Tailwind config의 extend.colors에 토큰을 주입하세요.
