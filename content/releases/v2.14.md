## Highlights

Claude Design 관련 docs 5페이지와 moai-design 스킬 2종에 Anthropic 공식 출시 공지(2026-04-17) 내용을 정확히 반영했습니다. 신규 스킬·플러그인 추가는 없으며, 정확성·정합성 보완에 집중한 MINOR 릴리스입니다. 부수적으로 루트 README 카탈로그에서 누락되어 있던 moai-book·moai-design 두 행을 정정했습니다.

- **A. 코드 기반 프로토타입 (음성·비디오·셰이더·3D)** — Anthropic 공식이 강조한 영역을 명시. 인터랙티브 HTML+JS 출력 vs 독립 비디오 파일(.mp4) 미지원의 정확한 경계
- **B. Canva 네이티브 통합 (공식 파트너십)** — Canva CEO Melanie Perkins 인용 + 마케팅 후속 편집 워크플로우 mermaid
- **C. 통합 빌더 단기 로드맵 — "coming weeks"** Anthropic 공식 명시를 외부 추정과 분리
- **D. Brilliant·Datadog 공식 도입 사례 인용** — best-practices에서 원칙 1·9 신뢰성 강화
