# 개인 연말정산 절세 (Personal Tax Saver)

## 개요

근로소득자가 연말정산에서 돌려받는 돈(13월의 월급)을 키우도록, 소득공제와 세액공제 구조를 이해시키고 연중·연말에 챙길 항목을 안내한다. 사업자 세무가 아니라 **직장인 절세** 관점이다.

**면책 고지: 일반적인 세무 정보이며 공인 세무 자문을 대체하지 않습니다. 공제 한도·요건은 매년 바뀌므로 신고 전 홈택스·국세청 최신 안내를 확인하세요.**

## 트리거 키워드

연말정산, 13월의 월급, 소득공제, 세액공제, 환급, 절세, 연금저축, IRP, 신용카드 공제, 주택청약, 의료비 공제, 맞벌이 절세

## 워크플로우

### 1단계: 구조 이해 (소득공제 vs 세액공제)

- **소득공제**: 세금을 매기는 **과세표준(소득)을 줄임** → 세율이 높은 고소득자에게 절세 효과 큼
- **세액공제**: 계산된 **세금에서 직접 차감** → 누구에게나 정액 효과(연금저축·의료비·교육비·기부 등)

→ 본인 과세표준 구간을 먼저 파악하면 어떤 공제가 유리한지 보인다.

### 2단계: 소득공제 항목 점검

| 항목 | 핵심 |
|------|------|
| 신용카드 등 사용액 | 총급여 25% 초과분부터 공제. 체크카드·현금영수증 공제율이 신용카드보다 높음 → 25% 넘긴 뒤엔 체크/현금 활용 |
| 주택청약저축 | 무주택 세대주 요건 시 납입액 일부 소득공제 |
| 주택자금(전세·주택담보 이자) | 요건 충족 시 공제 |
| 인적공제 | 부양가족(소득·나이 요건) 기본·추가공제 |

### 3단계: 세액공제 항목 점검 (절세 핵심)

| 항목 | 핵심 |
|------|------|
| 연금저축 + IRP | 노후 대비 + 세액공제(연 납입 한도 내). 직장인 절세의 1순위 — 환급과 노후를 동시에 |
| 의료비 | 총급여 3% 초과분, 본인·부양가족 |
| 교육비 | 본인·자녀 교육비 |
| 기부금 | 법정·지정기부금 |
| 월세액 | 무주택 근로자 요건 시 세액공제 |
| 보장성 보험료 | 한도 내 세액공제 |

### 4단계: 연중·연말 액션

- **연중**: 연금저축·IRP 자동납입, 카드 사용 25% 넘긴 후엔 체크/현금 전환, 기부·의료비 영수증 챙기기
- **12월 마감 전**: 연금저축·IRP 추가납입으로 한도 채우기, 청약·월세 요건 점검
- **간소화 기간(1월)**: 홈택스 연말정산 간소화 자료 확인, 누락 자료(안경·교복·기부 일부) 수동 추가

### 5단계: 맞벌이·시뮬레이션

- **맞벌이**: 부양가족·의료비·신용카드 공제를 누구에게 몰지 소득·세율로 최적화
- **환급 시뮬레이션**: 홈택스 '연말정산 미리보기'로 예상 환급액 확인 후 부족분 보완

## 사용 예시

**예시 1**: "연말정산 환급 더 받고 싶은데 12월 전에 뭘 해야 해?"
→ 연금저축·IRP 한도 점검 → 카드 사용 구조 → 청약·월세 요건 → 마감 액션 리스트

**예시 2**: "소득공제랑 세액공제 차이가 뭐야?"
→ 과세표준 축소 vs 세금 직접 차감 구조 + 본인 구간별 유불리

**예시 3**: "맞벌이인데 부양가족 누구한테 넣는 게 유리해?"
→ 소득·세율 비교 → 공제 배분 시나리오 + 미리보기 활용 안내

## 산출물

- 소득공제 vs 세액공제 구조 요약
- 항목별 점검 체크리스트
- 연중·연말 액션 캘린더
- 맞벌이 공제 배분 가이드
- 환급 시뮬레이션 절차

## 주의사항

- **면책**: 공제 한도·요건·세율은 매년 개정된다. 신고 전 국세청·홈택스 최신 안내 확인.
- 과다·허위 공제는 가산세 대상이다. 요건과 증빙을 정확히.
- 연금저축·IRP는 중도 인출 시 세제 혜택이 추징될 수 있다.
