# Chat vs Cowork vs Desktop — 세 제품 경계

Claude의 세 가지 제품은 서로 다른 목적을 가집니다. 이 페이지는 어느 것을 언제 써야 하는지를 한눈에 정리합니다.

## 한 줄 요약

- **Chat** (`claude.ai`): 브라우저에서 하는 대화형 Q&A. 파일 시스템 접근이나 자율 작업이 없습니다.
- **Cowork**: 에이전트 모드. 격리된 VM에서 다단계 작업을 자율 실행하고 완성된 산출물(엑셀·파워포인트·워드·PDF)을 만듭니다. 메시지 입력창에서 "Cowork"를 선택해 진입합니다.
- **Claude Desktop**: 데스크톱 응용프로그램(그릇). Cowork를 비롯한 Claude 기능을 데스크톱에서 호스팅합니다.

> 핵심: Desktop은 **그릇**, Cowork는 그 안의 **에이전트 모드**, Chat은 **일반 대화**입니다.

## 비교 표

| 구분 | Chat | Cowork | Desktop |
|------|------|--------|---------|
| 실행 위치 | 브라우저 | 격리된 VM | 데스크톱 앱 |
| 파일 접근 | 없음 | 폴더 범위 허용 | 호스트 연동 |
| 산출물 | 대화 텍스트 | Office 파일(실물) | — |
| 자율 실행 | 없음 | 있음 | Cowork 호스팅 |
| 진입 | claude.ai | 입력창 "Cowork" 선택 | 앱 실행 |

## 언제 뭘 쓰나요

- **빠른 질문·대화** → Chat
- **"보고서 만들어줘"처럼 완성된 결과물이 필요할 때** → Cowork
- **데스크톱에서 Claude를 자주 쓸 때** → Claude Desktop (그 안에서 Chat·Cowork 모두 가능)
