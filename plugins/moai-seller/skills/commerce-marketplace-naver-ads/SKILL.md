# 네이버 검색광고 풀세트 최적화 (Marketplace Naver Ads)

## 개요

한국 셀러의 1순위 유입 채널인 네이버 검색광고를 자연어 한 줄 입력으로 분석·최적화하는 전담 스킬입니다. 스마트스토어 셀러의 워크플로우에서 "상품 등록(`commerce-marketplace-naver`) → **광고 유입(본 스킬)** → 성과 분석(`marketing-performance-report`)"의 유입 단계를 담당합니다.

**책임 한 줄**: 네이버 광고 보고서 입력 → 4 캠페인 유형 자동 분류 + 키워드·소재 단위 진단 + 엔드 ROAS 대비 의사결정 분기 + 입찰가·제외 키워드 조정안.

## 네이버 검색광고 4종 캠페인 분류

| 캠페인 | 노출 위치 | 과금 | 적합한 상황 |
|--------|----------|------|------------|
| **쇼핑검색광고** | 네이버쇼핑 검색 결과 (상품 카드) | CPC | 스마트스토어·쇼핑윈도 상품 판매 — 셀러 1순위 |
| **파워링크** | 통합검색 상단·하단 텍스트 광고 | CPC | 자사몰 유입, 서비스·비상품 랜딩, 브랜드 방어 |
| **브랜드검색** | 브랜드 키워드 검색 시 상단 대형 배너 | 기간 정액 | 브랜드 지명검색량이 충분히 확보된 이후 |
| **GFA (성과형 디스플레이)** | 네이버 메인·밴드·지면 네트워크 배너 | CPC/CPM | 리타겟팅·신규 도달 확장 (검색 수요 밖 잠재고객) |

**진입 원칙**: 스마트스토어 셀러는 **쇼핑검색광고부터** 시작합니다. 파워링크는 자사몰·기획전 랜딩이 있을 때, 브랜드검색·GFA는 지명검색·재방문 모수가 쌓인 뒤 확장합니다.

## 축 1 — 쇼핑검색광고 (셀러 핵심)

### 상품 소재 진단

쇼핑검색광고의 소재는 별도 배너가 아니라 **상품 카탈로그 정보 그 자체**입니다. 소재 품질 = 상품명·이미지·리뷰·가격 경쟁력.

| 진단 항목 | 점검 내용 | 연계 스킬 |
|----------|----------|----------|
| 상품명 | 검색 키워드 포함 여부·카테고리 정합성 | `commerce-product-naming` |
| 대표 이미지 | 썸네일 클릭 경쟁력 (CTR 저하 시 1순위 교체 대상) | `commerce-product-image-pipeline` |
| 리뷰 | 최소 사진 후기 1 + 글 후기 1 확보 후 증액 (전환율 직결) | — |
| 가격·배송 | 동일 검색면 경쟁 상품 대비 가격·배송비 포지션 | `commerce-margin-calculator` |

### 키워드·입찰가 운영

- 쇼핑검색광고의 키워드는 **직접 등록이 아니라 상품 정보 기반 자동 매칭**이므로, 운영 레버는 (a) 상품명·카테고리 정비, (b) **제외 키워드 등록**, (c) 입찰가 조정입니다.
- 입찰가는 최소 입찰가에서 시작해 **노출 순위·CPC·전환 데이터를 보고 단계 조정**합니다. 권장 절차: 소액(일예산 하한)으로 1-2주 데이터 수집 → 전환 키워드 확인 → 전환 나오는 상품만 증액.
- 최소 입찰가·일예산 하한 등 구체 수치는 네이버 정책 변동 대상이므로 광고시스템 화면 기준으로 확인합니다 (확인 필요).

### 확장소재

쇼핑검색광고·파워링크 모두 확장소재(추가 노출 요소)를 지원합니다. 유형(추가 문구·홍보 이미지·카운트다운 등)은 캠페인별 지원 범위가 다르므로 광고시스템의 확장소재 탭 기준으로 적용하고, 적용 후 **CTR 변화를 소재 단위로 비교**합니다.

## 축 2 — 파워링크 (키워드 매칭·품질지수)

### 키워드 매칭 설계

| 구분 | 전략 |
|------|------|
| 대표 키워드 (예: "원피스") | 검색량 크지만 CPC 높고 의도 분산 — 소액 테스트만 |
| 세부 키워드 (예: "하객 원피스 여름") | CPC 낮고 구매 의도 명확 — **세부 키워드 다수 확보가 기본기** |
| 브랜드 키워드 (자사 지명) | 저CPC 고전환 — 방어 목적으로 상시 운영 |
| 제외 키워드 | 클릭만 발생하고 전환 0인 검색어를 보고서에서 추출해 상시 등록 |

### 품질지수

파워링크 품질지수는 7단계 막대로 표시되며, **키워드-소재-랜딩페이지의 연관도**가 핵심 결정 요인입니다. 품질지수가 높으면 같은 입찰가로 더 좋은 순위(또는 같은 순위를 더 낮은 CPC로) 달성이 가능합니다.

개선 순서:
1. 키워드를 광고그룹 단위로 **주제별 세분화** (한 그룹에 잡다한 키워드 금지 — 소재 문안과의 연관도 하락)
2. 소재 문안에 그룹의 핵심 키워드를 자연스럽게 포함
3. 랜딩페이지를 키워드 의도와 일치시키기 (대표 키워드 → 카테고리, 세부 키워드 → 해당 상품)

## 축 3 — 브랜드검색·GFA (확장 단계)

- **브랜드검색**: 지명검색량이 유의미할 때만. 기간 정액제이므로 "월 지명검색수 × 예상 CTR"로 기대 유입을 먼저 추산해 정액 대비 손익을 계산합니다 (조회수 기준 단가는 상품 유형·기간별 상이 — 확인 필요).
- **GFA**: 검색 수요를 다 소진한 뒤의 확장 채널. 스마트스토어 방문자·구매자 리타겟팅부터 시작하고, 콜드 타겟 배너는 소재 제작비 대비 효율을 별도 검증합니다.

## 엔드 ROAS (본전 ROAS) 기준 의사결정

적자가 나지 않는 최소 광고수익률. end-ROAS 공식은 moai-seller:commerce-margin-calculator references/end-roas.md 참조.

요약: `엔드 ROAS = (판매가 ÷ 마진) × 100`. 예시(1만원 상품, 원가+배송비 8천원 → 마진 2천원)면 엔드 ROAS 500%.

> **네이버 ROAS 해석 주의**: 네이버 광고 보고서의 전환매출은 전환 추적 설정(프리미엄 로그 분석 등)과 어트리뷰션 기간에 따라 달라집니다. 스마트스토어 실제 결제액과 대사(對査)한 뒤 의사결정하십시오.

## 워크플로우 — 보고서 업로드 기반 진단

**입력**: 네이버 광고시스템 보고서 (엑셀 — 캠페인 > 광고그룹 > 키워드/소재 단위, 기간 최소 2주 권장) + 상품 원가·배송비·수수료 (엔드 ROAS 계산용)

```
[Step 1] 캠페인 유형 분류 (쇼핑검색/파워링크/브랜드검색/GFA)
       ↓
[Step 2] 엔드 ROAS 계산 (상품별) → 실제 ROAS와 비교
       ↓
   ┌─────────────────────────────────────────────────────────┐
   │ A. ROAS > 엔드 ROAS × 1.5 → 입찰가·예산 증액 + 별도     │
   │    광고그룹 분리 (전환 키워드 모아서)                     │
   │ B. ROAS < 엔드 ROAS       → 살릴 수 있는지 진단          │
   │     ├─ CTR 높음 + CVR 낮음 → 상세페이지·리뷰·가격 개선   │
   │     ├─ CTR 낮음 + CVR 높음 → 썸네일·상품명·소재 문안 개선│
   │     └─ 둘 다 낮음          → 제외/입찰 하향 (타 상품 집중)│
   └─────────────────────────────────────────────────────────┘
       ↓
[Step 3] 키워드 레벨 정리
        - 클릭 多 + 전환 0 → 제외 키워드 등록안
        - 전환 발생 세부 키워드 → 파워링크 별도 그룹 승격안
       ↓
[Step 4] 품질지수·소재 개선안 (파워링크) / 상품 소재 개선안 (쇼핑검색)
       ↓
[Step 5] 확장 판단 — 브랜드검색·GFA 진입 조건 충족 여부
```

**출력**:
1. 캠페인·상품·키워드별 ROAS·CTR·CVR·엔드 ROAS 표
2. 증액/개선/제외 3분류 의사결정 표
3. 제외 키워드 등록안 + 세부 키워드 승격안
4. 품질지수·소재 개선 체크리스트
5. 브랜드검색·GFA 확장 판단 (진입/보류 + 근거)

## 사용 예시

```
"네이버 광고 보고서 분석해줘" (엑셀 첨부)
→ 4종 분류 → 상품별 엔드 ROAS 대비 판정 → 제외 키워드 12개 + 증액 상품 2개 제안

"파워링크 품질지수가 낮아"
→ 광고그룹 키워드 구성 진단 → 그룹 세분화안 + 소재 문안 키워드 반영안

"브랜드검색 할 만해?"
→ 지명검색량 확인 절차 안내 → 정액 대비 기대 유입 추산 → 진입/보류 판정
```

## 주의사항

- 최소 입찰가·일예산 하한·브랜드검색 단가·확장소재 지원 유형 등 **네이버 정책 수치는 수시 변동**되므로 본 스킬은 수치를 단정하지 않고 광고시스템 화면 기준 확인을 안내합니다 (확인 필요 라벨 유지).
- 광고 문구(소재 문안·확장소재 텍스트)는 발행 전 `commerce-ad-claim-compliance-kr`(표시광고법·식약처·전상법 게이트)를 통과시킵니다.
- 본 스킬은 광고 계정 상태를 직접 변경하지 않습니다 — 조정안 산출까지가 책임 범위입니다.
