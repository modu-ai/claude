# 한국 드라마·영화 대본 표기 관행

> 한국 대본은 S# 넘버링과 국문 지문/대사 관행을 기본으로 한다. 아래는 통용되는 표기 관습이며, 방송사·제작사·공모전마다 세부 양식이 다를 수 있으므로 제출 전 해당 요강을 확인한다. 분량·회당 씬 수 같은 수치는 작품·편성별로 다르므로 `[미확인 — 제작사/공고 요강 확인]`로 둔다.

## 1. 씬 헤더 — S# 넘버링

```
S#12. 카페 '노을' / 낮 / 내부
```

- **S#** 뒤에 씬 번호(연속). 회차가 나뉘면 회차 표기와 함께 관리.
- **장소 / 시간(낮·밤·새벽 등) / 내·외부**를 슬래시로 구분.
- 씬 = 한 장소·한 시간대. 장소나 시간이 바뀌면 새 S#.

## 2. 지문

- 인물·상황·행동을 **현재형**으로 기술.
- 카메라 지시(클로즈업 등)는 대본에서 절제 — 연출 영역. 감정·행동 위주.
- 들여쓰기·줄 간격은 제작사 양식을 따른다(통상 지문은 좌측 정렬 또는 소폭 들여쓰기).

## 3. 대사

```
지은
  (조심스럽게)
  …그 얘기, 지금 꼭 해야 돼?

민준
  응. 지금 아니면 못 해.
```

- 인물명을 대사 위에 두고, 대사는 들여쓴다.
- 감정·행동 지시는 대사 앞 **괄호 지문**으로 짧게.
- 방백·나레이션은 관행 표기(예: (N), (E) — 제작사 양식 확인).

## 4. 전환

- 기본 전환(CUT)은 생략 가능(암묵 전제). 특수 전환만 명시.
- 국문/영문 대응은 `transition-vocabulary.md`.

## 5. 분량·회당 씬 수

- 회당 원고 분량·씬 수는 편성 형태·러닝타임에 따라 다르다 → `[미확인]`, 편성 확정·레퍼런스작으로 조율.
- 200자 원고지 / A4 환산 기준도 제작사별 상이 → 제출 요강 확인.

## 6. 제출 구성

제작사·공모 제출 시 대본 앞에 붙이는 통상 구성:

- **표지**: 제목 / 장르 / 작가 / 연락처 / (공모) 응모 부문.
- **인물 소개**: 주요 인물 관계·성격(시놉시스에서 이관).
- **기획의도 / 시놉시스**: `story-synopsis` 산출물 연계.
- **본문 대본**: 요구 회차 분량.

## 7. 안티패턴

- 미국식 씬 헤딩(INT./EXT.)을 한국 드라마 대본 기본값으로 사용 → 관행 불일치. 필요 시 병기만.
- 분량·양식 임의 단정 → 반려. 제출 요강 확인.
