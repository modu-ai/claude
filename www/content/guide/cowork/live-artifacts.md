# 라이브 아티팩트 (Live Artifacts)

Cowork 안에서 만들어지는 **대화형·지속형 산출물**입니다. 단순 텍스트가 아니라 미리보기·편집·다운로드가 가능한 인터랙티브 결과물입니다.

## 일반 아티팩트와의 차이

| 구분 | Chat 아티팩트 | 라이브 아티팩트 |
|------|--------------|-----------------|
| 만들어지는 곳 | 일반 대화 | Cowork 자율 실행 흐름 |
| 지속성 | 대화 안에서 | 세션·프로젝트에 지속 |
| 상호작용 | 미리보기 | 실시간 편집·갱신 |
| 갱신 | 정적 | 작업 진행에 따라 자동 갱신 |

## 지원하는 산출물 유형

- **문서** — 보고서·기획서·회의록
- **스프레드시트** — 데이터 표·계산식 포함
- **슬라이드** — 프레젠테이션 덱
- **차트·대시보드** — 시각화
- **인터랙티브 앱** — 동작하는 작은 도구

## 사용 방법

1. Cowork에 작업을 설명해 **생성**
2. 산출물을 **미리보기**
3. 필요하면 **편집**
4. 완성하면 **다운로드**
5. 팀과 **공유**
