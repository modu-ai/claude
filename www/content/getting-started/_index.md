Claude Desktop은 하나의 프로그램 안에 여러 가지 일하는 방식이 들어 있는 "AI 사무실"입니다. 처음에는 그냥 대화창(챗)으로 시작하지만, 익숙해질수록 파일을 다루는 코워크, 그림과 화면을 만드는 디자인, 프로그램을 만드는 코드까지 단계적으로 넓혀갈 수 있습니다. 이 섹션은 그 첫걸음 — 설치, 기본 개념, 첫 요청 — 을 순서대로 안내합니다.

낯선 용어가 나와도 걱정하지 않으셔도 됩니다. 이 가이드의 모든 용어는 처음 등장할 때 일상 비유와 함께 풀어서 설명합니다. 예를 들어 **플러그인**은 "회사에 새로 들어오는 직원 패키지", **스킬**은 "그 직원이 가지고 온 업무 매뉴얼"이라고 생각하면 됩니다. 컴퓨터를 켜고 인터넷을 쓸 수 있는 분이라면 누구나 따라올 수 있도록 구성했습니다.

## 데스크탑 앱으로 무엇을 할 수 있나요 — 4단계 여정

Claude Desktop 활용은 보통 네 단계의 여정으로 넓어집니다. 운전을 배우는 과정에 비유하면, 처음엔 동네 마트까지 가보고(챗), 다음엔 짐을 싣고 나르고(코워크), 그다음엔 차를 꾸미고(디자인), 마지막엔 정비까지 직접 하게 되는(코드) 흐름입니다. 중요한 것은 **모든 단계를 다 배울 필요가 없다**는 점입니다. 챗만 잘 써도 이메일·보고서·요약 같은 일상 업무의 절반이 해결됩니다.

1. **챗(Chat)** — 대화창에 질문하고 답을 받는 가장 기본 단계입니다. 이메일 초안, 문서 요약, 아이디어 정리가 여기서 이루어집니다.
2. **코워크(Cowork)** — Claude가 내 컴퓨터의 작업 폴더에 직접 파일을 만들고 고치는 단계입니다. PPT, 엑셀, 보고서 파일이 실제로 폴더에 생깁니다.
3. **디자인(Design)** — 카드뉴스, 발표 슬라이드, 웹 화면 시안 같은 시각 결과물을 만드는 단계입니다.
4. **코드(Code)** — 프로그램과 웹사이트를 직접 만들고 고치는 단계입니다. 개발자가 아니어도 간단한 자동화부터 시작할 수 있습니다.

```mermaid

    B --> C["③ 디자인<br/>시각 결과물 제작"]
    C --> D["④ 코드<br/>프로그램 제작"]

    style D fill:#d6ebe7,stroke:#1c7c70,stroke-width:2px,color:#09110f
```

각 단계에서 "직원"을 채용해 쓸 수 있습니다. 모두의 클로드 마켓플레이스에는 문서 직원(코워커), 마케터, 셀러, 디자이너 같은 전문 직원 플러그인이 준비되어 있어서, 필요한 직원만 골라 데려오면 됩니다. 직원 채용 방법은 [첫 작업](first-task/) 문서에서 다룹니다.

## 이 섹션의 읽는 순서

아래 순서대로 읽으시는 것을 권장합니다. 급하다면 ①과 ④만 읽어도 첫 결과물까지 도달할 수 있고, 중간에 용어가 낯설게 느껴질 때 ②로 돌아오면 됩니다.

1. **[설치 가이드](install/)** — Claude Desktop을 내 컴퓨터에 설치하고 로그인하기. 요금제가 어떻게 다른지도 여기서 봅니다.
2. **[도구와 기능 개념 기초](concepts/)** — 채팅, Projects, 스킬, 플러그인, 에이전트, MCP 커넥터, 아티팩트 등 앞으로 계속 만날 아홉 가지 개념을 비유로 정리합니다.
3. **[빠른 시작](quick-start/)** — 전체 동선과 대표 스킬을 한눈에 조망합니다.
4. **[첫 작업](first-task/)** — 마켓플레이스에서 첫 직원을 채용하고 실제 요청을 던져 결과물을 받아봅니다.

## 기대 효과

이 섹션을 끝까지 따라오면 다음 상태가 됩니다.

- Claude Desktop이 설치되어 있고 내 계정으로 로그인되어 있습니다.
- "플러그인이 뭐예요?"라는 질문에 스스로 답할 수 있습니다.
- 마켓플레이스에서 내 일에 맞는 직원을 골라 채용할 수 있습니다.
- 첫 요청을 던져 실제 결과물(문서·표·슬라이드)을 받아본 경험이 생깁니다.
