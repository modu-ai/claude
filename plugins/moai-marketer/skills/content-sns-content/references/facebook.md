## 플랫폼 특성 (2026)

- **디스커버리 엔진 전환**: 피드의 **35-50%가 미연결·추천 콘텐츠** — 팔로우 기반 브로드캐스트에서 관심사 기반 추천으로 이동
- **릴스 우선 (Reels-first)**: "모든 영상은 이제 릴스" — 페이스북이 릴스에 오가닉 도달을 몰아줌
- **그룹(Groups) 우대**: 알고리즘이 그룹을 의도적으로 우대(2027까지 MSI/Andromeda 신호) — 커뮤니티가 도달의 핵심 자산
- **3표면 구조**: 릴스(발견) → 피드(연결 심층) → 그룹(커뮤니티) 조합으로 운영

---

## 알고리즘·도달 메커니즘 (2026)

- **페이지 오가닉 도달 ~1.65%**: 단순 페이지 브로드캐스트는 사실상 종료
- **속도(velocity) 가중치**: 게시 후 **30분 내 댓글 10개 → 도달 3-5배** 증폭
- **결정 창(window)**: 첫 **30-60분**의 인터랙션이 전체 도달 궤도를 결정
- **NLP 콘텐츠 매칭**: 알고리즘이 캡션·자막을 읽어 관심 주제와 매칭 → 소셜 SEO 키워드 중요
- **주제 태깅**: 페이지의 **최근 9-12개 게시물**로 주제를 판단 → 테마를 2개 내외로 일관 유지
- **저장·공유 우선**: 좋아요보다 **저장·공유** 신호를 상위 최적화 지표로

### 표면별 도달 비교

| 표면 | 오가닉 도달 | 역할 |
|------|-----------|------|
| 릴스 | 페이지 게시물 대비 5-10배 | 신규 발견·바이럴 |
| 페이지 피드 | ~1.65% | 연결 팔로워 심층 콘텐츠 |
| 그룹 | 40-50% (페이지 대비 10-30배) | 커뮤니티·전환 |

## 잘 먹히는 콘텐츠 유형

- **릴스**: 페이지 게시물 대비 5-10배 성과. **첫 3초**가 리텐션·완주율 결정
- **피드 심층 콘텐츠**: 사진, 캐러셀, 3-8분 영상 (연결 팔로워 대상)
- **릴스 + 피드 페어링**: 릴스로 발견을 만들고 피드 게시물로 증폭
- **그룹 콘텐츠**: 투표, 숏폼 영상(주 3회 내외), 양방향 프롬프트 — **5배 인게이지먼트·2배 재방문**
- **앵커 테마**: 명확한 1명의 시청자를 겨냥한 2개 내외 테마 + 소셜 SEO 키워드

---

## 발행 주기 (Cadence)

- **릴스**: 주 3-5회 (발견 표면 유지)
- **피드**: 주 3-5회 (연결 팔로워 심층)
- **그룹**: 매일 다양한 게시물(투표·프롬프트·숏폼) + **게시 후 30-60분 내 직접 응답** 필수
- 페이스북은 메타 생태계로 인스타그램과 피드 리듬을 공유 → 한국 기준 최적 시간대는 인스타그램 가이드(평일 12-13시·19-21시, 주말 10-12시·20-22시)와 동일 패턴 적용

## CTA·링크 배치 패턴

- **링크는 첫 댓글에** (캡션 아님) — 외부 링크 포함 게시물은 도달이 억제됨
- **저장·공유 유도** CTA를 우선 (좋아요보다 알고리즘 신호 강함)
- **양방향 질문·투표**로 댓글 속도 확보 → 첫 30-60분 velocity 극대화
- 그룹에서는 **가입 후 30일간 판매 금지** — 신뢰 선구축 후 전환

## 광고 vs 오가닉

| 구분 | 오가닉 | 광고 |
|------|--------|------|
| 페이지 도달 | ~1.65% (브로드캐스트 종료) | 예산 기반 도달 확보 |
| 최선 전략 | 릴스·그룹으로 도달 우회 | 릴스 소재 + CAPI 전환 추적 |
| 학습 기간 | 해당 없음 | **활성화 후 48-72시간 무수정** (SKILL.md 메타 학습 기간 가이드 참조) |
| 전환 추적 | — | iOS 14.5+ 이후 pixel-only 신뢰 저하 → CAPI + GA4 교차 검증 필수 |

- **광고 규격**: 피드 광고 1200×628px(1.91:1), 스토리 광고 1080×1920px(9:16)
- **금지 관행**: 인게이지먼트 베이트, 매수한 참여, 대량 복붙 재사용, 동일 게시물 다수 그룹 자동 살포(스팸 위반), 조작된 지표

## 콘텐츠 체크리스트

- [ ] 영상은 릴스로 업로드 (일반 영상 게시물보다 5-10배 도달)
- [ ] 첫 3초 내 시선 집중 후크 (릴스 완주율 결정)
- [ ] 캡션 첫 125자 안에 핵심 메시지 (더 보기 전)
- [ ] 자막 필수 (무음 시청 대응)
- [ ] 외부 링크는 첫 댓글에 배치
- [ ] 저장·공유·양방향 CTA 명시
- [ ] 게시 후 30-60분 내 댓글 적극 응답 (velocity 확보)
- [ ] 최근 9-12개 게시물 테마 일관성 유지 (2개 내외 앵커 테마)
- [ ] 세로 4:5(1080×1350) 또는 릴스 9:16(1080×1920) 규격 준수
- [ ] 그룹: post-and-ghost 금지, 가입 30일 내 판매 금지
