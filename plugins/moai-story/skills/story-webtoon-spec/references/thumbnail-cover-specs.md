# 썸네일·표지 규격

> **원칙:** "썸네일·표지가 있다"·"모바일에서 축소돼 보인다"는 사실. 비율·픽셀·최소 크기 **수치는 `[미확인 — 공식 안내 확인 필요]`.**

## 1. 노출 지점 (사실)

| 요소 | 어디에 쓰이나 |
|------|---------------|
| 썸네일/타이틀 카드 | 작품 목록·추천 영역의 작은 대표 이미지 |
| 카카오페이지 표지 | 작품 상세·목록의 세로형 표지 |
| 네이버시리즈 표지 | 웹툰·웹소설 유통 표지 |
| 웹소설 표지 | 텍스트 작품의 대표 이미지 |

## 2. 수치 표 ([미확인] 표기)

| 항목 | 값 | 확인 방법 |
|------|-----|-----------|
| 썸네일 비율/픽셀 | `[미확인]` | 각 플랫폼 공식 이미지 규격 안내 |
| 표지 비율(세로형) | `[미확인]` | 공식 표지 규격 안내 |
| 최소 크기 | `[미확인]` | 공식 규격 안내 |
| 파일 포맷/용량 | `[미확인]` | 공식 업로드 안내 |

## 3. 가독 하한 원칙 (사실)

- 썸네일·표지는 **모바일에서 크게 축소**돼 노출된다. 축소 상태에서 제목 글자와 핵심 인물이 식별돼야 한다.
- 작은 크기에서 뭉개지는 얇은 획·과밀한 요소는 피한다. (구체 최소 크기는 [미확인].)
- 표지 타이틀 서체 지정은 `story-webtoon-lettering`의 `references/korean-lettering.md` 참조.
