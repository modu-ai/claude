## 카테고리 A — AI 규제

### 1. 생성형AI 사용 사실 미고지 / AI 생성물 미표시

- **근거**: 인공지능기본법(인공지능 발전·신뢰 기반 조성법) 제31조, 제43조
- **2026 변화**: ✅ **2026.1.22 시행** (세계 최초 AI 종합법)
  - 사전 고지 의무: AI 활용 사실을 이용자에게 사전 고지
  - AI 생성물 표시 의무: 텍스트·이미지·영상·음성 AI 생성물에 표시(워터마크 등)
  - 딥페이크 고지·표시 의무
- **의무 주체**: 인공지능사업자(개발·이용). 단순 '이용자'(도구로만 사용)는 면제 — 판별은 `ai-act-user-vs-provider.md` 참조
- **제재**: 3천만원 이하 과태료
- **체크**: [ ] AI 사용 사실을 이용자에게 사전 고지했는가? [ ] AI 생성물에 표시를 했는가?
- **위반 시 액션**: 약관·서비스 화면에 AI 사용 고지 추가, AI 생성물에 워터마크·메타데이터 표시

## 카테고리 B — 개인정보

### 2. 앱 접근권한 필수/선택 미구분

- **근거**: 정보통신망법 제22조의2, 제76조
- **핵심**: 필수적 접근권한과 선택적 접근권한을 구분하여 고지. 선택권한 거부 시 서비스 거부 금지
- **제재**: 3천만원 이하 과태료
- **체크**: [ ] 접근권한을 필수/선택으로 구분했는가? [ ] 각 권한의 필요 항목·사유를 표시했는가? [ ] 선택권한 거부해도 서비스 이용 가능한가?
- **위반 시 액션**: 권한을 본질적 필수/선택으로 재분류, 안드로이드 manifest·iOS Info.plist 수정

### 3. 개인정보 국외이전 고지 누락 (해외리전)

- **근거**: 개인정보보호법 제28조의8
- **2026 변화**: ✅ **2026.2.12 국회 통과** — 과징금 산정기준 구체화
- **해당**: Supabase·Firebase·AWS 등 해외 리전 클라우드 사용 시
- **제재**: 관련 매출의 100분의 3 이하 과징금
- **체크**: [ ] 해외 클라우드를 사용하는가? [ ] 개인정보처리방침에 국외이전(국가·제공받는 자·이용목적)을 고지했는가?
- **위반 시 액션**: 개인정보처리방침에 국외이전 항목 추가, 또는 국내 리전으로 이전

### 4. 개인정보처리방침에 CPO 미공개

- **근거**: 개인정보보호법 제31조, 제75조
- **2026 변화**: ✅ CPO 책임 명문화, 지정·변경 시 위원회 신고 의무, CEO 최종 책임자 명시
- **제재**: 1천만원 이하 과태료
- **체크**: [ ] 개인정보처리방침에 개인정보보호책임자(CPO) 성명·연락처를 공개했는가?
- **위반 시 액션**: CPO 지정 후 처리방침에 공개, 위원회 신고

### 7. 처리위탁 고지 누락

- **근거**: 개인정보보호법 제26조, 제75조
- **2026 변화**: ✅ 수탁자 관리·감독 책임 강화
- **해당**: PG사·문자발송업체·클라우드 등에 개인정보 처리를 위탁 시
- **제재**: 1천만원 이하 과태료
- **체크**: [ ] 개인정보처리방침에 처리위탁(수탁자·위탁 업무)을 고지했는가?
- **위반 시 액션**: 처리방침에 위탁 목록 추가, 수탁자 관리·감독 체계 구축

### 14. 개인정보 침해사고 72시간 통지 체계 미비

- **근거**: 개인정보보호법 (2026.9.11 개정 시행)
- **2026 변화**: ✅ 침해사고 발생 시 72시간 이내 정보주체 통지 의무 신설. 고의·중과실 대규모 유출 시 매출 최대 10%
- **제재**: 매출 최대 10% 과징금 (고의·중과실 대규모 유출)
- **체크**: [ ] 침해사고 발생 시 72시간 통지 체계가 구축되어 있는가? [ ] 유출 가능성 인지 시점부터 통지 절차가 있는가?
- **위반 시 액션**: 침해사고 대응 절차(SOP) 수립, 모니터링·알림 시스템 구축

## 카테고리 C — 전자상거래

### 5. 사업자 정보 미표시

- **근거**: 전자상거래법 제10조, 제45조
- **2026 변화**: ✅ **2026.7.21 시행 개정** (다크패턴 규제 강화)
- **필수 표시**: 상호·대표자 성명·주소·전화번호·사업자등록번호·이용약관
- **제재**: 500만원 이하 과태료
- **체크**: [ ] 앱/웹 하단에 사업자 정보 6종을 모두 표시했는가? [ ] 통신판매업 신고번호 표시했는가?
- **위반 시 액션**: 푸터에 사업자 정보 전체 표시, 통신판매업 신고 후 번호 표시

### 8. 무료체험 자동전환 / 구독료 인상 사전동의 누락

- **근거**: 전자상거래법 제13조 제6항
- **2026 변화**: ✅ 다크패턴 규제 — 유료전환·대금증가 명확 고지 의무 (OTT 시정명령 사례)
- **핵심**: 무료체험 후 유료 자동전환 또는 구독료 인상 시 30일 이내 별도 사전 동의 필요
- **제재**: 500만원 이하 과태료
- **체크**: [ ] 무료체험→유료전환 시 사전 고지·별도 동의를 받았는가? [ ] 구독료 인상 시 사전 동의를 받았는가? [ ] 해지를 어렵게 하는 다크패턴이 있는가?
- **위반 시 액션**: 자동결제 전 명확한 사전 고지, 별도 동의 체크박스, 간편 해지 경로 제공

### 9. 통신판매업 신고 누락

- **근거**: 전자상거래법 제12조
- **해당**: 앱/웹에서 유료로 상품·서비스 판매(인앱결제·구독·콘텐츠) 시 — 시·군·구청 신고 필수
- **예외**: 연 50회 미만 거래 등 일부 예외
- **제재**: 시정명령 + 과태료
- **체크**: [ ] 유료 판매를 하는가? [ ] 통신판매업 신고를 했는가? [ ] PG사 사용 시 구매안전서비스 이용확인증 제출했는가?
- **위반 시 액션**: 정부24(온라인) 또는 관할 구청에 통신판매업 신고

### 13. 인앱결제 외부결제 차단

- **근거**: 전자상거래법 (2024 개정)
- **핵심**: 앱 마켓이 외부결제를 차단하면 매출 2% 과징금
- **제재**: 매출의 100분의 2 과징금
- **체크**: [ ] (앱 마켓 사업자인 경우) 외부결제를 허용하고 있는가?
- **위반 시 액션**: 외부결제 수단 허용

## 카테고리 D — 정보통신망

### 6. 광고 "(광고)" 미표기 / 수신거부 미기재

- **근거**: 정보통신망법 제50조 제4항, 제76조
- **2026 변화**: ✅ 스팸가이드 7차 개정 — 불법스팸에 매출액 6% 과징금 신설
- **핵심**: 광고성 정보(문자·메일·앱푸시) 전송 시 사전 명시적 동의, "(광고)" 표기, 수신거부 방법 기재. 수신동의일 기준 2년 이내 수신거부 방법 안내
- **제재**: 3천만원 이하 과태료 (+불법스팸 시 매출 6% 과징금)
- **체크**: [ ] 광고성 정보에 "(광고)"를 표기했는가? [ ] 수신거부 방법을 기재했는가? [ ] 사전 명시적 동의를 받았는가?
- **위반 시 액션**: 발송 전 "(광고)"·수신거부 문구 추가, 사전 동의 절차 구축

## 카테고리 E — 기타 필수

### 10. 자체 결제시스템 시 전자금융업 검토 누락

- **근거**: 전자금융거래법
- **핵심**: 결제·충전·송금을 자체 구축 시 전자금융업(선불전자지급수단·종합지급결제업) 해당 검토. PG사 연동 시 PG사가 부담
- **제재**: 영업정지 등 (해당 시)
- **체크**: [ ] 자체 결제/충전/송금 시스템이 있는가? [ ] 전자금융업 해당 여부를 검토했는가?
- **위반 시 액션**: 자체 구축 시 금융감독원 자문, 또는 PG사 연동으로 전환

### 11. 청소년유해매체물 등급분류 누락

- **근거**: 청소년보호법
- **핵심**: 서비스 콘텐츠가 유해매체물에 해당하면 등급표시·인증 필요. 앱 마켓 등급(전체/12/15/18세) 표기
- **제재**: 과태료
- **체크**: [ ] 콘텐츠에 선정·폭력 요소가 있는가? [ ] 앱 마켓 등급을 표기했는가?
- **위반 시 액션**: 영상물등급위원회 심의, 앱 마켓 등급 설정

### 12. AI 생성물 저작권 표시 누락

- **근거**: 저작권법
- **2026 동향**: 순수 AI 생성물은 한국에서 저작권 미인정. 인간의 창의적 기여 입증 필요. 학습데이터 출처 공개 입법 논의 중
- **제재**: 손해배상 (타인 저작물 침해 시)
- **체크**: [ ] AI 생성물을 상업적으로 사용하는가? [ ] 학습에 타인 저작물이 사용되었는가? [ ] AI 생성물임을 명시했는가? (1번과 연동)
- **위반 시 액션**: AI 생성물 표시, 학습 데이터 출처 확인, 인간 창작 기여 입증 자료 보관
