## 2. 선택 판단 축

플랫폼 선택은 아래 순서로 좁힌다.

1. **장르·독자 지향** — 남성향 현판/무협 신규 → 문피아·노벨피아 계열, 여성향 로판/BL → 리디·카카오페이지 계열.
2. **결제 구조 적합** — 편당 결제(문피아·노벨피아)는 회차 밀도·절단이 즉시 수익. 기다무(카카오페이지·네이버시리즈)는 연참 리듬과 대기 해제 지점 설계가 핵심.
3. **독점 감내 여부** — 독점 편성 플랫폼(카카오페이지·네이버시리즈·리디)은 노출·프로모션이 크지만 계약이 배타적. 비독점 친화(노벨피아·조아라)는 여러 곳 동시 연재 가능.

## 3. 확인처 (수치 재확인 경로)

각 `[미확인]` 항목은 사용 전 아래에서 확인한다. URL은 정책 개편으로 변경될 수 있으므로 플랫폼명 + "작가 센터/투고 가이드"로 재검색해 최신본을 연다.

- 문피아 작가 센터 / 노벨피아 작가 가이드 / 조아라 창작 지원
- 카카오페이지·네이버시리즈: CP(콘텐츠 제공사)·편성 담당 문의 (개인 직접 투고 경로 제한적)
- 리디: 작가/출판 파트너 문의
