# 비주얼 시트 표 — 인물 8항목 / 동물 8항목 + 2계층 예산

> `story-character-sheet` Step 2에서 로드한다. 캐릭터 외형을 이미지 생성에 바로 쓸 수 있는 수준으로 고정하는 표. 스토리 본문에는 시각 정보를 나열하지 않고 이 시트에만 모은다(비주얼-서사 분리).

## 인물 비주얼 시트 (8항목)

| 항목 | 내용 |
|------|------|
| 성별·나이대 | |
| 헤어 | 색 · 길이 · 스타일 |
| 얼굴·눈 | 눈매·눈색, 인상, 특징(주근깨·보조개·점 등) |
| 체형·키 | |
| 피부톤 | |
| 기본 복장 | 상의·하의·신발, **색까지** (평상시 기본 룩) |
| 상징 소품·특징 | 안경·모자·액세서리·흉터 등 한눈에 구별되는 것 |
| 전체 인상 한 줄 | 이 인물을 한 문장으로 |

## 레귤러 / 게스트 2계층 예산

캐릭터를 두 계층으로 나눠 시트 상세도와 관리 비용을 통제한다.

| 계층 | 정의 | 시트 상세도 | 보관 위치 |
|------|------|-------------|-----------|
| **레귤러** | 매 화 또는 자주 등장, 독자가 애착을 갖는 인물. **최대 5명 이내** | 위 8항목 전체 + 일관성 앵커 | 전체 기획서(`story-series-bible`) 캐릭터 절 |
| **게스트** | 특정 화에만 등장. 여러 화 일관성 부담 없음 | 그 화에서 그릴 수 있을 만큼의 외형만 간략히 | 그 화의 에피소드 원고 비주얼 항목 |

### 예산 원칙

- **레귤러 5명 상한.** 인원이 늘수록 화마다 일관성 유지 부담과 (웹툰화 시) 생성 크레딧이 함께 는다. 초과분은 게스트로 돌린다.
- **레귤러는 주인공과 같은 수준의 8항목 시트**로 설계한다 — 화마다 같은 모습으로 그리는 근거.
- **게스트는 이름·역할만 전체 기획서에**, 외형 상세는 등장하는 그 화의 원고에만 둔다.
