# 콘텐츠 발행 캘린더 (Content Calendar)

## 개요

마케팅·소셜·콘텐츠 팀이 체계적인 편집 계획과 발행 스케줄을 수립하여
모든 채널에서 일관된 오디언스 소통을 유지하도록 돕는 스킬입니다.

지원 영역:
- 콘텐츠 전략 (콘텐츠 필러, 캠페인 테마, 성공 지표)
- 편집 캘린더 (월간/주간/분기 발행 일정)
- 채널별 게시 주기 (블로그, 소셜, 이메일, 영상, 팟캐스트)
- 캠페인·계절 콘텐츠 계획
- 크로스 플랫폼 배포 조율

## 트리거 키워드

콘텐츠 캘린더, 발행 일정, 게시 주기, 편집 캘린더, 콘텐츠 계획표, 콘텐츠 스케줄, 캠페인 캘린더

## 워크플로우

### 1단계: 맥락 수집

1. **콘텐츠 목표 정의**: 콘텐츠가 달성해야 할 것 명확히 (인식, 교육, 리드 생성, 유지, 옹호)
2. **오디언스 조사**: 인구통계, 통증점, 정보 출처, 선호 콘텐츠 형식 기록
3. **경쟁사 콘텐츠 감시**: 경쟁사 전략, 갭, 차별화 기회 검토
4. **가용 리소스 파악**: 콘텐츠 제작 역량 목록 (사내팀, 프리랜서, 에이전시)
5. **플랫폼 분석**: 타겟 오디언스 활동 채널, 알고리즘 선호도, 참여 패턴 기록

### 2단계: 전략 및 캘린더 작성

**콘텐츠 필러 정의**
- 3-5개 핵심 콘텐츠 테마 수립 (교육, 영감, 제품, 커뮤니티, 엔터테인먼트)
- 모든 콘텐츠가 핵심 브랜드 메시지와 가치 제안을 강화하는지 확인

**편집 캘린더 개발**
- 3-6개월 선행 콘텐츠 배포 캘린더, 주요 날짜·휴일·캠페인 마일스톤 매핑
- 각 콘텐츠의 발행 채널과 시간 명시
- 주제별 관련 콘텐츠 그룹화(배칭) 및 누적 영향을 위한 순서 지정
- 계절/트렌드 주제와 신속 대응 기회용 버퍼 콘텐츠 유지

**채널별 게시 주기 예시**
```
유튜브: 주 1~2회 (알고리즘 안정화)
블로그: 주 2회 (SEO 타겟 — 트렌드 큐레이션 + How-to)
SNS: 주 3~5회 (영상 클립, 인사이트, 커뮤니티 참여)
이메일: 주 1회 또는 격주 (뉴스레터)
팟캐스트: 격주 또는 월 2회
```

### 3단계: 검수 및 배포

1. **이해관계자 정렬**: 리더십·제품·고객 성공 팀과 캘린더 공유
2. **리소스 검증**: 팀 역량이 계획 볼륨을 전달 가능한지 확인, 필요시 범위 조정
3. **비상 계획**: 트렌딩 주제·신속 대응용 백업 콘텐츠 옵션 수립
4. **홍보 조율**: 발행 스케줄과 콘텐츠 홍보(유료·이메일·소셜 공유) 정렬
5. **모니터링 설정**: 성과 추적·최적화·캘린더 업데이트 프로세스 수립

> 상세 전문가 역할·워크플로우·에러 처리: `references/content-calendar.md`

## 사용 예시

- "3개월 콘텐츠 캘린더 짜줘"
- "유튜브 주 1회 + 블로그 주 2회 발행 일정 만들어줘"
- "제품 출시 캠페인 콘텐츠 스케줄 설계해줘"
- "채널별 게시 주기 정해줘"

## 출력 형식

### 월간 콘텐츠 캘린더

```markdown
## [X월] 콘텐츠 캘린더

| 요일 | 플랫폼 | 콘텐츠 유형 | 주제 | 담당자 | 상태 |
|------|--------|-----------|------|--------|------|
| 월 | 블로그 | 트렌드 큐레이션 | [주제] | [담당] | [ ] |
| 화 | 블로그 | How-to 가이드 | [주제] | [담당] | [ ] |
| 목 | 유튜브 | 영상 업로드 | [주제] | [담당] | [ ] |
| 금 | SNS | 영상 클립 | [주제] | [담당] | [ ] |

월말: 다음 달 캘린더 계획 + 성과 분석
```

### 콘텐츠 전략 프레임워크

```markdown
## 콘텐츠 전략
- 콘텐츠 목표: [인식/교육/리드/유지]
- 콘텐츠 필러: [필러 1] / [필러 2] / [필러 3]
- 성공 지표: [참여율/도달률/전환율 목표]
- 캠페인 테마: [분기별 캠페인]
```

## 주의사항

- **플랫폼 알고리즘 변경**: 각 플랫폼의 권장 발행 주기·알고리즘은 수시로 변경됩니다. 최신 정보는 플랫폼 공식 Creator 채널에서 확인하세요.
- **리소스 과대 계획**: 팀 역량을 넘는 발행 빈도는 콘텐츠 품질 저하로 이어집니다. 지속 가능한 주기로 시작해 점진적으로 확대하세요.
- **표 vs 산문**: 캘린더 표는 데이터이므로 그대로 사용하되, 전략·캠페인 설명 문장은 AI 슬롭 검수 후 사용하세요.
