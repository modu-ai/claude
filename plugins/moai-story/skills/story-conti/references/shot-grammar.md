# 화각·앵글·전환 어휘

> 콘티 프레임을 지정하는 촬영 어휘 사전이다. 프레임마다 화각·앵글·전환을 골라 붙인다. **컷 수는 감정 비트가 정한다** — 아래 어휘는 감정을 화면에 싣는 도구이지 채워야 할 칸이 아니다.

## 1. 화각 (숏 사이즈)

| 약어 | 이름 | 담는 것 | 감정 용도 |
|------|------|---------|-----------|
| ELS | 익스트림 롱숏 | 공간 전체·인물 점 | 고립·규모·상황 제시 |
| LS | 롱숏(풀숏) | 인물 전신 + 배경 | 장소·전신 동작 |
| MS | 미디엄숏 | 허리 위 | 대화 기본, 관계 |
| MCU | 미디엄 클로즈업 | 가슴 위 | 대사 강조 |
| CU | 클로즈업 | 얼굴 | 감정 정점 |
| ECU | 익스트림 클로즈업 | 눈·손·소품 | 긴장·디테일 강조 |
| OTS | 오버 더 숄더 | 어깨 너머 상대 | 대화 시선 교환 |
| POV | 시점 숏 | 인물이 보는 것 | 몰입·주관 |

## 2. 앵글

| 앵글 | 효과 |
|------|------|
| 아이레벨 | 중립·객관 |
| 하이앵글(부감) | 대상 왜소·취약 |
| 로우앵글(앙각) | 대상 위압·강함 |
| 더치(기울임) | 불안·혼란 |
| 버즈아이(수직 부감) | 상황 조망·비현실 |

## 3. 전환

| 전환 | 뜻 | 언제 |
|------|-----|------|
| CUT | 즉시 전환 | 기본, 속도감 |
| DISSOLVE | 겹쳐 넘김 | 시간 경과·회상 |
| FADE IN/OUT | 암전 진입/퇴장 | 장/막 경계 |
| MATCH CUT | 형태·동작 일치로 넘김 | 은유·연결 |
| SMASH CUT | 급격한 대비 전환 | 충격·리듬 파열 |
| WIPE | 밀어내기 | 장면 이동(스타일리시) |

## 4. 프레임 시퀀스 설계 원칙

- **화각으로 감정을 실어라**: 상황 제시(LS) → 관계(MS) → 감정 정점(CU) → 여운(ELS) 같은 리듬으로 화각을 변주한다.
- **180도 선(이매지너리 라인)**: 두 인물의 시선 방향을 일관되게 — 어겨야 할 이유가 없으면 넘지 않는다.
- **컷 수는 감정 비트**: 정적인 대화 씬을 8컷으로 쪼갤 필요 없고, 격정 장면을 2컷으로 뭉갤 필요 없다.
- **톤 통일**: 색온도·밝기·렌즈 느낌을 시퀀스 전체에 걸쳐 프롬프트에 반복 명시.
