# 광고 콘티 프리셋

> 광고 콘티는 서사 콘티와 산출물(정지 프레임 보드)은 같지만 **목적이 다르다** — 짧은 시간 안에 제품/브랜드 메시지를 각인하고 행동(CTA)을 유도한다. 이 프리셋은 구 story-ad-conti의 광고 지식을 이관한 것이다.

## 1. 시간 단위 구성

| 길이 | 통상 컷 수 | 골격 |
|------|-----------|------|
| **15초** | 6~8컷 | 후크(3초) → 문제 → 해결(제품) → CTA |
| **30초** | 10~14컷 | 후크 → 문제 → 빌드업 → 해결(제품 USP) → 감정 → CTA |

> 컷 수는 목표이지 규칙이 아니다 — 메시지가 몇 컷을 요구하는지가 우선.

## 2. 첫 3초 후크 (최우선)

디지털 광고는 첫 3초에 스킵 여부가 갈린다. 후크가 없으면 이후 컷은 노출되지 않는다.

- 후크 유형: 의외성 / 질문 던지기 / 시각적 임팩트 / 공감 상황.
- 브랜드·제품은 후크 뒤에 배치해도 되지만, 브랜드 인지가 목적이면 초반 노출.

## 3. TVC vs 디지털

| 축 | TVC(방송) | 디지털(유튜브·SNS) |
|----|-----------|--------------------|
| 길이 | 15s/30s 정형 | 6s~가변, 세로/정사각 포함 |
| 화면비 | 16:9 | 9:16·1:1·16:9 혼재 |
| 후크 | 브랜드 무드 빌드업 여지 | 첫 3초 즉시 후크 |
| 사운드 | 사운드 온 전제 | 무음 재생 전제(자막·시각 우선) |

## 4. 브랜드 가이드 준수

- **색·로고·폰트**: 브랜드 가이드라인 값을 프레임 프롬프트에 명시. 임의 색 사용 금지.
- **로고 노출 규칙**: 최소 노출 크기·세이프 영역·클리어 스페이스 준수(브랜드 가이드 확인).
- **제품 정확성**: 제품 외형·패키지는 실제와 일치해야 한다 — 지어낸 디테일 금지.

## 5. 광고주 제안 형식

광고주 승인을 받는 문서이므로, 콘티 보드에 다음을 함께 담는다.

- **캠페인 목표·타깃·핵심 메시지(1문장)**.
- **컷별 화각 + 화면 설명 + 카피/나레이션 + 초 단위 타이밍**.
- **CTA 문구·랜딩 경로**.
- (필요 시) 대안 후크 2안 — 광고주 선택지 제공.

## 6. 안티패턴

- 후크 없이 브랜드 로고부터 → 스킵.
- 무음 재생 무시(자막 없이 대사 의존) → 디지털에서 메시지 손실.
- 브랜드 색·로고 임의 변형 → 광고주 반려.
