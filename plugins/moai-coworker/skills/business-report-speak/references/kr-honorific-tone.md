# 존댓말·톤 규격 (kr-honorific-tone)

> moai-coworker | business-report-speak 소유 공유 규격
> 이메일·보고·CS 응대·협상·공지 등 한국어 대인 커뮤니케이션 산출물을 만드는 스킬이 이 파일 하나를 참조합니다.
> 목적: 수신자 위계에 맞는 종결어미·서명·화법을 일관되게 적용해 "AI가 쓴 어색한 존댓말"을 방지합니다.

## 1. 6단계 수신자 위계 × 종결어미

| 수신자 | 기본 문체 | 종결어미 예 | 비고 |
|--------|-----------|-------------|------|
| 대표(CEO)·사외 최고위 | 하십시오체 | ~드립니다, ~하시기 바랍니다, ~보고드립니다 | 압존법은 현대 비즈니스에서 강제하지 않음 |
| 임원 | 하십시오체 | ~드립니다, ~검토 부탁드립니다 | 두괄식 + 간결 우선 |
| 팀장·직속 상사 | 하십시오체 기본, 사내 문화에 따라 해요체 혼용 | ~드립니다 / ~해요, ~할게요 | 문서·공식 보고는 하십시오체 고정 |
| 동료(수평) | 해요체 | ~해요, ~부탁드려요 | 반말 전환은 상호 합의 후에만 |
| 거래처·파트너 | 하십시오체 | ~드립니다, ~요청드립니다 | 친분이 있어도 문서는 하십시오체 유지 |
| 고객 | 하십시오체 + 공감 표현 | ~도와드리겠습니다, ~안내드립니다 | 사과·해결책은 구체적으로 |

원칙: **한 문서 안에서 문체를 섞지 않습니다** (해요체·하십시오체 혼용은 대표적 AI 티). 사내 채팅은 해요체, 공식 문서·사외 발신은 하십시오체가 기본값입니다.

## 2. 서명 (드림 / 올림 / 배상)

| 표기 | 쓰임 |
|------|------|
| **올림** | 손윗사람·격식 최상 (대표·임원·고객 대상 공식 문서) — "홍길동 올림" |
| **드림** | 일반 비즈니스 표준 (동급~손윗사람, 거래처) — "홍길동 드림" |
| **배상(拜上)** | 전통 격식 서한·경조사 답례 등 매우 격식적 문맥. 일반 이메일에는 과함 |

동료 간 캐주얼 메일은 "감사합니다. 홍길동"으로 충분합니다. 자신을 높이는 "홍길동 님 드림" 같은 표기는 금지.

## 3. 상황별 화법

### 요청
- 쿠션어 + 구체 요청 + 기한 + 선택권: "바쁘신 줄 알지만, ○○ 자료를 금요일 오전까지 공유해 주실 수 있을까요?"
- 금지: 명령형("공유 바랍니다."만 단독 사용 — 상급자 발신 외), 기한 없는 "빠른 회신 부탁드립니다" 남발

### 거절
- 공감 → 사유(상황 탓, 한 줄) → 대안 순서: "말씀 주신 취지는 충분히 이해합니다. 다만 현재 일정상 이번 주 진행은 어렵습니다. 다음 주 화요일 이후라면 가능합니다."
- 금지: 사과만 반복하고 대안 없는 거절, 애매하게 여지를 남기는 거절

### 사과
- 인정 → 원인 → 조치 → 재발 방지 순서: "혼선을 드려 죄송합니다. ○○ 확인이 늦어 발생한 문제이며, 금일 중 수정본을 전달드리겠습니다."
- 금지: 조건부 사과("불편하셨다면 죄송합니다"), 책임 희석("착오가 있었던 것 같습니다")

### 독촉
- 1차: 확인 요청 프레임 — "지난주 요청드린 ○○ 건 진행 상황을 여쭙습니다."
- 2차: 기한·영향 명시 — "○일까지 회신이 없으면 △△ 일정에 영향이 있어 다시 연락드립니다."
- 금지: 감정 표현("아직도 안 주셨네요"), 수신자 비난

## 4. 사내 vs 사외 톤

| 구분 | 사내 | 사외 (거래처·고객) |
|------|------|--------------------|
| 문체 | 문화에 따라 해요체 허용 | 하십시오체 고정 |
| 호칭 | 직함+님 (팀장님, ○○님) | 회사명+직함+성함 (△△사 김 부장님) |
| 자사 지칭 | 우리 팀/저희 부서 | **저희** 회사 ("우리 회사" 금지) |
| 상급자 언급 | 직함 그대로 | 사외에는 자사 상급자를 낮춰 지칭 가능 ("김 대표께 전달하겠습니다" — 과도한 압존 불필요) |
| 이모지·구어 | 채팅 한정 허용 | 원칙 금지 |

## 5. 적용 절차

1. 산출물 작성 전 수신자를 6단계 중 하나로 판별합니다 (불명확하면 사용자에게 확인).
2. 해당 행의 문체·종결어미·서명을 적용하고, 요청·거절·사과·독촉이 포함되면 3절 화법 구조를 따릅니다.
3. 완성 후 점검: 문체 혼용 없음 / 서명 적절 / 쿠션어 과잉 없음(문단당 1회 이내) / 사외 문서에 "저희" 사용.
