Quick Start

mono 스텝 라벨(코랄), 세리프 타이틀, 본문. 3단계 시작 가이드 카드.

Step 01

마켓플레이스 추가

Cowork에서 cowork-plugins 마켓플레이스를 등록합니다.

$ /plugin marketplace add
  cowork-plugins
Step 02

플러그인 설치

필요한 플러그인을 선택해 설치합니다 — 28개 중 골라 담기.

$ /plugin install
  moai-content
Step 03

워크플로우 실행

자연어로 스킬을 호출하면 체인이 자동으로 연결됩니다.

> 블로그 초안을
  작성해줘

5분이면 충분합니다

설치부터 첫 산출물까지. 빌드 과정 없이 바로 시작하세요.
