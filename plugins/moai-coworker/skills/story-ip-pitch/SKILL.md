# story-ip-pitch

> 완성된 작품을 2차 저작 시장에 피칭하는 문서를 만드는 스킬. 원작 가치·확장성·제안 조건을 한 제안서로 정리한다.

## 1. 개요

IP 피칭은 **무엇을** 제안할지(2차 저작 형태)와 **언제** 누구에게 접근할지(제작사·플랫폼·에이전시)를 설계한다. 원작 데이터(조회수·완독률·팬덤)가 협상력의 기준이다.

## 2. 2차 저작 확장 경로

| 원작 → 확장 | 비고 |
|------------|------|
| 웹소설 → 웹툰 | 인물 시트·콘티 선행 |
| 웹툰 → 드라마/영화 | 시놉·스크린플레이 |
| 웹툰 → 게임 | 세계관·캐릭터 라이선스 |
| 출판 → 드라마 | 단행본 실적·인지도 |
| IP → 굿즈 | 캐릭터 라이선스 |

## 3. 워크플로우

### Step 1: IP 현황 정리
원작 형태·실적(조회수·부수·팬덤 규모)·핵심 자산(캐릭터·세계관).

### Step 2: 확장 후보 선정
2차 저작 경로별 적합도·난이도·예상 소요를 평가.

### Step 3: 제안서 작성
- IP 요약(로그라인·핵심 자산)
- 확안 시나리오(2차 저작 형태·타깃 제작사)
- 협상 조건(판권 범위·기간·로열티)

### Step 4: 타깃 매핑
제작사·플랫폼·에이전시 후보를 우선순위화.

## 4. 출력 형식

IP 피칭 제안서(마크다운) + 타깃 목록.
