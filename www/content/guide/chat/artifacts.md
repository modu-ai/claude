# 아티팩트 (Artifacts)

**아티팩트**는 Claude가 만드는 공유 가능한 산출물 — 앱·문서·코드·대시보드·시각화 — 을 대화 옆 전용 패널에서 보여주는 기능입니다.

## 아티팩트 패널

Claude가 아티팩트를 만들면 대화 오른쪽에 전용 패널이 열립니다. 거기서 미리보기·다운로드·복사·공유가 가능합니다.

## 만들 수 있는 것

- **인터랙티브 앱** — React/HTML로 동작하는 작은 앱
- **문서** — 마크다운 보고서·초안
- **코드 스니펫** — 실행 가능한 코드
- **시각화** — SVG·Mermaid 다이어그램, 차트/대시보드

## 활용 예시

- 퀴즈/설문 인터랙티브 앱
- 이력서 초안
- 데이터 시각화 대시보드

## 게시하고 공유하기

아티팩트를 게시(publish)하면 링크가 생성되어 다른 사람과 공유할 수 있습니다.
