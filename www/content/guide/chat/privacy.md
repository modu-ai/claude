# 개인정보와 데이터

대화 데이터를 직접 통제하는 방법입니다.

## 대화 삭제

- 개별 대화: 대화 메뉴에서 삭제
- 전체 대화: 설정에서 일괄 삭제

## 데이터 보관 기간

플랜에 따라 정해진 기간 동안 보관됩니다. 자세한 기준은 Privacy Center와 개인정보처리방침을 참고하세요.

## 모델 학습 동의/거부

기본적으로 대화 내용은 모델 학습에 쓰이지 않습니다. **Settings → Privacy**에서 "학습 동의" 토글로 명시적으로 관리할 수 있습니다.

## 데이터 내보내기·청구

본인 데이터를 내보내거나 삭제를 청구(DSAR)할 수 있습니다. 요청 후 정해진 기일 내 응답합니다.

## 민감한 데이터

주민번호·비밀번호·금융정보 등은 입력을 피하세요.
