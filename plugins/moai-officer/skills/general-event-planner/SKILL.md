## 실행 규칙

1. 사용자 요청 수신 → 해당 전략 가이드 판별
2. `references/{id}.md` 로드 → 전략 가이드에 따라 실행
3. `--deepthink` 또는 복잡 의사결정 → sequential-thinking MCP가 설치돼 있으면 `mcp__sequential-thinking__sequentialthinking`를 활용, 없으면 `ultrathink` 키워드 기반 심층 추론 또는 일반 단계 추론으로 대체
4. 결과물 생성 후 사용자 검토 요청

## 사용 예시

- "100명 규모 사내 워크샵 기획서를 작성해줘"
- "6개월 후 결혼 준비 타임라인과 체크리스트를 만들어줘"
- "스드메 업체 선정 기준과 비교 방법을 알려줘"
- "B2B 콘퍼런스 진행 순서와 MC 스크립트를 짜줘"
- "스몰웨딩 예산 2,000만 원으로 계획을 세워줘"

## 독립 실행 워크플로우

전략 가이드 파일 없이도 다음 단계로 실행 가능합니다:

1. **행사 유형 파악**: 기업 행사(세미나/워크샵/컨퍼런스) vs 개인 이벤트(웨딩/파티/기념식) 구분
2. **핵심 조건 수집**: 예산, 일정, 참석 인원, 장소 조건, 목적 확인
3. **기획서 구조 작성**: 행사 목적 → 일정표 → 예산 배분 → 담당 역할 → 준비 체크리스트
4. **위험 요소 점검**: 날씨·장소·장비·참석자 관련 비상 계획 포함
5. **결과물 정리**: 기획서, 타임라인, 진행 순서표, 예산 시트 제공

## 2026 한국 웨딩 트렌드 컨텍스트

결혼비용 실태 조사(듀오 2025), 정부 결혼서비스 발전 지원방안(2024.11.) 반영:

### 2026년 평균 결혼 비용

| 항목 | 평균 비용 | 비고 |
|------|----------|------|
| 총 결혼 비용 | 5,000-8,000만 원 | 신혼집 제외 |
| 예식장+식대 | 1,790만 원 평균 | 웨딩홀 600-1,500만 원 |
| 스드메 | 400-500만 원 | 스튜디오+드레스+메이크업 |
| 스몰웨딩 (50명) | 500-1,000만 원 | 일반 대비 1/3 수준 |

### 주요 웨딩 트렌드

- **스몰웨딩 확산**: 예비부부 34%가 스몰웨딩 고려. 하우스웨딩·호텔웨딩 다양화
- **비용 균등 분담**: 양가 50:50 분담 선택 커플 42% (2024 기준)
- **스드메 가격 공시 추진**: 정부 '결혼서비스 발전 지원방안'(2024.11)에 따라 참가격(price.go.kr)에서 일부 업체 가격을 비교할 수 있음 (법적 강제 아님)
- **표준계약서 도입(권장)**: 공정위 결혼·예식 표준약관 사용 권장으로 계약 조건 투명성 제고 (법적 사용 의무는 아님)

### 비용 절약 팁

- 비수기(1-2월, 7-8월): 예식장 최대 30% 할인
- 평일 예식: 주말 대비 10-20% 할인
- 패키지 계약: 스드메+예식장 패키지로 20-30% 절약
- 가격 비교: 한국소비자원 참가격(www.price.go.kr) 활용

## 문제 해결

| 상황 | 해결 방법 |
|------|-----------|
| 예산 정보 미제공 | 총 예산 범위나 인원수만 알려주시면 배분 계획을 잡아드립니다 |
| 일정 조건 부족 | 예식 희망 시기(분기 수준)만 알려주셔도 D-6개월 타임라인을 작성합니다 |
| 장소 미결정 | 지역·규모·예산 조건에 맞는 장소 유형을 안내해 드립니다 |
| 기업 행사 사내 결재 필요 | 기획서 + 예산 승인 품의서 양식을 함께 작성해 드립니다 |

이 스킬은 기획서·진행 순서표·MC 스크립트·품의서 같은 **글(텍스트) 산출물**을 만듭니다. 글 산출물은 발행 전 다음 체인으로 후처리하세요.

```

```

- **moai-coworker:general-ai-slop-reviewer**: AI 티 나는 표현·과장 어조를 검수하고 다듬습니다
