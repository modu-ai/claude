카페에서 음료를 고를 때 가볍게 한 잔만 마실지, 매일 들러 정기권을 끊을지 고민하는 것처럼, Claude도 사용 습관에 맞춰 요금제를 고를 수 있습니다. 어쩌다 한 번 쓰는 분은 무료로 시작하고, 매일 업무에 활용하는 분은 더 넉넉한 용량의 유료 요금제를 선택하면 됩니다.

이 글에서는 어떤 요금제가 나에게 맞는지, 구독을 선물하거나 선물받은 구독을 어떻게 사용하는지, 결제할 때 세금은 어떻게 계산되는지, 그리고 용량이 부족할 때 추가로 채워 넣는 방법까지 차근차근 정리했습니다.

## 어떤 요금제를 선택할까

Claude는 개인 사용자를 위해 네 가지 요금제를 제공합니다. 월간 또는 연간 결제 중에서 고를 수 있고, 요금제마다 사용할 수 있는 용량이 다릅니다.

| 요금제 | 가격 | 누구에게 어울리나 |
|---|---|---|
| **Free** | $0 | 가끔 가볍게 써보는 분 (제한된 용량) |
| **Pro** | 월 $20 또는 연 $200 | 평소에 꾸준히 사용하는 분 (표준 용량) |
| **Max 5x** | 월 $100 | 자주 사용하는 분 (세션당 Pro의 5배 용량) |
| **Max 20x** | 월 $200 | 매일, 협업까지 활발히 하는 분 (세션당 Pro의 20배 용량) |

Max 요금제의 "5x", "20x"는 Pro를 기준으로 세션당 용량이 몇 배인지를 나타냅니다. 즉 Max 5x는 한 세션에서 Pro보다 5배, Max 20x는 20배 더 많은 작업을 처리할 수 있다는 의미입니다.

{{< hint type="note" >}}
이 네 가지는 모두 개인 사용자를 위한 요금제입니다. 회사나 팀 단위로 함께 쓰려면 Team 또는 Enterprise 요금제가 따로 마련되어 있습니다.
{{< /hint >}}

처음이라면 Free로 가볍게 시작해 보고, 용량이 자주 부족하다고 느껴질 때 Pro나 Max로 올리는 방식을 추천합니다.

## 구독을 선물하기

Claude Pro나 Max 구독은 가족, 친구, 동료에게 선물할 수 있습니다. 1개월부터 12개월까지 기간을 정해서, 이메일이나 공유 링크로 전달할 수 있어요.

선물할 수 있는 요금제는 **Pro, Max 5x, Max 20x** 세 가지이고, 기간은 **1·3·6·12개월** 중에서 고를 수 있습니다.

### 선물하는 방법

1. Claude 계정에 로그인하고, 왼쪽 아래의 프로필 메뉴를 엽니다.
2. **Gift Claude**를 선택하거나 [claude.ai/gift](https://claude.ai/gift) 로 이동합니다.
3. 요금제(Pro, Max 5x, Max 20x)와 기간(1·3·6·12개월)을 선택합니다.
4. 카드 색상을 고르고, 원한다면 메시지를 함께 적습니다.
5. 전달 방식(이메일 또는 공유 링크)을 선택합니다.
6. 이메일로 보낼 경우 받는 사람의 이름과 이메일 주소를 입력합니다.
7. 결제를 완료합니다.
8. 주문 내역이 담긴 확인 이메일을 받습니다.

{{< hint type="warning" >}}
선물은 구매일로부터 365일이 지나면 만료됩니다. 받는 분이 너무 늦지 않게 사용할 수 있도록 안내해 주세요.
{{< /hint >}}

### 선물할 때 알아둘 점

- 선물을 구매하려면 본인이 **Pro 또는 Max 구독자**여야 합니다. 무료 사용자는 선물을 구매할 수 없습니다.
- Team·Enterprise 조직 구성원은 현재 선물을 구매할 수 없습니다.
- 받는 분은 신용카드 없이도 선물을 사용할 수 있습니다.

## 선물받은 구독 사용하기

선물을 받았다면, 이메일이나 링크를 통해 Claude 계정으로 로그인한 뒤 사용 확정만 하면 됩니다.

### 사용하는 방법

1. 보낸 사람에게서 받은 선물 이메일이나 링크를 엽니다.
2. 이메일 또는 링크 안의 사용(redeem) 버튼을 선택합니다.
3. Claude 계정에 로그인하거나, 새 계정을 만듭니다.
4. 사용 절차를 확인하여 완료합니다.
5. 또는 [claude.ai/gift/redeem](https://claude.ai/gift/redeem) 에 접속해 선물 코드를 직접 입력해도 됩니다.

### 내 요금제 상태에 따라 달라지는 점

선물을 받았을 때 어떻게 적용되는지는 현재 내 구독 상태에 따라 다릅니다.

| 내 현재 상태 | 받은 선물 | 적용 방식 |
|---|---|---|
| 신규 또는 무료 사용자 | 모든 선물 | 즉시 해당 요금제로 업그레이드 |
| Pro 구독자 | Pro 선물 | 구독 기간이 연장됨 |
| Pro 구독자 | Max 선물 | 일시적으로 Max로 업그레이드, 이후 Pro로 복귀(남은 Pro 기간은 계정 크레딧으로 전환) |
| Max 구독자 | 더 낮은 등급 선물 | 그 가치만큼 계정 크레딧으로 지급 |
| Max 구독자 | 같거나 더 높은 등급 선물 | 기간 연장 또는 업그레이드 |

{{< hint type="note" >}}
선물 사용은 웹에서만 가능합니다. iOS App Store나 Google Play에서 모바일 앱으로 구독 중이라면, 현재 구독이 끝난 뒤에 선물을 사용할 수 있습니다.
{{< /hint >}}

알아둘 점:
- 선물은 구매일로부터 365일이 지나면 만료됩니다.
- 여러 개의 선물을 동시에 사용할 수는 없습니다.
- 어떤 Claude 계정으로든 선물을 사용할 수 있습니다. 선물 이메일을 받은 주소와 꼭 같지 않아도 됩니다.

## 청구 주소와 세금

결제할 때 부과되는 세금은 **청구 주소**를 기준으로 계산됩니다. 청구 주소는 보통 주된 사업장(또는 거주지) 위치를 나타내야 합니다.

청구 주소를 바꾸는 방법은 요금제 종류에 따라 조금씩 다릅니다.

| 요금제 종류 | 청구 주소 변경 방법 |
|---|---|
| Pro / Max / 셀프서브 Team | 결제 수단의 주소를 따름. **Settings > Billing**에서 결제 수단 옆 **Update**로 변경 |
| 영업 지원 Team / Enterprise | **Settings > Billing > Billing Addresses**에서 직접 변경 (이후 청구서부터 즉시 반영) |
| API (사용량 등급 1~4, 셀프서브) | 결제 수단에 연동됨. Claude Console **Settings > Billing**에서 카드 옆 연필 아이콘으로 변경 |
| 유료 API 계약 | Claude Console **Settings > Billing > Billing Addresses**에서 변경 |

{{< hint type="tip" >}}
결제 수단과 다른 청구 주소를 사용하려면 지원팀에 증빙 서류와 함께 문의해야 합니다. 미국 외 고객은 요청 위치와 일치하는 VAT 등록 증명서가, 미국 고객은 법인 설립 주(state), 사업자 등록증, 또는 세금 ID 서류가 필요합니다.
{{< /hint >}}

한 번 검증을 마친 청구 주소는, 이후 결제 수단을 바꿔도 자동으로 변경되지 않습니다.

## 사용량 번들로 용량 추가하기

요금제에 포함된 용량을 다 쓴 경우, 크레딧을 미리 사두는 **사용량 번들(usage bundle)**로 더 저렴하게 용량을 채울 수 있습니다. 한 번에 많이 살수록 할인 폭이 커집니다.

| 번들 금액 | 할인율 |
|---|---|
| $50 | 10% 할인 |
| $250 | 20% 할인 |
| $1,000 | 30% 할인 |

### 구매하는 방법

1. **Settings > Usage**로 이동합니다.
2. **Buy usage**를 선택합니다.
3. 번들 크기($50, $250, $1,000)를 고릅니다.
4. 결제를 완료합니다. 할인은 자동으로 적용됩니다.

번들 크레딧은 하나의 공용 잔액으로 모여, Claude·Claude Desktop·Claude 모바일(iOS·Android)·Claude Code·Cowork·외부 연동 제품에서 함께 사용됩니다.

{{< hint type="note" >}}
구매 한도가 있습니다. 개인 Pro·Max 구독자는 한 달에 최대 $2,000, Team 요금제 리더는 한 달에 최대 $3,000까지 번들을 살 수 있습니다. 번들은 Pro·Max·Team 요금제에서만 구매할 수 있고, 한도와 포함 용량을 넘어선 사용분은 표준 요금으로 청구됩니다.
{{< /hint >}}
