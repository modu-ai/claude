# 웹·데스크톱

## Claude Code on the web

`claude.ai/code` — 브라우저에서 GitHub 저장소를 연결해 코딩합니다. **설치 불필요**.

### 시작하기

1. claude.ai/code 접속
2. GitHub 저장소 연결
3. 자연어로 질문

## 데스크톱 앱

로컬 앱으로 다음과 통합합니다:

- **Google Calendar** — 일정 연동
- **Slack** — 메시지·채널
- **GitHub** — 저장소
- **Linear** — 이슈 추적
- **Notion** — 문서

## 웹 vs 데스크톱

| 구분 | 웹 | 데스크톱 |
|------|-----|---------|
| 환경 | 브라우저 only | 로컬 앱 |
| 통합 | GitHub 중심 | 캘린더·Slack 등 다수 |
| 설치 | 불필요 | 필요 |
