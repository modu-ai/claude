# MoAI-Design · 비주얼 디자인 플러그인

{{< hint type="note" >}}
**2026 Q4 출시 예정** — 현재 내부 개발 중입니다. 출시 알림을 받으려면 [뉴스레터를 구독](https://claude.mo.ai.kr)하세요.
{{< /hint >}}

## 이런 분께 필요합니다

- 브랜드 디자인을 빠르게 만들고 싶은 마케터·스타트업 대표
- Figma를 활용하는 디자이너
- 마케팅 시각 자료를 자동화하고 싶은 분

## 준비 중인 기능

### 브랜드 아이덴티티

- **브랜드 가이드라인** — 로고, 컬러, 타이포그래피 규칙 생성
- **디자인 시스템 토큰** — Tailwind / shadcn 호환 컬러·컴포넌트 세트
- **비주얼 아이덴티티** — 명함, 레터헤드, PPT 템플릿 일괄 적용

### 마케팅 비주얼

- **SNS 콘텐츠** — 인스타그램·블로그·뉴스레터용 이미지 레이아웃
- **광고 배너** — 다양한 사이즈 자동 리사이즈
- **인포그래픽** — 데이터 기반 시각화 자동 생성

### 설계 도구 연동

- **Figma 통합** — 디자인 에셋 직접 내보내기·가져오기
- **디자인 리뷰** — 접근성·일관성 자동 검수
- **소셜 템플릿** — 플랫폼별 최적 사이즈 템플릿
